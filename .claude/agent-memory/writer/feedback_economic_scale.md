世界经济尺度校准基准：

**阿尔温收入**：每月8000-15000，取决于案子和委托人。不稳定收入。

**物价参考**（Ch7确定）：
- 灵稳剂：正规渠道500/盒，黑市150/盒
- 米娅月药费：3000（灵稳剂）+ 护理/检查/床位 = 8000+/月
- 根治/转院王都：几十万
- 灵晶病手术：5000 + 药 + 诊金 = 6100
- 阿尔温为陌生孩子付了5000 = 他大约半个月到一个月的收入

**Why:** 我draft中用的数字（灵稳片45/盒、治疗费140）太小了，读者感受不到经济压力的重量。用户调高数字后，"付5000救一个孩子"变成了真正的牺牲，结尾"希望口袋里的钱还付得起饭钱"才有分量。

**How to apply:**
- 写到任何金钱相关内容时，对标阿尔温的收入（8k-15k），让读者能算出"这对他意味着什么"
- 经济数字要让人肉疼，不是随手就能付的
- 体制性问题要用具体数字表达（报销三成、补贴从三阶开始且必须为贵族服务、每月限开四盒）
