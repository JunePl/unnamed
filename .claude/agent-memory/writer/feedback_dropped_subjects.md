---
name: 严禁省略主语
description: 写正文时必须保留主语（角色名/代词），文风参考中没有任何作者使用省主语的电报体，这是反复被退回的核心问题
type: feedback
---

写作和改写正文时，必须保留主语（角色名字或代词"他/她"）。这是硬性规则，不是风格选项。

**Why:** 用户多次反馈"为什么完全没有主语""写的太冷了""文风参考里没人是这样子的"。验证过文风参考.md——听日写"亚修坐在冰冷的石座上"，澎湃写"高阳努力睁开被触手挤压的半只眼睛"，用户自己写"埃尔希娅猛地睁开眼睛"。所有参考作者都保留主语。第五章重写时就因省主语滥用被退回。省主语不是"碎片感"，是写作错误。

**How to apply:**
- 每个动作句都必须有明确主语（角色名或代词）
- 唯一允许省主语：紧接上一句、主语完全无歧义的同一人连续动作（如"他站起来，走到窗边"）
- 即使是高冲击段落也不能整段省主语，最多连续1-2个短句省略后必须恢复
- 写完后自查：逐句确认每句话读者都能明确知道"谁"在做动作
