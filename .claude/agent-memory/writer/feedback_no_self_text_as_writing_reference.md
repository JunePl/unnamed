# 本作前文不能作为写作语感参考，只能用作剧情逻辑参考

## 规则

写作时每一句必须从**参考材料**（术师手册/一世之尊/调研_*.md/文风参考.md）独立构建语感。

本作前文（V1 原文/已发布章节/正文.novel 现有内容）只能用作以下用途：
- 剧情逻辑（事件序列、因果关系）
- 设定锚点（角色能力、世界规则、物品定义）
- 人物背景（关系、过往）
- 环境描述（角色当前在哪、刚发生了什么）

**绝对不能借用本作前文的：**
- 句式骨架（"X 的 Y 在 Z" 这种结构）
- 词汇组合（"悬在掌心""一节一节都在抖" 这种）
- 措辞节奏
- 列举词组（"指尖、手腕、肘弯" 这种三词列表）

## Why

本作 V1 原文很可能本身就是 AI 拐杖。如果借用 V1 句式来"重写"，等于在 AI 拐杖上套娃——形式上是改写，语感上还是 AI。

具体踩雷案例（Ch3244 重写）：
- V1 写 "「司契」契书悬在那里没掉" — 这一句本身就是 AI 拼贴（"悬在"+"没掉"是参考里两个不同语境的词被硬拼）
- 我延用 V1 句式写 "悬在掌心的「司契」契书没掉" — 等于把 AI 拐杖换皮重发

更深的踩雷：v6 整段都在 V1 骨架上改写
- "哈罗德的左手在抖" — 整句照抄 V1
- "指尖、手腕、肘弯都在抖个不停" — 同三词列举
- "但他的右手..." — 同句首结构
- "黑泥...涌过去" — 同结构
- "去吞那个站在...顶上的人" — 同骨架
- 用户原话："不要使用之前的原文进行写作参考啊，看逻辑可以但不是写作"

## How to apply

每次重写正文段时：

1. **第一遍读 V1 原文** —— 只为提取剧情逻辑和事件序列：
   - 这段发生了什么事件（A 做了 X，B 反应 Y）
   - 哪些设定/事物是前文已建立（角色能力、当前环境、物品定义）
   - 角色的情绪/状态（流血、断腿、维持契约）

2. **关闭 V1 原文，从参考材料独立构建语感**：
   - grep reference 文件夹里同类场景（高强度战斗、反噬、维持、挣扎）
   - 提取参考的句式、动词、节奏
   - 用参考的语感写出"重新发生"这段事件的版本

3. **写完后对照检查（必做）**：
   - 把 v7 vs V1 对照——任何句子和 V1 高度相似（>50% 相似度）都标红
   - 标红的句子必须重新设计，让句式骨架完全独立

## 例外：剧情设定锚点

以下情形允许和 V1 措辞相同，因为是设定的固定名词/位置：
- 角色名（哈罗德、阿尔温）
- 能力名（司契、越阶契书）
- 已建立的物理位置（废墟堆、烧烤店）
- 流血部位（鼻孔、嘴角、耳道——这是前文 3224 行设定的伤势位置）
- 物品名（水泥板、黑泥、灵能）

但即使是这些设定锚点，描写它们的**动词、形容词、修饰**必须独立。

## 验证方法

写完每一段后，逐句问自己：
- "这一句的句式骨架，能在 reference/ 文件夹里 grep 到完整对应吗？"
- "如果只看 V1 原文找不到这个句子，但去掉 V1 原文我还能写出这一句吗？"
- "我有没有因为 V1 写过 X 就跟着写 X 的变体？"

任何一题答 yes，必须重写。
