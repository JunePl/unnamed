阿尔温的语言有两个明确模式，写正文时必须根据场景情绪切换：

**放松模式**：喜欢调戏、开玩笑、嘴贫。不是冷面侦探，是一个松弛的聪明人。
- 例：叫艾伦"大画家"
- 例：对米娅说"其实我是暗恋学姐"
- 例：内心吐槽"简称化学，你有上过学吗"，嘴上也说出来
- 例：面对沉重场景先叹气揉头发，然后用行动回应（掏钱），不说大道理

**压力模式**：句子变短，停顿增多，幽默消失。
- 这个切换本身就是情绪信号——读者从"他不开玩笑了"就知道事情严重了

**Why:** 用户Ch7改稿中大量增加了阿尔温的幽默对白，并在角色设定文件中明确写了这两种模式。我的draft把阿尔温写得太冷、太"侦探"了，缺少人味和温度。

**How to apply:**
- 写阿尔温之前先判断场景压力等级：低压→贫嘴模式，高压→沉默短句模式
- 不要把"聪明"等同于"冷"。真正聪明的人松弛，不需要端着
- 阿尔温对熟人（米娅）和半熟人（艾伦）都会开玩笑，不是只对亲近的人
- 他用幽默化解尴尬和沉重，但不是回避——行动上他很直接（掏钱、按住米娅）
