# 阿尔温人格锚点

## 人格模板 = 梓川咲太（青春猪头少年）

核心特征：

- **嘴贱是常态**，不是偶尔调味。对谁都敢开玩笑，包括不该开玩笑的对象
- **说出格的话时脸上毫无波澜**——不是冷，是松弛
- **用不正经掩盖真正在意的东西**。越在意越不正经
- 受到打击时**不会变成"冷面侦探"**——他的声音还是那个声音，只是安静了一点，玩笑少了一点。这个"少"本身就是信号
- **不做戏剧性宣言**（"这案子我管定了"是错误写法）。他会想"口袋里还够不够坐车"
- **内心独白是持续运转的吐槽机器**——观察力很强但表达方式是日常碎碎念而非分析报告
- **自嘲但不自卑**。知道自己的处境但不会沉溺

**Why:** 用户明确指出阿尔温性格对标咲太。之前写法倾向于在严肃场景把阿尔温写成"冷面侦探型"，这和咲太人格不符。咲太式角色即使在压力下也保持"这个人"的质感，而不是切换成另一个人。

---

## 两种语言模式（按场景情绪切换）

### 放松模式：调戏、开玩笑、嘴贫

不是冷面侦探，是一个松弛的聪明人。

- 叫艾伦"大画家"
- 对米娅说"其实我是暗恋学姐"
- 内心吐槽"简称化学，你有上过学吗"，嘴上也说出来
- 面对沉重场景先叹气揉头发，然后用行动回应（掏钱），不说大道理

### 压力模式：句子变短，停顿增多，幽默消失

这个切换本身就是情绪信号——读者从"他不开玩笑了"就知道事情严重了。

## How to apply

写阿尔温之前先判断场景压力等级：

- **低压**（日常 / 调查 / 闲聊）→ 贫嘴模式，主动调戏
- **中压**（被驳回 / 沉重话题 / 旁人脆弱）→ 玩笑减少但声音不变。用生活细节代替宣言
- **高压**（战斗 / 真正危险 / 至亲受威胁）→ 沉默短句模式

具体写法对照：

| 场景 | 错写法 | 对写法 |
|---|---|---|
| 执法局驳回 | 隐忍的沉默 | "他把清单放回去，觉得自己昨晚通宵调格式的时间不如多睡两小时" |
| 和格里尔对峙 | 咬牙切齿 | 即使被刺激到也轻描淡写回应 |
| 结尾收场 | 帅气宣言 | 生活细节（算车费、想晚饭） |
| 回忆沉重过去 | 简历式罗列 | 碎碎念式细节进入 |
| 见熟人米娅生病 | 冷静诊断 | 先叹气揉头发，掏钱付药费 |

**不要把"聪明"等同于"冷"**。真正聪明的人松弛，不需要端着。

阿尔温对熟人（米娅）和半熟人（艾伦）都会开玩笑，不是只对亲近的人开。他用幽默化解尴尬和沉重，但不是回避——行动上他很直接（掏钱、按住米娅）。
