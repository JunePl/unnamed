# 可callback信号设计与和解情绪曲线

来自第17章兄妹戏（阿尔温×阿涅丝·和解+立约定）重写的三条强反馈。

## A. 可callback的约定/誓言/信号——必须从角色史长出来

**Why（用户原话）：**
- "消息写个1很奇怪啊"——否决用数字"1"做约定符号（空符号、网络梗味、没温度）
- "肯定不能是汪啊，阿尔温又不是狗，你咋想的呢"——否决让角色学狗叫报平安（OOC卖萌）

**三条标准（缺一不可）：**
1. **从角色共同历史里长出来**（不凭空造）。第17章最终用"拉钩·以后不做会死的事，违约就是小狗"——"小狗/看门狗"源自第二章阿涅丝吐槽哥哥"不知道的还以为我养了只看门的"。
2. **有温度 + 专属**（外人不懂这梗的分量），不能是空符号。
3. **极简 + 可重复触发，最好带刺/能反转**。阿尔温的活注定要冒险，这个拉钩他迟早违背——"迟早当那条小狗"，callback 自带张力，不用点破。

**调研锚点（不尴尬的"活着信号/誓言"范式）：**
- 《情书》"你好吗？我很好"——报平安句式天花板（极简但有温度）
- 《寻梦环游记》Remember Me——同一个东西从亲昵玩笑长成生死纽带（=拉钩小狗的逻辑）
- TV Tropes「Something Only They Would Say」/「Trust Password」——只有彼此懂的话，自带防伪+关系厚度
- 《进击的巨人》围巾——信物式 callback

**禁忌**：空数字/字符（"1"）、让角色做出格于人设的卖萌动作（学狗叫）。

## B. 嘴贫型角色（费渡/梓川谱系）的和解钥匙

**和解的钥匙 = 他"贫不下去"那一下**（壳裂开），不是软话、不是物质讨好。
- 第17章：旧办法（嘴贫分析松饼+煎牛排）只赢一半 → 大衣引爆（"连大衣都能没，那你呢"）→ 阿尔温惯常那句"你哥命大"到嘴边却**说不出来** = 和解真正发生的点。
- 参考默读费渡：嘴贫掩盖在意，壳裂开才动人。

## C. 情感高潮后必须有充分缓和（用户："这里生气的缓和不够"）

**Why：** 拉钩（情绪最高点）后若直接跳日常（谈房租），太硬。角色刚哭过/怕过/郑重承诺，需要一段被哄回来、慢慢软下来的过渡。

**How：** 用角色擅长的方式做回暖。第17章：揉头发 → 用看门狗梗逗 → 玩笑无意说成真话（"每天蹲门口等你回家"）→ 手忙脚乱掩饰（费渡式：暴露真心赶紧岔开）→ 噗笑破涕 → 拌嘴回日常。情绪从"哭/怕"完整落回日常基调，再接日常情节才不突兀。

## D. 写 callback 锚点前，先 grep 校对原文设定

**Why：** 我用"1"做约定锚点，没核对到第12章 2848（阿尔温濒死群发的其实是求救句"港口区，收到给执法局打电话"），导致和 4470/5563 的"1"矛盾，是用户校对发现的。

**How to apply：** 任何要反复 callback 的锚点（信号/信物/台词），动笔前 grep 全文相关设定，确认它在原文里到底是什么、确认无矛盾。
