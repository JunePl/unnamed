## Writer Agent Memory Index

### Feedback
- [角色内心声音密度规则](feedback_psychological_description.md) — 角色内心声音（口语化感受/吐槽）占比25-35%，是叙事核心燃料，不是需要限制的"心理描写"
- [去文学化铁律](feedback_style_deliterarify.md) — 正文必须像"讲故事"不像"写文学作品"，砍掉拟人/精致比喻/纯氛围句/文艺修辞，参考听日/入间人间/澎湃
- [Ch5-Ch6重写流程经验](feedback_ch5ch6_rewrite_lessons.md) — 分步执行流程、人味增强技巧（对话扩充/阿涅丝模板/间谍角色/活人填充/碎片内心）、V1保留vs重写判断、伏笔植入方法、验证清单
- [阿尔温语言模式规则](feedback_alwin_voice.md) — 放松时爱开玩笑调戏人，压力下句子变短停顿增多，不要把聪明写成冷
- [世界观信息嵌入方法](feedback_worldbuilding_exposition.md) — 从主角视角渗出（内心/对话/观察），可以直说但不能让读者一头雾水
- [经济体系与物价校准](feedback_economic_scale.md) — 阿尔温月收入8k-1.5w，灵稳剂500/盒，数字要让人肉疼
- [信息释放节奏控制](feedback_info_release_timing.md) — 推理线索不能释放太早，情感场景不当侦探，"差点说出来被打断"比"推理出来"更好
- [句子节奏铁律](feedback_sentence_rhythm.md) — 基础句式必须是中长句连贯思绪流，短句仅极少数戏剧强调用。AI默认堆短句是偷懒，必须纠正
- [内心独白禁用文艺比喻](feedback_no_literary_metaphor_in_voice.md) — 情绪越重越要守住角色认知指纹，禁止文艺比喻/通用类比/叙述者概括情绪，阿尔温=分析拆解自嘲实务
- [阿尔温核心人格锚定：梓川咲太](feedback_alwin_sakuta.md) — 松弛嘴贱、用不正经掩盖在意、压力下幽默消退但声音不变人、不做戏剧性宣言
- [禁止"不是X，是Y"句式堆砌](feedback_no_negation_pattern.md) — AI高频拐杖，一段内最多1次，写完后检查"不是"频率
- [禁止破折号"——"滥用](feedback_emdash_overuse.md) — 200字内最多1个，优先用逗号/句号，只保留给真正的戏剧性打断
- [内心声音要人性化不要机械分析](feedback_no_mechanical_inner_voice.md) — 身体感受用口语反应不用医学定位，转场直接切不解释机制，压力下也要有人味
- [禁止信息汇报式写作](feedback_no_data_report_style.md) — 信息必须通过"人经历了什么"传递，不能列举结果。"剩一发""零伤害""预见更新"是游戏日志不是小说
