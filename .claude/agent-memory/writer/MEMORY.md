## Writer Agent Memory Index

### 🔑 核心新工作流（2026 神殿章后必读）

- [递归骨架填空法](feedback_skeleton_first.md) — **写作唯一核心步骤**。🟥🟥🟥最高一条(2026-06第20章)：大骨架＝「一整段连续、来自同一处」的真实母段，整段搬来只换专名、剧情迁就、一字不自造。❌禁东拼西凑(从多处各捡短语拼)❌禁先列叙事拍点/节奏蓝图再找词(=先自创结构·病根，Step2/2.5已废弃)。给用户看"母段原文逐句 vs 成稿"对照。S级三铁律：①一句一母句不缝合②无孤立词拼句③确切组合grep(近义不算)。grep必须`-Hn`带文件名，"或"用`|`不是`\|`

### Feedback（13 大主题）

- [章节信息密度](feedback_chapter_density.md) — **2026 新增**：单章目标 8-10k 中字。阿尔温内心总结（"在心里把这条挂了一格""时间线对了一遍""挂在 X 的话旁边"等）默认全删——读者第一遍能看出的钩子不需叙述者画箭头。流水账过渡/重复物品细节/配角背景对话/长劝说要砍
- [98% 贴合度流程](feedback_98_percent_process.md) — 每个词/句式必须能在参考里 grep 到。Step 0-7 流程不可跳过。"差的很远"=视角错，"特别 AI"=句式拐杖。本作前文不做写作参考。前文已建立的不重复
- [AI 句式拐杖与自造表达](feedback_ai_crutches.md) — 14 类拐杖统一禁用清单（破折号 / 不是X是Y / X的那种 / 文艺比喻 / 自造词 / 现代词 / 省主语 / 短句独段 / spoiler / bullet list / 配角术语 / 拟人怪物 等）+ grep 自检命令
- [具体化铁律](feedback_concrete_writing.md) — **2026 新增**：抽象触感词（烫/痛/麻）和视觉名词（腿/眼睛）必须具体化；用户提示词的抽象指令（"死一样的压力"）不能直接搬到正文；中文 cliché 比喻黑名单（"无形的手攥住""心被攥住""死一样的压力"等）必须 0
- [角色智识水平校准](feedback_character_intelligence_calibration.md) — **2026 新增**：阿尔温是顶级分析师，内心独白不能是小学生格言式"原则"（如"能看清的先看清楚"）。禁止"X 是他在 Y 养出来的本能/原则/习惯"句式
- [场景一致性](feedback_scene_consistency.md) — **2026 新增**：描写匹配场景类型（神殿不写"灰未沉"等犯罪现场用词）。反应强度匹配威胁级别（L3 致命压力=僵硬冻结，不是"胃里翻"）。行为感知逻辑一致（不敢看≠数眼睛）。物理逻辑自洽（单印记≠"哪个位置亮"）
- [伏笔克制](feedback_foreshadowing_restraint.md) — **2026 新增**：禁止叙述者直接挂钩（"X 亮的位置正对着 Y"）。禁止双重明示。目标 90% 用"客观陈述+留给读者"
- [内心独白方法](feedback_inner_voice.md) — 内心声音 25-35% 占比 + 人性化（不机械分析）+ 不写信息汇报式 + 作者情绪 vs 角色情绪分离（含五道检查 + 情绪词黑名单）
- [节奏 / 动作 / 信息释放 / 世界观](feedback_action_rhythm.md) — 中长句基础短句调味（绝不允许连续 3+ 短独段）+ "想要X但Y所以Z"挣扎结构（5 要素）+ 情感场景不当侦探 + 世界观从主角视角渗出
- [人类认知过程](feedback_human_cognition_process.md) — 角色遇陌生事物必须走"看见特征→慢半拍→猜测→结论"四拍。§E 已知类型事物的第一眼识别也要走过程。§F 凭空植入背景禁令（按理说/按规矩等所有变体）。比喻必须从角色生活记忆库取材（阿涅丝缝玩偶/办公桌/执法局教官/博物馆骨片），不能用抽象通用词
- [阿尔温人格与语言模式](feedback_alwin_persona.md) — 梓川咲太式人格锚点 + 两种语言模式（放松贫嘴 / 压力沉默短句）
- [章节审核与重写经验](feedback_audit_lessons.md) — Ch5-6 人味技巧 + Ch16-18 五类踩雷 + 经济体系物价校准
- [可callback信号设计与和解情绪曲线](feedback_callback_signal_and_reconciliation.md) — **Ch17 新增**：约定/誓言/活着信号必须从角色史长出来（禁空符号"1"/禁OOC卖萌"学狗叫"）；嘴贫型角色和解钥匙=贫不下去那一下；情感高潮后必充分缓和；callback锚点先grep校对原文
