骨架速写（只有"动作层＋环境层"）≠饱满。轻小说的"饱满＋味道"，是在动作/环境之上再叠一层 **"内心层＋主观滤镜＋过程化"**。第三人称限制视角照样能做——天使大人本身就是第三人称轻小说。

**Why**：科尔·芬奇视角段第一版只有 433 字、纯骨架，用户两次指"加的不够厚，细节和轻小说感不够"。去九本精读三段范例（终局2040-2108 夜街独行 / 玩乐498-536 第一人称日常 / 天使大人940-965 第三人称日常）提炼出规律。

**How to apply**：写一个情节时，骨架搭好后用下面 8 手法"撑满"——

1. **内心独白"说—修正/转折"**：陈述一句→立刻翻转或补充。终局「自由了」→「不，并没有那回事」；天使「全要感谢天使，但周内心也有一点觉得她多管闲事」。
2. **环境/物件带主观滤镜**：不止客观描述，加"他"的想象/对比/体感。终局「此刻染上夜的黑，在蓝天下应该会更加赏心悦目吧」；天使「料理带著微温，暖意传递至手心」。
3. **移动/动作过程化、空间化**：把"走过去"展成有空间纵深的旅程。终局「穿过东侧已杳无人迹的市集，走过萧条的铁皮屋顶道路，走了二十分钟」。
4. **琐碎真实的随手观察**：视角人物注意到不相干的细节，制造在场感。终局「能看见有大大的鸡舍与拖拉机。啊，还有山羊」。
5. **结论先行→自证展开→"可是…"真实情感**：玩乐「我一辞职这店就完蛋」→大段自证→列举该辞职的理由→「可是…」→藏在最后的真实理由。
6. **自嘲式履历/递进列举**：把一个状态用时间递进具体化。玩乐「一开始只是…结果后来…再后来…最后…」。
7. **情绪身体化＋克制夸张**：情绪落到具体身体反应。终局「哭到能装满一整个水桶」；天使「脸颊不禁阵阵抽搐」「惊出些许冷汗」。
8. **矛盾心理并置**：对同一件事两种相反感受。天使「周还是坦率地收下保鲜盒…尽管心里觉得愧疚」。

**关键纪律**：这些"内心层"句子同样要走 [[feedback-skeleton-first]] 递归骨架、grep 九本——区别在于要去找 **"带内心活动/主观色彩的母句"**，而不只是"客观陈述事实的母句"。客观陈述母句→写出来骨感；带味道的母句→写出来饱满。找母段时按情绪同构去捞（夜街独行的怂、收到照顾的别扭、受气后的自嘲），而不是按事件去捞。
