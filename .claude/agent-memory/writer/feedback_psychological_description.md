---
name: 心理描写比例规则
description: 正文允许少量心理描写（≤10%），主要用对话和动作推进叙事，心理描写用于节奏放松
type: feedback
---

心理描写是允许的，但占比必须低于10%。主要叙事推动力来自对话和人物动作。

**Why:** 用户认为完全零心理描写会让小说太紧绷，少量心理描写能提供阅读的放松感。章纲V7.0中的"零心理描写"铁律过于严格，实际执行时需要放宽。

**How to apply:**
- 每千字中心理描写不超过1-2句
- 心理描写用于节奏调节（紧张段落间的呼吸），不用于直接解释情绪（"他很伤心"）
- 优质心理描写标准：精准、简短、带有角色语言特征（阿尔温的分析师口吻、阿涅丝的直觉式思维）
- 仍然禁止：情绪形容词堆砌、替读者感受、用心理独白点明伏笔
