用户在2026-03-23反馈：小说"人感太少"，"完全不像从某个人的视角讲故事"。根因是旧规则（心理描写≤10%）严重压缩了角色内心声音的空间。

**旧规则：** 心理描写≤10%，每百字不超过1-2句。
**新规则：** 角色内心声音占比25-35%。

**Why:** 文风参考中的作者——竹宫、听日、不讲理——内心独白密度极高（估计30-40%）。他们的"人感"就是靠这个密度撑起来的。"心里慌得一批"、"但是再烦躁也不能拿镜子怎样啊！"、"好不甘心啊"——这些不是传统意义上的"心理描写"，是角色的声音，是叙事的核心质感。把它压到10%就等于把角色变成了提线木偶。

**How to apply:**
- "角色内心声音"≠"心理描写"。前者是角色用自己的口语在脑子里说话，后者是叙述者用文学语言解说角色心理。前者大量使用，后者仍然禁止
- 占比25-35%是目标区间，不是硬上限。如果一段读起来很有"人在说话"的感觉但超过了35%，那也没问题
- 仍然禁止的：叙述者视角的情绪形容词堆砌（"他感到无比悲伤"）、替读者感受、用内心独白点明伏笔
- 自检方法：读完一段，问"我知道这个角色此刻在想什么、感受什么吗？"——如果不知道，就缺少内心声音了
