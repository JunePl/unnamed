用户在2026-03-23明确反馈：第十章draft短句密度严重超标，不符合任何一个文风参考样本。

**核心问题：** 提示词中的"短句打节奏"和"压力模式句子变短"两条规则被过度执行，导致短句成为默认句式，中长句反而成了例外。

**文风参考的真实句子节奏：**
- 听日："这不能怪他，毕竟他前一秒的记忆还在那该死的手游公司里通宵到凌晨贡献剩余价值，后一秒就忽然来到这个奇怪的地下大厅里接受众人的膜拜，换成谁都反应不过来。"——一整条连贯长句
- 竹宫："明明总是小心翼翼将水气擦干，明明上个礼拜才花了一整天在厨房、盥洗室等处举行除霉大会的。但是在这间破房子差劲的透气能力前，不论怎么样的努力都有如泡影。"——中长句连缀
- 不讲理："虽然我也不知道为什么前世的记忆都还在，不过这样也没有造成什么困扰。"——松弛的中长句
- 三个作者的内心独白都是**连贯的思绪流**，句子该长就长，碎碎念也是连着说的

**Why:** AI天然倾向于用短句堆叠来模拟"节奏感"和"力度感"，但这是偷懒。真正的节奏是长短交替，基础是中长句。

**How to apply:**
- 基础句式 = 中长句（2-4句一个自然段，句内用逗号/连词连接）
- 短句独立成段 = 极少使用，仅用于：单字母揭示（D。）、关键对话结尾（没了。）、真正的情绪爆发
- 绝不允许连续3个以上的短句独立段落
- 内心独白 = 连贯思绪流，一个念头自然接下一个念头，用逗号而非句号+换行
- 竹宫式段落结构：一段2-4句，段间在话题/动作/情绪转换处断开
- 写完后自查：如果一页里超过1/3的段落只有一句话，节奏就出了问题
