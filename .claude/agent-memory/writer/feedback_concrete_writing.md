# 具体化铁律

**Why（用户原话）：**
- "什么是死一样的压力？"
- "烫一下是什么感觉？"
- "也不描述腿什么样？人腿？蜘蛛腿？"
- "心也跟着被一只无形的手攥住，攥得越来越紧 — 写的一点压力都没有，和给小学生看的文艺小说里面的描写一样"

AI 默认偏好用"抽象词+笼统名词"高效传递信息。但小说不是信息传递，是**让读者感觉到**那个东西。**所有抽象词必须翻译为具体感官细节**。

---

## A. 三类必须具体化的词

### A.1 抽象触感词（必须具体化）

| 错（抽象） | 对（具体） |
|---|---|
| 烫了一下 | 像被烟头按了一下／温热从指节蔓延／一道电流从手背窜上手腕 |
| 痛了一下 | 像被针扎／像被钳子夹／像有东西从内侧顶 |
| 麻了一下 | 像针扎过的麻／像血没流通的麻／像电过的麻 |
| 紧了一下 | 像有东西收着脖子／像衣领突然小了一号 |

**规则**：单字感官动词（烫/痛/麻/紧/酸/胀）+"了一下"= AI 偷懒。必须**指明像什么东西+在身体什么具体位置+持续多久**。

### A.2 抽象视觉名词（必须描述特征）

| 错（抽象单名词） | 对（带特征） |
|---|---|
| 是腿 | 是六条又细又长的腿，关节多得不像人腿，每节都有一圈短毛 |
| 眼睛太多 | 数不清几只眼睛，分上下两排嵌在脸上，瞳孔是竖的 |
| 是个女神像 | 一个比他高出一倍的女人形状，头戴某种冠，手里抱着东西看不清 |
| 一座神殿 | 一座白色的大建筑，弧形屋顶，柱子比他粗三圈 |

**规则**：单一类型名词（腿/眼睛/神像/桌子/剑）= AI 偷懒。必须**至少 2 个视觉特征**（数量/形状/颜色/材质/比例之一两个）。

### A.3 用户提示词的抽象意图（不能直接搬到正文）

**这是最严重的错**——用户给的关键词是**意图描述**，是给我看的"应该写出什么感觉"，不是**最终文字**。

| 用户提示词（意图） | 我错误的搬运 | 对（具体化） |
|---|---|---|
| "死一样的压力让他喘不过气来" | "一种死一样的压力从前面压过来" | "再迈出去一寸，他就会死——这个判断不是想出来的，是身体先告诉他的：肺里的空气一下子抽空，喉咙后面有什么东西卡住，皮肤底下每一寸都在冒冷汗" |
| "心脏彷佛停跳" | "心脏像停了一下" | "他听不见自己的心跳了——不知道是停了还是被别的什么声音盖住了" |
| "气短" | "气也短" | "他想吸一口气，吸不下去；想吐出来，也吐不出来" |

**铁律**：**用户的提示词不是文字，是指令**。每个抽象词必须用 5W1H 拆解：是什么样的死？什么样的停？什么样的短？

## B. 中文 cliché 比喻黑名单（扩充）

以下比喻是中国网络小说/动漫滥到无以复加的描写，**必须 0 命中**。

### B.1 心脏/胸口类（最高频踩雷）

| 黑名单 cliché | 替代方向 |
|---|---|
| 无形的手攥住（心/胸口） | 描述身体真实反应（喉咙卡/肺空/冒汗） |
| 心被攥（住/紧） | 同上 |
| 心脏一沉 | "他听不见自己的心跳了" |
| 心脏漏了一拍 | 已在 [[feedback_inner_voice]] 黑名单 |
| 死一样的压力 | 拆解（见 A.3） |
| 窒息的恐惧 | 拆解 |
| 心被狠狠揪了一下 | 拆解 |

### B.2 身体反应类

| 黑名单 cliché | 替代方向 |
|---|---|
| 胃里翻一下（用在死亡威胁场景） | 死亡威胁=僵硬冻结，不是温和不适。详见 [[feedback_scene_consistency]] |
| 脊背一阵发凉 | 已在 [[feedback_inner_voice]] 黑名单 |
| 头皮发麻 | 拆解（什么样的麻？） |
| 浑身的血都凉了 | 拆解 |
| 后背瞬间湿透 | 拆解 |

### B.3 感官类

| 黑名单 cliché | 替代方向 |
|---|---|
| 时间彷佛静止 | 描述具体感受（听不见声音了/眼睛失焦） |
| 周围的一切都消失了 | 同上 |
| 思绪一片空白 | 描述具体（他不知道接下来该做什么） |

## C. 自检命令

写完段落后跑：

```bash
# B.1-B.3 cliché 黑名单（必须 0）
for w in "无形的手" "心被攥" "心脏一沉" "死一样的压力" "窒息的恐惧" \
         "心被狠狠揪" "胃里翻" "脊背一阵发凉" "头皮发麻" \
         "浑身的血都凉" "后背瞬间湿透" "时间彷佛静止" \
         "周围的一切都消失" "思绪一片空白"; do
    c=$(grep -c "$w" /tmp/check.txt 2>/dev/null)
    [ "$c" != "0" ] && echo "cliché 残留: $w -> $c 次"
done

# 抽象触感词+"了一下"模式
grep -nE "(烫|痛|麻|紧|酸|胀)了一下" /tmp/check.txt && echo "▲ 抽象触感未具体化"

# 单一类型名词独段（"是腿。""是神像。"等）
grep -nE "^是[一-龥]{1,3}。$" /tmp/check.txt && echo "▲ 单名词独段未带特征"
```

---

## D. How to apply

1. **写之前**：检查用户提示词里的每个抽象指令词，列出对应的"具体感官清单"
2. **写时**：每写一个"X 了一下"或"是 X" 立刻问——X 是什么样的？读者能不能用脑子画出来？
3. **写完**：跑 §C 命令。任何残留 = 重写那一句。
