阿尔温的人格模板 = 梓川咲太（青春猪头少年）。

核心特征：
- 嘴贱是常态，不是偶尔调味。对谁都敢开玩笑，包括不该开玩笑的对象
- 说出格的话时脸上毫无波澜——不是冷，是松弛
- 用不正经掩盖真正在意的东西。越在意越不正经
- 受到打击时不会变成"冷面侦探"——他的声音还是那个声音，只是安静了一点，玩笑少了一点。这个"少"本身就是信号
- 不会做戏剧性宣言（"这案子我管定了"是错误写法）。他会想"口袋里还够不够坐车"
- 内心独白是持续运转的吐槽机器——观察力很强但表达方式是日常碎碎念而非分析报告
- 自嘲但不自卑。知道自己的处境但不会沉溺

**Why:** 用户明确指出阿尔温性格对标咲太。之前的写法倾向于在严肃场景把阿尔温写成"冷面侦探型"，这和咲太的人格不符。咲太式角色即使在压力下也保持"这个人"的质感，而不是切换成另一个人。

**How to apply:**
- 执法局被驳回时：不要写成隐忍的沉默。写成"他把清单放回去，觉得自己昨晚通宵调格式的时间不如多睡两小时"
- 和格里尔对峙时：即使被刺激到也用轻描淡写的方式回应，不是咬牙切齿
- 结尾收场：不要帅气宣言，要生活细节（算车费、想晚饭）
- 回忆沉重过去时：用碎碎念式的细节进入，不是简历式罗列
