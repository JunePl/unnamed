用户在2026-03-15明确反馈：修改后的第一章"太像文学作品了，不像故事"。核心问题是叙述"太端着"，不够松弛随意。

**Why:** 网文读者要的是被故事拽着走的感觉，不是欣赏作者的遣词造句。每一处精致修辞都会让读者从故事中"跳出来"意识到有人在写作。

**How to apply:**
- 拟人修辞 → 功能性描述："黑暗往这边挪了挪" → "暗了一点"
- 精致比喻 → 直白说法："语气像在念一行无关紧要的备注" → "语气随意"
- 文艺味觉 → 感受碎片："苦涩的味道荡漾在口腔" → "苦，带点橄榄的咸"
- 讲究的被动句 → 简单句："被特意烫得温热而不烫手" → "盘子是温的"
- 纯氛围句（不推进剧情/角色/信息）→ 直接删除
- 文艺结尾 → 直白结尾："消融在夜色里" → "走出几步就看不见了"
- 冷幽默保留不动："但尸体不在乎这种气氛"这种是好的
- 短句鼓点保留不动："惊讶。""杰作。""走了。"
