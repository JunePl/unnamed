# 章节审核与重写经验

## A. Ch5-Ch6 重写经验（人味增强技巧已验证有效）

### A.1 流程：分步执行 + 逐步 review 是最优

- 大规模重写（多章）分步执行，每步完成后给用户 review 再继续
- 每步的 prompt 必须包含：前文风格锚定（50+ 行正文样本）、文风参考样本、人味模式清单、完整场景节拍、伏笔指令
- **先改上游依赖**（Ch4 结尾改时间线），**再改下游内容**（Ch5-Ch6），避免时间线矛盾

**Why:** Ch4 结尾的时间线变更会级联影响 Ch5 所有场景的时间引用。如果先写 Ch5 再改 Ch4，会出现"昨天/刚才"这类引用不一致。

**How to apply:** 任何涉及时间线变更的重写，都从最上游的改动点开始，逐步向下推进。

### A.2 对话场景扩充法

- **问题**：V1 的对话场景太短（艾琳 4 轮、兄妹 6 轮），像功能性问答
- **解决**：
  1. 给角色一个**物理上的理由停留更久**（艾琳从"点外带咖啡"改为"坐下来吃东西" → 对话时间自然变长）
  2. **不要直球问答**——绕着弯子，中间穿插小动作、被打断、转移话题
  3. 每个对话场景至少 **8-10 轮**有质量的来回（"有质量" = 推进信息/关系/性格，不是废话）

### A.3 阿涅丝存在感模板

- **小织的递进式小动作**（第一次拍甜品页→被拒→第二次拍蛋糕→被拒→趴桌面扒面包屑）：同一个行为的三次变体，越来越偷偷摸摸
- **阿涅丝的吐槽要有具体细节**（不是"你又放我鸽子了" → 是"你知道我在大街上站了十分钟才拦到一辆车吗？十分钟。大太阳底下。"）
- **阿涅丝的观察力**：她注意到哥哥的习惯变化（"你今天怎么不揉头发"）、声音变化（"你跟好看的人说话声音都会变的"）
- 让她讲一个有画面感的小故事（企鹅走路），不是一句话概括

### A.4 间谍型角色的"真心不在意"表现法（艾琳模式）

- 不是冷冰冰——是松弛。"什么关系？""朋友关系。"这种轻描淡写比冷淡更可疑
- 拿证物时手很稳（正常人会犹豫/不安/好奇，至少有一个）
- 说真话但没价值的信息（间谍本能：真话比谎话安全）
- 离开时扫一眼全场（间谍习惯，非刻意，细节渗出）
- 注意到主角帅的微小信号：视线多停半秒、"你跟我想象中的分析师不太一样"——克制，不是动心

### A.5 政府机构/公共场所的"活人"填充

- 等候区的人在做具体的事（打电话片段、抱怨等候时间、翻白眼）
- 环境有"没人管"的细节（歪了的告示牌、走得慢的钟、暖气省着用）
- 工作人员有个人 tic（拇指蹭笔帽的固定节奏）——一个就够，反复出现形成标识

### A.6 V1 保留 vs 重写的判断标准

- **保留条件**：场景节奏对、信息传递准确、没有时间线冲突、文字质量 OK → 不要为了重写而重写
- **必须重写**：对话太短（<8 轮）、角色没有辨识度（遮名猜不出谁在说话）、环境没有活人、时间线冲突
- **微调即可**：加一两句个性化台词、加一个角色 micro-action

**关键原则**：好的段落只需要改开头几行 + 修正时间引用 + 加 1-2 句碎片内心，其余原封不动。**手术刀式修改 > 推倒重来**。

### A.7 新悬念 / 伏笔的自然植入

伏笔最好是角色的自然行为/对话中渗出来的，不是旁白点明的：

- 圆脸执法官"上头挺关注的""注意点"——四行对话，不解释，读者自己想
- 马蒂亚斯"这不是我"——对自己画作的否认，恐怖感通过细节比直接写"他很恐惧"强一百倍
- 灵稳片（来路不明的药）——"没有生产厂家，没有批号"，暗示有人在接触艾伦
- 地上纸张反复出现的圆形——视觉恐怖，不解释

**方法论**："他说'这不是我'的时候声音很轻，像是在跟自己说" 比 "马蒂亚斯似乎对自己的画产生了认知混乱" 好一万倍。

### A.8 验证清单（每次重写完必须逐条检查）

1. 每个对话场景是否 ≥8 轮有质量的来回？
2. 阿涅丝在场时是否有 ≥5 句有辨识度的台词？
3. 重要 NPC（艾琳/老妇人等）是否有个性化对话，而非纯工具人？
4. 碎片式口语内心是否自然穿插（不是旁白感的分析）？
5. 是否有像"顺贝果""棒棒糖""蹭笔帽"这种有人味的小细节？
6. 维辛妮 / 伏笔 / 单镜等设定是否严格符合档案？
7. 时间线引用是否全部正确？（"昨天""刚才""隔天"等）
8. 人味模式 H1-H8 是否至少用到 2-3 条？

## B. Ch16-18 整体审核发现的 5 类高频踩雷

### B.1 现代词污染世界观（详见 [[feedback_ai_crutches]] §7）

- **问题**：Ch16 写彼界石柱残影时用了 "投影仪打出来的画面" 作比喻
- **修改样例**："投影仪打出来的画面被雾吞掉" → "一张泡在水里的旧画，颜料往四面化开"

### B.2 自己 spoil 自己的 punchline（详见 [[feedback_ai_crutches]] §11）

- **问题**：叙述者已经自己用"不是预见"做了预告，punchline 到达时没有冲击力
- **修改样例**："不是预见。" → "不是往前，是往后。"

### B.3 K 读物/档案/教材用 bullet list 格式（详见 [[feedback_ai_crutches]] §12）

- **问题**：bullet list 是维基百科格式不是小说
- **修复**：所有"角色阅读文献"场景必须展开为阅读动作

### B.4 叙述者替读者做笔记（详见 [[feedback_ai_crutches]] §13）

- **问题**：叙述者直接解释"这说明了 X"，破坏第三人称贴近视角
- **修复**：所有"某个细节的变化"通过角色内心的对比动作传递

### B.5 配角口中不能出现专业术语（详见 [[feedback_ai_crutches]] §14）

- **问题**：米娅（普通平民）说"统计学上的关联"
- **修复**：配角对白通过"这个人的生活经验语料库"过滤

### B.6 审核工作流

每次完成连续多章后必须跑一次整体审核：

1. Grep 统计 "——" 和 "不是X，是Y" 次数 → 超标先机械修复
2. 逐章通读，寻找以上 5 类踩雷
3. 修复后再 grep 一次验证
4. 给用户汇报审核通过的部分 + 修改的部分

## C. 经济体系与物价校准（数字必须让读者肉疼）

### C.1 阿尔温收入基准

**每月 8000-15000**，取决于案子和委托人。**不稳定收入**。

### C.2 物价参考（Ch7 确定）

| 项目 | 价格 |
|---|---|
| 灵稳剂（正规渠道） | 500/盒 |
| 灵稳剂（黑市） | 150/盒 |
| 米娅月药费（灵稳剂 + 护理 + 检查 + 床位） | 8000+ |
| 根治/转院王都 | 几十万 |
| 灵晶病手术（5000 + 药 + 诊金） | 6100 |
| 阿尔温为陌生孩子付的 | 5000（约半个月-一个月收入） |

### C.3 Why

draft 中用的数字（灵稳片 45/盒、治疗费 140）太小了，读者感受不到经济压力的重量。**用户调高数字后，"付 5000 救一个孩子"变成了真正的牺牲**，结尾"希望口袋里的钱还付得起饭钱"才有分量。

### C.4 How to apply

- 写到任何金钱相关内容时，对标阿尔温的收入（8k-15k），让读者能算出"这对他意味着什么"
- 经济数字要让人肉疼，不是随手就能付的
- 体制性问题要用具体数字表达（报销三成、补贴从三阶开始且必须为贵族服务、每月限开四盒）
