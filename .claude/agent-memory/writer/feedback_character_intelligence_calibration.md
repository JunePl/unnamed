# 角色智识水平校准

**Why（用户原话）：**
"'能看清的东西先看清楚，看不清的东西更要走过去看清楚' 这会是执法局，一国的执法机构教出来的？而且阿尔温还是最厉害的分析师？"

我把阿尔温写成了"小学生作文老师教的学生"——这是对角色智识水平的根本误解。

---

## A. 阿尔温的智识基线（写之前必须想清楚）

### A.1 设定锚点
- 卡萨林皇家学院毕业
- 二十三岁就成为执法局特聘灵痕分析师
- 一国最高执法机构的顶级分析师
- 七层思维（V1 设定）
- 即使被革职，对真灵的波动有"近乎本能的敏感"

### A.2 这意味着什么

| 类别 | 智识水平 |
|---|---|
| **观察** | 能从一个细节推出三层因果链 |
| **分析** | 不需要"列原则"，直接执行；原则是 background knowledge，不会浮到表面 |
| **语言** | 内心独白有专业术语+口语吐槽混合，不是"小学生作文" |
| **工作流** | 工作流是肌肉记忆级的反应，不是"想了想该怎么做" |

---

## B. 错例 vs 对例

### B.1 工作流"原则"

**错（小学生格言）：**
> 能看清的东西先看清楚，看不清的东西更要走过去看清楚——这是他在执法局养出来的本能。

—— 这种"X 的 X，X 也 X"的对仗句式 + "本能" 的笼统词 = 小学生作文。
执法局**不会教这种废话**。
真正的顶级分析师**不会用这种格言式句式自言自语**。

**对（专业级直接反应）：**
> 他下意识朝右边走了一步。右边那尊神像最近，看不清，他想知道是什么。

—— 直接动作 + 简洁动机。没有"原则"装腔作势。

### B.2 凭空植入"按理说"

**错（假大空背景知识）：**
> 底座上有一块铭牌的痕迹，按理说该刻着这位神的名字。

—— "按理说"是空话。**什么理？阿尔温是怎么知道这个理的？** 如果不能回答，就是凭空植入。

**对（让发现本身说明问题）：**
> 底座有一块凹下去的方块，方块里光滑得反常——别的地方有灰，这一块没有。

—— 让读者自己看出"这里被磨过"，不要"按理说"。

### B.3 工作流伪装成原则

**错：**
> 在陌生环境中第一时间按右手原则尽量不错过任何信息。

—— "按 X 原则尽量 Y" 是教科书句式。阿尔温写的报告可能这么写，**心里不会这么想**。

**对：**
> 习惯朝右走。

—— 三个字。本能就是本能，不解释。

---

## C. 三条铁律

### C.1 禁止"X 是他在 Y 养出来的本能/原则/习惯"句式

这种"显式标注本能"的句式 = 作者在替角色解释。角色自己**不会想**"哦，这是我的本能"。

替代：直接写动作 + 简短动机。

### C.2 禁止"按理说/按规矩/按常理/正常来说/X 都该有 Y"凭空植入背景

任何不能回答"这个知识是从哪学的"的"背景知识"必须删除或改写为"当场观察推断"。

详见 [[feedback_human_cognition_process.md]] §F。

### C.3 内心独白的智识高度=角色实际水平

- 阿尔温内心独白 ≈ 二十三岁特聘分析师在自言自语
- 不能是"中学生作文范本"
- 不能是"执法局新人手册"
- 应该是"懒得对自己端着，但脑子永远比嘴快两步的人"——参考 [[feedback_alwin_persona.md]]

## D. 自检命令

```bash
# 禁用句式
grep -nE "这是他在.{0,8}养出来的(本能|原则|习惯)" /tmp/check.txt
grep -nE "按理说|按规矩|按常理|正常来说" /tmp/check.txt
grep -nE "按.{0,4}原则" /tmp/check.txt

# 小学生对仗句式（X 的 X，X 也 X）
grep -nE "能.{1,8}的.{1,8}先.{1,8}，.{1,8}的.{1,8}更要" /tmp/check.txt
```

任何命中 = 该句重写。

## E. How to apply

1. **写阿尔温内心独白前**问自己：这话**二十三岁顶级分析师**会想吗？还是**写作文的初中生**会想？
2. **检查**：把句子放回 V1 已发布的阿尔温内心独白上下文，违和 = 重写
3. **参考锚点**：用户已发布正文中阿尔温的内心独白片段——智识高度的真实校准
   - 正文 2387："阿尔温没追问。司契这类契约型真灵在执法局档案里他翻过几例..."（专业分析+不解释）
   - 正文 5052："阿尔温用手指在桌面上敲了两下，心里极快地算了一遍..."（精确计算+不绕弯）
   - 正文 5887："阿尔温在心里翻译了一下：我会想办法问出你不想说的东西。"（人精级翻译）
