## 关键事实
- **童年记忆**：5 岁前无任何记忆。所有"小时候"的回忆只能定位在 5 岁之后。
- **成长环境**：孤儿院长大，和阿涅丝一起被领养，养至 16 岁。
- **现存亲人**：仅阿涅丝。无父母、无大姑/姨妈/叔伯/堂表兄妹。养父母关系/状态待确认。
- **可提及的童年场景锚点**：孤儿院、阿涅丝、养父母家（5-16 岁被领养期间）

## Why:
