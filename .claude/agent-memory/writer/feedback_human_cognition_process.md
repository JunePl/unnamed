# 让角色像人类一样认知陌生事物

**Why（用户原话）**："还是不像人类在这里的反应啊，比如为什么就突然知道那个是蛛丝，知道会死，等等？"

AI 默认会让角色"立刻知道"陌生事物——直接命名、直接判断、直接对接到已有概念。这是**作者上帝视角硬塞给角色**，违反了人类的认知顺序，让角色感觉像 NPC 在念出剧情说明而不是在经历这件事。

---

## A. 铁律：禁止"立刻知道"，必须走完认知过程

### A.1 禁止的句式

| 错误（作者上帝视角立刻命名） | 修正（角色认知过程） |
|---|---|
| "头顶不是黑暗，是织得密密的白丝" | "头顶上有什么白白的东西，从看不见的高处一缕一缕挂下来" → 慢半拍 → "……丝？" |
| "他知道：再走一步，他会死" | "左腿一软差点跪下去。胃翻了一下。心跳乱了节奏。" → 身体先停了 → "再走一步，他会出大事" → "不是出大事。是会完。" → "死这个词阿尔温平时不会立刻往那儿想" |
| "再看就不对了——女神腰后还多了六只细足" | "第一眼，是个女神像。挺正常。" → "第二眼，他觉得女神腰后那一截轮廓不太对。" → "他眨了眨眼，再看。" → "那不是装饰。" → "是腿。" |
| "她对印记的熟悉度跟主人差不多"（直接给结论） | "指的不是手背正中，是离印记最近的那只细足尖" → "那个位置，光看见印记是指不到的" → "得熟悉到不用看就知道印记在哪儿，才能那么准" → 自然推论："熟悉到这种程度的，全世界大概只有一个人。或者说，一只蜘蛛。" |

### A.2 人类认知的四拍

每次让角色识别/判断陌生事物，都必须走完这四拍：

1. **眼睛/身体先感知物理特征**（白色的、一缕一缕、腿软、胃翻、心跳乱）
2. **大脑慢半拍反应**（"什么？" / "等等" / 身体先停了）
3. **猜测/疑问/对比**（"……丝？" / "他会出大事？" / "那不是装饰" → "是腿"）
4. **才得出结论或继续怀疑**

**禁止跳拍**。跳拍 = AI 上帝视角 = 不像人类。

### A.3 角色"已经知道"的事可以快速反应

如果角色基于前文已经掌握的设定/经验，他可以快速反应——但反应过程仍要"慢半拍"：

- ❌ "右手手背热了一下。蛛印。"（立刻命名）
- ✅ "右手手背烫了一下。他低头瞄了一眼。蛛印亮起来了，是他二阶那天印上去的那个。"（看 → 反应 → 命名+回忆）

## B. 铁律：比喻必须从角色生活经验取材

**Why（用户原话）**："手工活是什么？完全可以写就像是阿涅丝在做玩偶"

抽象/通用的比喻（"像谁在做手工活"、"像棉花"、"像红色小蜡烛"）让读者出戏——感觉不是这个角色在比喻，而是叙述者在套修辞。

### B.1 修复策略

每个比喻必须问：**这个比喻是这个具体角色会用的吗？**

判定方法：
- 这个比喻里的"参照物"在该角色的生活记忆库里出现过吗？
- 如果换个角色（比如换成另一个执法局调查员），这个比喻还会一模一样吗？

### B.2 阿尔温专用的"生活记忆库"（写他的视角时优先调用）

| 类别 | 可调用的具体画面 |
|---|---|
| **家庭日常** | 阿涅丝缝玩偶（前文 178：缝了一半的布偶/毛线团/小剪刀）/ 阿涅丝织毛线 / 阿涅丝喝热汤脸红 / 阿涅丝早晨做甜甜圈 |
| **办公室/工作** | 那张永远歪一边的办公桌 / 沙发/天花板水迹（4756）/ 桌上前一晚剩的咖啡 / 灵痕分析仪 |
| **执法局** | 新人开学第一周教官教的鞠躬动作 / 审讯流程 / 存档室/霍尔办公室门口的"不舒服"感 |
| **童年/学习** | 老博物馆里摸过的骨片 / 卡萨林皇家学院 |
| **街区/小区** | 早点铺的炉子声 / 巡逻车的灯 / 提耶林咖啡馆 |

### B.3 修复对照表（实战案例）

| 抽象/通用（错） | 阿尔温记忆库（对） |
|---|---|
| "像谁拿着针在穹顶上慢慢钩花" | "像阿涅丝那只缝了一半的布偶上拉下来的线" |
| "踩上去像踩在棉花上" | "脚下的白丝格外的软，每一个动作都让丝织物陷下去一点又弹回来" |
| "跟桌子一样具体" | "跟他办公室那张永远歪一边的桌子一样具体" |
| "练过几千次"（鞠躬动作） | "执法局新人开学第一周见过这个动作，教官就是这么教学员的" |
| "肉眼几乎看不清纺线在哪条上来回" | "他想起阿涅丝织毛线时手指快到他根本看不清，但这个比那还快" |
| "打磨得非常细的骨片"（凭空比喻） | "硬，但比纸硬。摸起来更像骨头被打磨得很细之后的感觉。他小时候在博物馆里摸过类似的东西。" |
| "眼睛里居然有点期待"（抽象） | "像阿涅丝早晨给他做了甜甜圈，等他第一口反应时的样子" |
| "像有人在她皮肤底下点了一根均匀的红色小蜡烛"（文艺意象，没解释为什么阿尔温会用这个比喻） | "阿涅丝喝热汤后脸也会这样红一下，但阿涅丝的不会铺得这么均匀" |

## C. 综合应用：写陌生场景的标准流程

写**任何**角色第一次进入陌生环境/遇到陌生事物的场景，按这个清单逐项过：

1. **每个"识别"动作都拆四拍**：物理特征 → 慢半拍 → 猜测 → 结论
2. **每个比喻都要 grep 角色生活记忆库**：参照物在前文出现过吗？换个角色还能用这个比喻吗？
3. **不写"他知道是 X"**：写"他看见了什么 → 想到了什么 → 觉得可能是 X"
4. **危险感不直接命名"死/危险"**：先写身体反应（腿软/胃翻/心跳乱），再让角色慢慢逼近"死"这个词
5. **新名词第一次出现不立刻命名**：用具体描述+疑问号，让角色自己慢慢得出结论

## D. 自检命令

写完任何"角色第一次接触异界/陌生事物"的段落后，必须扫描：

```bash
# 查找"立刻知道"句式（必须 0 命中）
grep -nE "立刻知道|他知道是.{0,4}。|就是X。" /tmp/check.txt

# 查找抽象/通用比喻（每处必须能解释"这个比喻为什么这个角色会用"）
grep -nE "像谁|像有人|某种" /tmp/check.txt

# 查找"那种 Y"贴标签（必须 0）
grep -nE ".的那种|.的那样|.的那种 Y" /tmp/check.txt

# 检查比喻是否对接到角色生活记忆
grep -nE "像阿涅丝|像他办公室|像执法局|他想起" /tmp/check.txt
# 如果上面这条命中数远少于"像谁/像有人/像某种"，说明比喻没对接生活
```
