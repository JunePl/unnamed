# AI 句式拐杖全清单（写完每段必须 grep 一遍）

**Why（统一根因）：** AI 默认偏好"看起来精致 / 高效传递信息 / 有节奏感"。这三种偏好在中文小说里全部表现为同一类病——句子被压成可识别的"AI 默认形状"。每一条拐杖都是必须主动对抗的训练偏差。判定标准是"看着像人类在写的小说"，不是"看起来文笔好"。与 [[feedback_98_percent_process]] 协同——这条管"该删什么"，那条管"该怎么找参考"。

---

## 1. "不是 X，是 Y" 否定再肯定结构

- **限额**：一个自然段内最多 1 次。
- **替代**：直接描述感受/状态，不需要先否定再肯定。
- **自检**：写完 grep "不是"，超过 1 次/段就改。
- **典型踩雷**：200 字样本里出现过 4 次。

## 2. 破折号 "——" 滥用

- **限额**：200 字内最多 1 个。
- **替代**：逗号自然连接 / 句号分开 / 直接省略补充。
- **保留场合**：真正的戏剧性打断或语气急转。
- **自检**：grep "——"，超过就改。

## 3. "X 的那种 Y" / "X 似的 Y" / "X 般的 Y" / "X 一样的 Y" 贴标签式概括

- **必须 0 命中**。
- **错例**：
  - ❌ "被弄痛了的那种喊"
  - ❌ "刀锋似的眼神" / "刀锋般的目光"
  - ❌ "火烧一样的疼"
  - ❌ "听起来像在哭的笑"
- **正例**：
  - ✅ "后面那一声尖叫是琳莎的。"（删判断让画面揭示）
  - ✅ "那个笑没有什么力气，嘴角只是抖了抖，但阿尔温看清了。"（动作呈现）
- **鉴别问题**：这是作者在给读者归类，还是读者能从动作/事实里自己推断？前者删。

## 4. 文艺比喻 / 文学意象 / 情绪状态概括（用在角色内心独白时）

- **必须 0 命中**。
- **错例**：
  - ❌ "就好比你知道考试没考好，但拿到卷子看到分数的那一下还是会难受"（大众比喻，谁都会这么想）
  - ❌ "觉得自己有点像在往大海里扔石头，知道溅不起什么水花"（叙述者修辞）
  - ❌ "是那种发现自己站在一个比想象中深得多的洞口边缘的感觉"（文学意象）
  - ❌ "连愤怒都被磨平了，剩下的只是一种'又来了'的疲倦"（叙述者解说情绪）
- **正例（阿尔温式分析认知）**：
  - ✅ "'隐隐觉得不行'跟'被人当面驳回来'是两回事，前者是推测，后者是盖了章的结论"
  - ✅ "前后不到二十分钟，平均每条驳回理由不到七分钟，算上翻页的时间"
  - ✅ "留个底……日期在上面，内容在上面，谁驳的也在上面"
- **铁律**：角色情绪越重，认知指纹越要突出。阿尔温被打击 = 算数字、列条目、做因果分析，不打比方。
- **自检**：这句话换成任何角色都能说吗？如果是，就不是这个角色的声音。
- 与 [[feedback_alwin_persona]] / [[feedback_inner_voice]] 协同。

## 5. 拟人化怪物 / 拟人物理动作

- **必须 0 命中**。
- **错例**：
  - ❌ "它认得清 X"（怪物拟人）
  - ❌ "火吃掉蜡纸"（物理腐蚀拟人）
  - ❌ "黑暗往这边挪了挪"（环境拟人）
- **替代**：功能性描述。"暗了一点"。

## 6. 自造词级表达（每个动词必须能在参考里 grep 到）

**核心铁律：每一个动词、每一个修饰副词、每一个句式骨架，都必须能在参考材料里 grep 到对应。0 命中 = 自造，必须替换。**

### 6.1 数值变化类（AI 用力让数字"动起来"的痕迹）

| 自造 | 替换 |
|---|---|
| 跳到 / 跳回 / 跳起来（数值） | 补到 / 重新补到 |
| 跌到（数值下降） | 掉到 |
| 涨 / 涨到 / 涌 / 涌出 / 窜 / 窜到 | 朴素实务动词 |

### 6.2 修饰副词加"地"

| 自造 | 替换 |
|---|---|
| 坚韧地（修饰光） | 删 |
| 稳稳地 | 稳稳 / 删 |

### 6.3 战术游戏术语（AI 把战斗写成 RTS 指令）

| 自造 | 替换 |
|---|---|
| 再切（战术换位术语） | 再走 / 再挪 |
| 扩大锁定（瞳孔） | 扩大盯住了 |
| 借的是动静（独立短句） | 删独段 |

### 6.4 文艺意象转场

| 自造 | 替换 |
|---|---|
| 剥到最里层 | 删 / 改硬切 |
| 从画面里退出去 | 删 |
| 褪了一层颜色，露出底下另一件 | 忽然变了 |
| 承压点先一步崩 | 撑不住 / 顶不住 |
| 这一秒变了 | 忽然变了 |

### 6.5 拟人物理动作 / 省略形式

| 自造 | 替换 |
|---|---|
| 吃掉（火吃掉蜡纸） | 烧掉 |
| 挂歪（动词） | 朴素描述 |
| 挂在墙上吊着 | 挂在墙上 |
| 目光锁着 | 目光盯着 |
| 在闪（数字闪烁） | 在闪烁 |
| X 熟（"这种声音他熟"） | 他听过 / 他很熟悉 |

### 6.6 其他 AI 文绉腔

| 自造 | 替换 |
|---|---|
| 终极于此 | 到此为止 |
| 一格一格往下掉 | 一点一点往下掉 |
| 挑词（说话停顿） | 才说 |
| 挑挑眉（重叠） | 挑了挑眉 |
| 没顾上 | 顾不上 |
| 抠了一下（口袋） | 摸了一下 |
| 倒抽一口气 | 倒吸一口凉气 |
| 甜腻味（单字"味"作名词） | 甜腻味道 |
| 这一线（机会） | 这个机会 |
| 腰腹的力 | 腰腹用力 |
| 唇形（读唇） | 口形 |
| 收了一收（瞳孔） | 缩了一下 |
| 颜色发着烫白 | 颜色烫白 |

### 6.7 句式骨架黑名单（参考里没有的组合）

| 自造骨架 | 参考存在的骨架 |
|---|---|
| "X 在 Y 上紧了一下" | "握紧了 Y" / "紧了紧 + 衣物" |
| "凉里带着一种 X" | OK（"X 里带着 Y" 在参考里多次出现） |
| "声音笑了一下，像在某个无形的喉咙里震了一下又散开" | 删比喻，只留 "声音笑了一下" |
| "他的视野黑了，像油画颜料被一层层剥下来" | 删比喻，硬切转场 |

## 7. 现代词污染世界观（蒸汽灵能世界禁用）

- **必须 0 命中**。
- **错例**：投影仪、相机、电脑、肌肉记忆、数据、模式、统计学。
- **鉴别**：这个世界有这个东西吗？无 → 换该世界存在的物件（画、水、蜡、火、玻璃、布、陶瓷）。
- **修复样例**："投影仪打出来的画面被雾吞掉" → "一张泡在水里的旧画，颜料往四面化开"。

## 8. 精简短句独段（必须有戏剧功能否则合并）

**鉴别测试**：每个独段短句都过这道筛——

- ✅ 战术判断锚点：priest "是鞋——这里没人打扫"
- ✅ 反诘吐槽：听日 "石头来的啊！"
- ✅ 实务落点：澎湃 "系统消失。"
- ✅ 识别瞬间：听日 "是的，记忆！"
- ✅ 钩子重复：角色内心重复关键信息
- ❌ "X 没 Y"模板（"它没回过头" / "他没等" / "那个人没回头"）
- ❌ 数值变化独段（"灵能数掉到十八" → 必须并入下文）
- ❌ 段首三字主谓（"灵能在掉" → 并入下文长句）
- ❌ 决断短句（"不行。" → "不行，..."）

## 9. 省略主语（电报体）

- **禁用程度**：硬性规则，不是风格选项。
- **唯一例外**：紧接上一句、主语完全无歧义的同一人连续动作（如"他站起来，走到窗边"）。
- **限额**：即使是高冲击段落也最多连续 1-2 个短句省主语后必须恢复。
- **自检**：逐句确认读者能明确知道"谁"在做动作。
- **参考验证**：听日 / 澎湃 / 用户自己已写章节全部保留主语。

## 10. 拟人 / 精致比喻 / 纯氛围句（去文学化）

让读者从故事中跳出来意识到"有人在写作"的，全部删。

| 文学化（错） | 去文学化（对） |
|---|---|
| "黑暗往这边挪了挪" | "暗了一点" |
| "语气像在念一行无关紧要的备注" | "语气随意" |
| "苦涩的味道荡漾在口腔" | "苦，带点橄榄的咸" |
| "被特意烫得温热而不烫手" | "盘子是温的" |
| "消融在夜色里" | "走出几步就看不见了" |
| 纯氛围句（不推进剧情/角色/信息） | 直接删 |

**保留不动**：
- 冷幽默："但尸体不在乎这种气氛"
- 短句鼓点（有戏剧功能时）："惊讶。""杰作。""走了。"

## 11. 自己 spoil 自己的 punchline

如果后面要有一句锋利的 punchline（"不是 X。是 Y。"），叙述者前面绝对不能用同一个否定词预告。

- **错**：叙述者写 "画面在阿尔温眼前闪了一下。**不是预见**。" → 几秒后阿尔温说 "这个能力**不是预见**。是考古。"
- **对**：叙述者写 "**不是往前，是往后**。"（方向词，不涉及预见 vs 考古的二元对立）

## 12. Bullet list / 表格化的"角色读文献"

bullet list 是维基百科格式，不是小说。角色读书必须展开为阅读动作——翻页、停顿、笔尖跟着。关键数字放在"停下"的那个瞬间。

- **错**："【渊秘源流·草原记录】- 数量不少... - 占比四成"
- **对**："K 这本书的渊秘源流那一章是写在第三部分的... 再翻两页，阿尔温停下了。K 在一个小脚注里写：科尔曼汗国的骑士议会里，这一系的灵使占比据说高达四成。"

## 13. 叙述者替读者做笔记

所有"某个细节的变化"必须通过角色内心的对比动作传递，不是叙述者直接解释"这说明了 X"。让读者自己完成那个推论。

- **错**："'埃德蒙'。不是'单镜先生'。阿尔温注意到了这个称呼的变化。上次格里尔叫他'单镜先生'的时候是带着挑衅的，这次直接叫他姓，是公务语气。"
- **对**：通过阿尔温起身时的动作或想法对比上次场景，自然带出。

## 14. 配角口中不能出现专业术语

配角对白必须通过"这个人的生活经验语料库"过滤。

- **错（米娅，普通平民）**："阿克曼症患者的家属有更高的灵质异常风险。医生跟我说过这件事。我当时问医生'更高'是多高，他说'没有数据，只是统计学上的关联'。"
- **对**："他说没数据，只是他们看的病人里碰巧有这种情况。"

## 写完每段的强制 grep 自检命令

```bash
# 切出本段到临时文件
awk 'NR>=START && NR<=END' "/path/to/正文.novel" > /tmp/check.txt

# 黑名单关键词扫描（应全部 0 命中）
for w in "跳到\|跳回\|跌到" "再切" "坚韧地" "终极于此" "一格一格" "剥到最里" \
         "挂歪" "扩大锁定" "唇形" "收了一收" "甜腻味$" \
         "这一线" "腰腹的力" "倒抽一口气" "借的是动静" "目光锁着" \
         "X熟$" "在闪$" "抠了一下" "这一秒变了" "稳稳地" \
         "脊背发凉" "后背发凉" "呼吸停了" "瞳孔一缩" "心跳漏了" \
         "愣在原地" "喉咙发紧" "投影仪" "肌肉记忆" "统计学"; do
    c=$(grep -c "$w" /tmp/check.txt 2>/dev/null)
    [ "$c" != "0" ] && echo "残留: $w -> $c 次"
done

# 数量化指标
echo "破折号 ——: $(grep -o '——' /tmp/check.txt | wc -l)（200字内≤1）"
echo "不是X，是Y: $(grep -oE '不是[^，。]+，是' /tmp/check.txt | wc -l)（一段≤1）"
echo "'X的那种' 次数: $(grep -oE '.的那种' /tmp/check.txt | wc -l)（必须 0）"
```

## How to apply

1. 写完任何段落，**必须**跑上面的 grep 命令。
2. 任何命中 → 立即替换。
3. 0 命中才能提交给用户。提交时附上 grep 输出。
4. 不跑 grep 直接交稿 = 违反 [[feedback_98_percent_process]]。
