## 米娅·斯费尔康

- 说"我不想死"时是真心的（对旧识的信任+发作后的脆弱）
- 但她被打断前想说的是：**她不想活了，因为她觉得自己是艾伦的累赘**
- "不想死"和"不想活"的矛盾是她的核心内心冲突
- 阿克曼症发作时她会变得极具攻击性（幻想与恶魔/神战斗，试图杀死周围的人），不是温和的幻觉

**Why:** 这个矛盾是她角色深度的来源。后续如果米娅再出场，她的言行都应该在这两极之间挣扎。

## 艾伦·斯费尔康

- 对阿尔温产生初步信任的原因：(1) 阿尔温帮助了生病的孩子 (2) 米娅认识阿尔温
- 但他的信任有**硬性上限**——幕后人不会让他透露更多
- "其实那家伙死的那天。。。"是他的极限，身体发作（手抖）是幕后人的控制机制在起作用
- 他说"人不是我杀的"是直球性格（闷但不绕弯），也是信任的表现

**How to apply:** 后续艾伦的信息释放必须有"被阻止"的物理表现（手抖、身体异常），不是他自己选择不说。
