description: 每一拍动作必须有角色意图+阻碍+情绪化反应，不能写成干净利落的"工作流汇报"。AI默认"X没回头，先打掉Y"是冷工作流，必须改成"X想要回头，但Y扑过来，X迫不得已先打掉Y"

用户在2026-04-30明确示范了正确的动作场景写法：

> "阿尔温想要立刻转身，但傀儡却扑了上来，整个脸变成一个巨大的口器，阿尔温迫不得以只能连续两枪将它解决。随着淡蓝色的火光炸起，阿尔温转头向后跑去。"

**Why:** AI默认把动作场景写成"信息汇报"——主语+动作+结果，干净利落但缺血。人类小说写动作的核心是**角色挣扎**：他想做X，但被Y阻碍了，所以不得不做Z。每一拍都有意图、阻碍、反应。**没有挣扎就没有紧张感，读者只看到事件不看到人。**

**对比示范：**

❌ AI冷工作流（用户判定70%以下）：
> "他没回头，先把最近的那具打掉。子弹打在它的太阳穴上，它倒下去之后阿尔温才转身。"

问题：每个动作都精打细算，没有挣扎，没有恐怖视觉，没有情绪化动词。读起来像警情汇报。

✅ 人类小说（用户认可）：
> "阿尔温想要立刻回头，后面那一声尖叫是琳莎的。但他左边那三具傀儡已经扑了上来，最前一具的整张脸裂成了一个巨大的口器，下颌脱钩张到胸口位置，里面密密麻麻是变形的牙齿。阿尔温迫不得已先把枪口扣下，连开两枪，子弹精准地钻入两个灰白色的光点，傀儡随着闷响炸成一摊污泥，淡蓝色的火光跟着炸开。"

关键差别：意图（想要立刻回头）+ 阻碍（但傀儡扑上来）+ 视觉冲击（口器/下颌脱钩/变形牙齿）+ 情绪化动词（迫不得已）+ 视觉效果（淡蓝色火光炸开）。

**每一拍动作的5个必备要素：**

1. **角色意图**——他此刻想做什么
   - "阿尔温想要立刻回头"
   - "他想抽出剪刀再扎"

2. **阻碍**——什么阻止了他
   - "但他左边那三具傀儡已经扑了上来"
   - "可剪刀还卡在那只眼眶里"

3. **恐怖/具体的视觉细节**——阻碍是什么样子
   - "整张脸裂成口器"
   - "下颌脱钩张到胸口位置"
   - "黑色的舌头从脸的中央伸出来"

4. **情绪化动词**——他被迫做出反应时带的情绪
   - 迫不得已、不得不、只能
   - 咬牙、狠狠、猛地、直接、立刻
   - 几乎没瞄就、来不及、还没X就
   - 一把、一下子、瞬间

5. **视觉/感官效果**——动作的真实质感
   - "淡蓝色的火光跟着炸开"
   - "扳机扣到机括发硬"
   - "咔的一声咬穿了肩骨"

**视觉冲击的强度梯度：**
- 普通战斗：直接的弹道+部位（"子弹打在太阳穴上"）
- 高强度暴力：身体异变（"整张脸裂成口器/两瓣"、"舌头从脸中央伸出来"）
- 极端暴力：身体损伤的物理细节（"整条右臂被它从根部直接咬掉"、"咬到米尔顿的整个左半身都没了"）

**How to apply:**
- 写动作场景前先列出每一拍的"想要X / 但Y / 所以Z"——这是骨架
- 每一拍至少出现：1个情绪化动词 + 1个具体视觉细节
- 写完后扫描：是否每一拍都只有"主语+动作+结果"无挣扎？如果是就重写
- 用户说"完全没写出情绪和紧张感"几乎100%是这个问题
- 与`feedback_sentence_rhythm.md`协同：基础句式是连贯长句的事件流，不是短句堆叠
- 与`feedback_no_data_report_style.md`协同：动作要有质感和过程，不是结果列表
