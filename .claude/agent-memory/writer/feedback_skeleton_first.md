# 递归骨架填空法（写作主方法）

## 一句话核心

**成稿里每一个句子、每一个短语、每一个词，都是从参考里 grep 到的真实母句"填空换名词"而来，没有任何一处是自己组织/拼接出来的。** 骨架是**递归**的：大骨架(一整段连续母段)→ 小骨架(每个小句的母句)→ 小小骨架 → 词。

**🟥🟥🟥 最高一条（2026-06 第20章反复写废后确立·高于下面所有铁律）：大骨架必须是「一整段连续的、来自同一处」的真实母段——整段搬过来，只换专名、让剧情迁就它的结构，一个字都不自己造。**
- ❌ **东拼西凑贴标签（头号错误）**：从 6 个不同出处各捡一个短语（喘口气←玩乐 / 绝口不提←魔战 / 独自承担←魔战…）拼成一段——每句的结构、句与句怎么连，全是自己写的。这不是骨架，是贴皮。
- ❌ **先列叙事拍点 / 节奏蓝图再找词填**：一旦先写"这段写什么、怎么推进"，句子结构就已自创。**跳过拍点，唯一动作＝先锁定一整段连续母段，让它决定写什么、怎么推进。**
- ✅ **唯一合格形态**：找到一段同一处的连续母段（如 魔战1158–1168、终局7808–7832、术师48868–48873），整段逐句套，给用户看"母段原文 vs 成稿"逐句对照。

**Why（我反复犯的三种自创，全是 AI 味来源）：**
1. 凭感觉写句子，事后给词贴出处标签 → "把词拼成句"的句式是自创的。
2. 找到大骨架后直接拿词去填 → 跳过了"每句要有母句"的小骨架层。
3. 用近义词/同词根冒充出处（"没停"借"没停下"）→ 确切组合根本不存在。
4. **找完骨架后凭语感先把成稿写出来、再 grep 验证（2026-06 科尔段·用户点破的头号操作错误）** → 必然大量 0 命中、必然东拼。"填不出词"是伪命题：只要是从 grep 命中里捞词，就不可能 0 命中；一旦 0 命中，就是你又"先自己写"了。**正确操作：骨架定下后，每个空位先回九本 `grep -hoE ".{0,4}关键词"` 捞出现成搭配，从命中结果里挑词填**——词和短语都是"捞"来的不是"想"出来的。连两字搭配（"跑腿打杂""挨骂受气"）也要 grep 到现成的，把两个单字词自己拼起来＝照样是自创。

## 🟥🟥🟥 S 级三铁律（最高优先级，一条都不能破）

1. **一个小句 = 一个母句的填空版，不准缝合两个母句。**
   母句「回身架起步枪」到此完整，就不许自己接上另一句的「扣下扳机」凑成「回身架起步枪，扣下扳机」——"把母句 A+母句 B 缝成一句"这个拼接动作本身就是自创。要的动作链必须找一个**本身就完整包含该动作链的母句**；找不到就把内容拆成多个独立小句，每句各对一个真实母句（由大骨架决定顺序，这不算缝合）。

2. **不准用孤立的词拼出没有母句结构的句子。**
   「那一枪落了空，对着的街角空无一人」＝把"落空""空无一人"两个词硬拼，**这个句子结构没有母句**，禁止。每个句子的结构都必须整体落在一个真实母句上。

3. **每一个确切词组合都要 grep 到，近义/同词根不算。**
   写「他没停」却拿「也没停下」当出处＝偷换，"没停"这个确切字符串参考里不存在。**成稿里每个 2–4 字短语都要 grep 到那个确切组合**；"没停下"搜得到 ≠"没停"能写。"那一枪""这一枪"实测三本皆无 → 当场弃用，不许凑。换进去的名词（约纳斯/铳/巷口）也要各自 grep 到。

**落地做法（贯穿三条）：母句决定成稿，不是成稿找词去凑。** 先看参考里某类动作/情绪的母句长什么样，**再让本作内容去迁就母句**。

**🟥🟥🟥 整句套用铁律（最根本·高于一切·2026-06 拍 G 被抓）：成稿 = 一条完整母句，原样保留它的全部结构、长度、标点、所有非专名的字，只换里面的专有名词（人名/物名/地名）。** 三种"借壳"破坏骨架的做法，一个都不许：
- **不许截取母句的一部分**：母句「她融入人群里继续寻找虚实素材」不能只挖"融入人群"丢掉后半再自己接——那是借壳。要么整条用（连"继续寻找…"一起换专名），要么换一条本就到此为止的母句。
- **不许加减/改动标点、改动结构**：母句「扶着剑袋大口喘气」不能改成「扶着墙，大口喘气」（加逗号、节奏变了；"剑袋"握手里≠"墙"靠着，动作也变了）。
- **不许把母句的词组拆出来重组成新句**：母句「变到再也回不来的地方」（"再也回不来"是修饰"地方"的定语）不能拆出"再也回不来"重组成「人要是没了，就再也回不来了」——句子结构是自创的。

**判定（用户原话）：你想写的意思，参考里找不到一条结构/长度都吻合、只需换专名的完整整句 → 说明"这个写法太烂、没人会这么写"，放弃它，改用参考里真实有整句的表达。** 不是去截取/改标点/拆词把它硬凑出来。

**✅ 允许的调整（2026-06 用户示范·别把铁律教条化到瘫痪）：在保留母句完整句式结构/标点/节奏的前提下，可以——换专名（人/物/地）、换人称代词、为通顺做近义动词微调。** 用户范例：术师15690「剑姬魔女**看见**石头在**观者**手里忽然化成齑粉，顺着他的指间滑落下来」→「雷纳德**看着**手环在**自己**手里忽然化成齑粉，顺着他的指间滑落下来」（剑姬魔女→雷纳德、看见→看着、石头→手环、观者→自己；整句结构一字未动，也没截取/改标点）。**判别线：结构完整保留=合格；截半句/改标点节奏/拆词重组=破坏。** 之前我把"去掉'剑姬魔女看见'"误判成截取而瘫痪，是想错了——换主语+近义动词不是截取。

**🟥 冰山暗流的写法尺度（2026-06 阿涅丝日常章·用户逐句调教）：** 角色间的复杂情绪（暗恋、落选的苦、对练落败、在意某人）只能极微暗示，敏感读者隐约觉出、普通读者只当日常：
- **暗流藏在「台词的潜台词」里**：薇薇安去不了王都的苦＝让她说"去王都**找你们玩**、带我挨家吃甜品"（潜台词：没考上、只能去玩），台词表面是贪甜品的玩笑；卢卡斯被拒＝一句"我过两天去沃伦了"。
- **叙述只写正常表面反应，绝不点破掩饰**：写"薇薇安笑了"可以；写"'啧'得有点用力，很快又笑了""掩饰着笑"＝直接点破她在掩饰＝禁。一个极微表情（"脸上的笑淡了一下"）可以，但不许解释它为什么淡。
- **绝不为一条暗流铺一整段、绝不用多个动作叠加强调**。一条暗流＝一句台词的潜台词，或一个动作（"自然地挽住阿涅丝的胳膊"＝暗恋），到此为止。

**此前已抓、仍盯死的两个特例：**
- **不准为塞剧情强改母句的词**：母句「静静地放下手机」就写"放下手环"，别为塞"记住脸"改成"记下"。
- **动作/画面流水最易退化成孤立词拼**："弯腰捡起""左右两边各站着一个"没整句母句＝禁止；群像找「穿着体育服站成一排」这种真实整句。

**🟥🟥 递归退化高发区（2026 第22章·同一错犯两回被用户抓）：推理串联 / 侦探分析 / 对话叙述 / 专业描述（节律感比对这种）——这些"参考里没有意思相同的母句"的段落，最容易偷懒自己造句式（当时前三块严格、从块四起全退化成"先想好句子再贴词"）。**
- **病根**：以为"轻小说里没有侦探串联线索的内容"就只能自创 → 错。**同构指修辞/情绪类型，不是内容**：推理串联去找「本来以为是X，结果越想越觉得Y」(世界末日65747)、「我忽然想起X」(术师8044)、「都被完美地串连在了一起」、「会不会也有…呢」、「无论如何都想弄清楚」(药师)、「只有一个人」——套这些句式结构，换成本作推理内容。
- **典型自创（都要改）**："一条看不见的线，慢慢在他脑子里连了起来"、"要验证这条线，只有一个地方"、"各归各的位，自己跟自己对得上"、"傲娇也好公事公办也好至少她会来" → 全部换成套句式母句。
- **自查**：写完推理/对话/描述段，随机抽 5 句，每句必须指得出它套的是哪条母句(文件行号)；指不出＝自创＝改。**别等用户抓，写的时候就该警惕这几类段落。**
- **另一个偷懒**：backup/暂存里用"(此段与前次一致，沿用)"括注省略正文＝禁，必须写完整全文。

## 递归操作流程（每一段都走一遍）

1. **定效果**：这段要达到什么（紧张/吐槽/落差/悲凉/神秘/克制…）。
2. **找大骨架**：去九本参考找一个**达到相同效果的真实连续母段**，它决定这段由哪些小句、按什么情绪推进与组合顺序构成。没有大骨架 = 不能写。
3. **逐小句找小骨架**：大骨架每个小句位置，去参考找一个**整句结构都能套的真实母句**。只换专有名词，其余句式/动词/连接词/副词一字不动。
4. **递归下钻**：某个小骨架不足以表达 → 把它当新骨架，在内部继续找小小骨架、小小小骨架，直到触底为词。
5. **逐组合 grep -Hn 复核**：成稿每个短语回 grep（带文件名），确认确切组合存在、且语域匹配（见配套·语用）。任一处 grep 不到 → 改写迁就母句，不许保留。
6. **画嵌套树给用户**：`【大骨架←出处【小骨架=母句←文件:行号【词←文件:行号】】】`，证明每一层（含大骨架的拼接逻辑）都有出处、无一自创。

**母段"同构"指修辞/情绪类型同构，不是剧情场景同构**：杀手死巷在参考里没有同场景母段，但"被逼绝境""猝死瞬间""临终念头"等情绪/修辞类型的母段到处都有，套其骨架即可。

## ⛔ grep 铁律（出过血的）

- **必须带文件名 `grep -Hn`，出处书名+行号直接从终端输出抄，禁止凭印象分配。** 用 `grep -h`（合并多本不显示文件名）再靠记忆分书名 = 必然张冠李戴（实测把终局 27646 标成"玩乐"、术师"孰轻孰重"标成"终局"，用户按错书名查→"根本没有"→判定全是假骨架）。展示给用户的每条出处都要能让他复制 grep 当场复现。
- **"或"用 `|` 不是 `\|`**：写 `\|` 被当字面竖线，造成大量假 0 命中。
- **记忆/summary 里的旧母段用前必 `grep -Hn` 复验**：如"术师 L87943 眨眼消失"在当前参考池查无此句，查不到立即作废重找。
- 标准参考池变量：`R="Project/Reference/终局委员会.txt Project/Reference/玩乐关系.txt Project/Reference/术师手册_utf8.txt Project/Reference/弑神者.txt Project/Reference/骑士之潭.txt Project/Reference/魔战公主.txt Project/Reference/天使大人.txt Project/Reference/药师少女的独语.txt Project/Reference/世界末日的世界录.txt"`（2026-06 扩到九本）
- 看片段真实上下文：`grep -hoE "[^，。」「]{0,6}片段[^，。」「]{0,9}" $R`

## 范例

**① 亚修填空式（段落骨架）**
母段（术师）：「可笑的是亚修刚才还以为这是自己穿越外挂，没想到是人手一份的大众货。简直就是古代人穿越到现代，捡到一部手机以为是神器。」
骨架：【前后落差自嘲】可笑的是 X 还以为 A，没想到是 B + 【夸张类比收尾】简直就是 C。
套用：阿尔温本来还以为捡到了免费大便宜，没想到是这种鸡肋能力。简直就是【阿尔温记忆库里的好笑比喻】。（"还以为→没想到是→简直就是"一字不动，只填实义词。）

**② 小句级一句一母句（第18章拍 D 开枪）**
- 母句①回身 ← 玩乐 5538「番长**回过身来**…」｜母句②开枪 ← 终局 10022「我**朝着**附近的石头**扣下扳机**。…**枪响『砰！』的一声炸开**。」
- 套写：「约纳斯**回过身**，**朝身后扣下扳机**，**枪响砰的一声炸开**。」
- 确切组合复核：回过身←玩乐5538｜朝身后←终局38852｜扣下扳机←终局10022｜枪响/一声炸开←终局10022｜砰←玩乐4186。两个独立小句顺接，无单句内缝合。

**③ 发现无出处当场弃用（第18章拍 D 扑空）**
想写"那一枪落了个空"→ grep「那一枪」「这一枪」三本皆无 → 弃用 → 改套终局 520「周围空无一物」整句填空 → 「他身后空无一物」（周围→他身后；身后←玩乐1384、空无一物←终局520）。

## 配套要点（骨架对了，缺这些照样废）

1. **参考池用九本，停用一世之尊。** 终局委员会(组织/盯梢/死亡/枪战)、玩乐关系(日常/吐槽/喜剧)、术师手册(战斗/危险/对峙)、**弑神者**(神话/魔王/异能战斗)、**骑士之潭**(校园/骑士决斗)、**魔战公主**(战争/战姬/奇幻)、**天使大人**(日常/恋爱/校园·细腻日常情感与人物观察)、**药师少女的独语**(宫廷/推理/古风·冷静观察+推理·语域偏古雅·防古宫词混入)、**世界末日的世界录**(作者本人另一部作品·末世/奇幻/神秘)，按情绪类型选库（战斗对峙：术师/弑神者/骑士/魔战；日常情感：玩乐/天使大人；推理冷静：终局/药师少女；末世神秘：世界末日）。一世之尊(仙侠白话)会把语感稀释成仙侠味(运起/古井无波/压箱底/催动/灵光)，只可做剧情逻辑参考，**不可借句式**。
2. **实义词要符合目标语感。** 江湖老派/侦探腔词（盯梢、不挪窝、做掉、手脚利索、节外生枝、够本、大活人、按说）在轻小说里 0 命中＝出戏，即使骨架对了也废。
3. **grep 命中 ≠ 语用正确。** 「钉在」有命中但母句是"皮被钉在木框"的字面钉，拿来写"坐着不动"=语域错位。每个片段除"存在"还要看它在母句里的**真实用法/语域**是否匹配。
4. **轻小说流速，不要电报体。** 极限压缩片段会产出冷硬电报体（用户"完全没感觉"）。轻小说流速＝长句呼吸 + 列举铺排（"有…的，有…的，也有…的"）+ 夸张怀疑（"几乎要怀疑自己跟踪的是一件行李"）+ 吐槽织进叙述。母句范例：术师「安静得如同一尊雕塑」、玩乐「在站台上重新开始尾随，以免跟丢」。
5. **"像主角"＝场景+对话+世界交互，不是更多内心/背景。** 背景档案段（"这工作三天前接的…三年都这样"）＝发给死人的简历；解释性顿悟（"约纳斯终于意识到…"）＝配角写法。威胁/背景压缩成一句、钉在决断点上；给角色至少一次世界交互对话（卖报小贩"看过了"）。
6. **从具体细节切入，不概括陈述。** 开篇别用"X 干了十二年、从没见过"式概括，从一个带世界钩子的具体细节切入（造梦仪广告→借反应立人性→把职业藏进动作→背景最后才带出）。
7. **战斗/藏设定段：纯客观镜头白描。** 只写发生了什么，不进角色内心、不补来历（能力练多久、代价、骗过谁全删）；实力差靠现象对比（我方倾尽所有 vs 对方毫不费力）让读者自己看出。内心/人格挪到脱险后再放。

## 挑大骨架去哪本书找的辅助表（⚠️ 不是"先写蓝图给用户确认"·那已废弃）

**绝不先列蓝图/拍点再找词填——先列＝先自创结构。** 下表只在 Step 2"去哪本书找一整段连续母段"时心里掂量一下，不写成步骤、不交给用户确认：

| 场景类型 | 去哪本找母段 | 该处母段的语感特征 |
|---|---|---|
| 日常对话+轻小说感 | 玩乐关系（吐槽日常） | 中长句+大量内心吐槽+对话密集 |
| 战斗紧张 | 术师手册（亚修挨打/对峙） | 长句串多动作+挣扎词+具体身体感受 |
| 盯梢/组织/死亡 | 终局委员会 | 冷静观察+现象不解释+压抑长句 |
| 感官冲击 | 术师手册+终局委员会 | 多感官同时+中长句连缀 |
| 内心独白（人性化） | 玩乐关系（吐槽流） | 思绪流+"…啊/…吧"口语词 |
| 牺牲/克制 | 终局委员会（告别/赴死） | 长句铺垫具体动作+关键独段一句话 |
| 上下级/博弈对话 | 终局委员会/术师手册 | 上位者通报+下属应+旁人插话 |
| 黑色幽默 | 玩乐关系 | 反差（严肃场景+口语吐槽）+不解释 |

---

## How to apply

1. 接到写作任务 → 直接进 Step 2 递归骨架：去九本找一整段连续母段做大骨架（⛔ 禁先列蓝图/拍点、⛔ 禁 grep 本作正文找语感）。
2. 走递归操作流程：大骨架→小骨架→词→grep -Hn 复核→画嵌套树。
3. 每段 ≤150 字给用户看，每一步都可被打断。
4. 写完调 `novel-rewrite-verify` 跑独立验证。

## 与 SKILL 流程整合

`novel-rewrite/SKILL.md`：Step 0 问问题 → 1 读参考（⛔禁 grep 本作正文）→ **2 递归骨架（唯一核心·找一整段连续母段整段套·大→小→词·S 级三铁律·grep -Hn 复核+画嵌套树）** → 3 调 verify → 4 修复至 0 违规 → 5 用户确认后写入。**节奏蓝图/叙事拍点/grep 本作正文已彻底废弃删除。**
