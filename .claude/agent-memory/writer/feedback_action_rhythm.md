# 节奏 / 动作 / 信息释放 / 世界观

## A. 句子节奏铁律：短句是调味不是主食

**Why:** AI 天然倾向于用短句堆叠模拟"节奏感"和"力度感"，这是偷懒。真正的节奏是长短交替，**基础是中长句**。

### A.1 文风参考的真实节奏

- **听日**："这不能怪他，毕竟他前一秒的记忆还在那该死的手游公司里通宵到凌晨贡献剩余价值，后一秒就忽然来到这个奇怪的地下大厅里接受众人的膜拜，换成谁都反应不过来。"——一整条连贯长句
- **竹宫**："明明总是小心翼翼将水气擦干，明明上个礼拜才花了一整天在厨房、盥洗室等处举行除霉大会的。但是在这间破房子差劲的透气能力前，不论怎么样的努力都有如泡影。"——中长句连缀
- **不讲理**："虽然我也不知道为什么前世的记忆都还在，不过这样也没有造成什么困扰。"——松弛的中长句

三位作者的内心独白都是**连贯的思绪流**，句子该长就长，碎碎念也是连着说的。

### A.2 规则

- **基础句式 = 中长句**（2-4 句一个自然段，句内用逗号 / 连词连接）
- **短句独立成段 = 极少使用**，仅用于：
  - 单字揭示（D。）
  - 关键对话结尾（没了。）
  - 真正的情绪爆发
- **绝不允许连续 3 个以上的短句独立段落**
- **内心独白 = 连贯思绪流**，一个念头自然接下一个念头，用逗号而非句号 + 换行
- **段落结构**：一段 2-4 句，段间在话题/动作/情绪转换处断开

**自检：** 如果一页里超过 1/3 的段落只有一句话，节奏就出了问题。

### A.3 反例（2026 神殿章踩雷血泪经验）

用户原话："**全是小短句子，有哪个参考这么写的？**"

**致命错误模板**（我写的，违反 A.2）：
> 他喘不过气。胃里翻一下，又翻一下。心被攥住了，攥得很紧。

—— 3 个连续短句独段堆砌，把一组**同时发生的身体反应**拆成了机关枪式独段。

**正确写法**（按用户已发布正文 2994 + 术师手册"喘不过气"65 次长句节奏）：
> 第一步刚踩到雾里，一种死一样的压力从前面压过来，他突然喘不过气，胃里翻了一下又翻一下，心也跟着被一只无形的手攥住，攥得越来越紧。

—— **一个长句串起所有反应**，主语清晰，节奏连贯。**身体反应是同时发生的，不能拆成"动作 1。动作 2。动作 3。"**

### A.4 用户已发布正文里的中长句节奏锚点（grep 必查）

- **正文 2994**（阿尔温被气浪打飞）："阿尔温下意识伸手借住，下一秒三楼倒塌的气浪从前面把他整个抛飞出去，胸口磕在身后一根断了一半的钢筋上。肺里的气一下子被碾空。"——5 个动作长句连缀，最后才一个独段
- **正文 758**（阿尔温自己喘气）："阿尔温推开侧门的时候喘得厉害，从港口街一路跑过来，中间还差点撞到一只送货的机器人。"——喘气嵌在长句中

### A.5 自检命令（写完段落必跑）

```bash
# 检测连续 3+ 短叙述独段（排除对话/卡牌）
awk 'NF == 0 { next }
{
  is_short = (length($0) <= 36 && !match($0, /^[""【]/))
  if (is_short) {
    chain++
    if (chain == 3) { print "▶ 连续 3+ 短独段:" }
    if (chain >= 3) print NR ": " $0
  } else { chain = 0 }
}' /tmp/check.txt
# 输出非空 = 违规，必须合并成中长句
```

## B. 动作场景的"想要 X 但 Y 所以 Z"挣扎结构

**Why:** AI 默认把动作场景写成"信息汇报"——主语+动作+结果，干净利落但缺血。人类小说写动作的核心是**角色挣扎**：他想做 X，但被 Y 阻碍了，所以不得不做 Z。每一拍都有意图、阻碍、反应。**没有挣扎就没有紧张感，读者只看到事件不看到人。**

### B.1 对比示范

❌ **AI 冷工作流（用户判定 70% 以下）：**
> "他没回头，先把最近的那具打掉。子弹打在它的太阳穴上，它倒下去之后阿尔温才转身。"

问题：每个动作都精打细算，没有挣扎，没有恐怖视觉，没有情绪化动词。像警情汇报。

✅ **人类小说（用户认可）：**
> "阿尔温想要立刻回头，后面那一声尖叫是琳莎的。但他左边那三具傀儡已经扑了上来，最前一具的整张脸裂成了一个巨大的口器，下颌脱钩张到胸口位置，里面密密麻麻是变形的牙齿。阿尔温迫不得已先把枪口扣下，连开两枪，子弹精准地钻入两个灰白色的光点，傀儡随着闷响炸成一摊污泥，淡蓝色的火光跟着炸开。"

关键差别：意图（想要立刻回头）+ 阻碍（但傀儡扑上来）+ 视觉冲击（口器/下颌脱钩/变形牙齿）+ 情绪化动词（迫不得已）+ 视觉效果（淡蓝色火光炸开）。

### B.2 每一拍动作的 5 个必备要素

1. **角色意图** —— 他此刻想做什么
   - "阿尔温想要立刻回头"
   - "他想抽出剪刀再扎"

2. **阻碍** —— 什么阻止了他
   - "但他左边那三具傀儡已经扑了上来"
   - "可剪刀还卡在那只眼眶里"

3. **恐怖 / 具体的视觉细节** —— 阻碍是什么样子
   - "整张脸裂成口器"
   - "下颌脱钩张到胸口位置"
   - "黑色的舌头从脸的中央伸出来"

4. **情绪化动词** —— 他被迫做出反应时带的情绪
   - 迫不得已、不得不、只能
   - 咬牙、狠狠、猛地、直接、立刻
   - 几乎没瞄就、来不及、还没 X 就
   - 一把、一下子、瞬间

5. **视觉 / 感官效果** —— 动作的真实质感
   - "淡蓝色的火光跟着炸开"
   - "扳机扣到机括发硬"
   - "咔的一声咬穿了肩骨"

### B.3 视觉冲击的强度梯度

- **普通战斗**：直接的弹道+部位（"子弹打在太阳穴上"）
- **高强度暴力**：身体异变（"整张脸裂成口器/两瓣"、"舌头从脸中央伸出来"）
- **极端暴力**：身体损伤的物理细节（"整条右臂被它从根部直接咬掉"）

### B.4 How to apply

- **写动作场景前先列每一拍的"想要 X / 但 Y / 所以 Z"** —— 这是骨架
- 每一拍至少出现：**1 个情绪化动词 + 1 个具体视觉细节**
- 写完后扫描：是否每一拍都只有"主语+动作+结果"无挣扎？如果是就重写
- 用户说"完全没写出情绪和紧张感"几乎 100% 是这个问题

## C. 信息释放节奏：情感场景不当侦探

**核心规则：** 不要在一个场景里同时做"情感建立"和"推理展示"两件事。场景有主次。

### C.1 Ch7 教训

我的 draft 在医院探病 + 艾伦对话场景中同时做了：

- 米娅的病情展示（情感）
- 阿尔温通过颜料推理代笔（推理秀）
- 马蒂亚斯凌晨来访的详细叙述（悬疑信息）
- "第二句话"悬念（线索钩子）

→ 信息释放太早，推理线索应该留到后面章节。

**用户保留的：** 艾伦自己快说漏嘴（"其实那家伙死的那天。。。"）→ 身体发作打断 → 自然的信息截断。**不是阿尔温推出来的，是艾伦差点说出来的** → 悬念更有效。

### C.2 How to apply

- 写每个场景前先定主任务：这个场景的核心功能是什么？
- 情感场景里不要做推理展示，推理场景里不要硬塞情感
- 线索释放宁可晚不可早——读者等得越久，揭示时冲击越大
- **角色"差点说出来但被打断"比"侦探推理出来"更好的悬念手法**
- 打断的方式要自然（身体发作、有人来了、角色自己意识到说多了）

## D. 世界观信息嵌入：从主角视角渗出

**关键原则：不能让读者一头雾水。** 世界观信息的嵌入不是越克制越好。

### D.1 正确方法（从视角角色周围体现）

- **主角的内心想法**："如果带着单镜，可能已经开始警告我异常灵能反应了"
- **对话中自然带出**："沃伦那边的实验室有不少新药，不需要灵晶不需要术法"
- **叙述中简洁定义**：阿克曼症开头的一句话定义
- **主角的观察推断**："多半是家里在给灵晶矿工厂打工"

### D.2 规则

- 新概念第一次出场时，用 1-2 句话从主角视角给出基本定义，不需要绕弯子
- 之后的细节通过对话、行为慢慢补充
- 判断标准：如果一个普通读者读到这里会问"这是什么"，那就需要解释
- 解释的载体优先级：**主角内心观察 > 角色间对话 > 叙述直接说明**
- **禁止百科词条式展开**，但一句功能性定义是必要的
- 与 [[feedback_ai_crutches]] §12 协同——读文献必须展开为阅读动作而不是 bullet list
