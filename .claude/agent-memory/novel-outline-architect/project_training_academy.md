**用户 2026-04-16 锁定 · V13 Ch24 启动训练副线：**

## 世界观前提 · 灵使训练工业化

这是一个**灵使训练已经成熟工业化**的世界·不是"主角孤独觉醒"套路。

- 各国有正规训练营（公开注册+学费+教官体系）
- 真灵研究是公开科研领域（分型/契合路径/晚期觉醒/阶位突破）
- 文献在训练营档案室/皇家学院图书馆/公开期刊

## 卡萨林灵使训练营

- **位置**：卡萨林本地
- **建立年限**：48 年（老牌）
- **架构**：登记处 / 理论讲堂 / 冥想室 / 体能训练场 / 实战模拟室 / 档案室
- **注册制度**：学员档案分"在籍柜"和"沉睡档案柜"（放弃/停训的学员档案归沉睡柜）

## 阿尔温的训练营历史【情感伤疤核心】

- **时间跨度**：16 岁到 22 岁·**6 年**（大学期间兼读 + 毕业后一两年）
- **成绩**：
  - 理论：**满分**（皇家学院灵学与社会顶班·机构单独为他开讲堂）
  - 体能 / 实战 / 短刀：**全优**
  - **冥想 / 真灵契合：空白**（6 年·契合仪/冥想/灵力通路·全做了·真灵一次都没降临）
- **放弃点**：**四年半前**皇家学院毕业前夕·档案归入"沉睡档案柜"
- **Ch24 重返**：档案尘封 4.5 年·此次蚀析觉醒后首次回来

## 新角色档案

### ⚔️ 帕维尔·科瓦列夫（Pavel Kovalev）

- **年龄**：45 岁
- **阶位**：三阶
- **真灵**：**守序·盾型**（誓约类·和安东尼奥同源但阶位低很多·卡萨林本地誓约系的实战代表）
- **身份**：卡萨林灵使训练营资深实战教官·20 年教龄
- **与阿尔温关系**：**当年最看好阿尔温的那个教官**——见过他每份测试报告·始终相信"真灵认人也看脾气·阿尔温的脾气得放下三分"
- **放弃那天的对话**（4.5 年前）：帕维尔送阿尔温出门说："真灵认人也看脾气·你的脾气得放下三分。下次再来的时候我还在这里。"阿尔温当时没回头。
- **性格**：沉稳·寡言·话少·但每句都有分量
- **代表动作**：看人的时候眼神不飘·说重话前会先沉默三秒
- **Ch24 核心对白**："六年你站在最后一排。" / "现在你站到前排了。"
- **全卷伏笔价值**：
  - Ch24-30 训练副线的实战导师
  - Ch32 训练系统正式建立时的系统方设计者
  - Ch40+ 核心训练章的对手戏主角
  - Ch55+ 可能因阿尔温卷入更深的事（誓约系三阶可能被卷入六阶局）

### 📖 巴克利·洛克伍德（Barkley Lockwood）

- **年龄**：62 岁
- **阶位**：未觉醒（普通人）
- **身份**：卡萨林灵使训练营登记处·管档案 20 年·退休前是前一级执法员
- **外表**：皮鞋永远擦得锃亮·桌上永远一壶黑茶·戴单边眼镜
- **性格特质**：冷外热内·话少·记忆力惊人·**每个来过的学员他都记得为什么离开**
- **与阿尔温关系**：**阿尔温放弃那天他就在登记处**·递回档案的是他
- **Ch24 情感锚点**：
  - 阿尔温 4.5 年前放弃时·在档案夹里留了一张小纸条："我会回来。但不知道要等多久。"

- **全卷伏笔价值**：Ch24 情感锚·后续章节作为"档案线"的情报源（他记得每一个来过训练营的人）
