**Ch25 锁定的核心追查动机（V13 细纲定稿·不可幻觉）：**

1. **雷纳德来卡萨林的原因**：他三年追圣座的**唯一同伴/朋友**·**两周前在卡萨林被圣座清理**。朋友死前用血在一小块布上写下"**卡萨林**"三字（就这三个字·没有名字/地点/时间）。
2. **血字布片**是雷纳德 V1 全程的情感锚+追踪原点·每晚在单间看一次·放在木箱底铁盒。朋友身份/姓名/如何死·在 V1 末或 V2 揭示。
3. **"卡萨林"三字省略了第四字 "K"**——朋友死前追的最后一个东西就是 K。
4. **K = 凯沙·瓦尔基里斯 (K.V.)**·48 岁·贵族院副官·瓦伦丁远亲·经手半灵人融合实验台账的中层转手人。
5. **雷纳德朋友死前追的也是 K.V.**——朋友死因=追 K.V. 经手的半灵人融合实验台账被圣座清理。
6. **阿尔温只是顺带**·不是雷纳德的主要目标。雷纳德在画廊外街闻到圣座味道·顺尾端定位到阿尔温（身上也有那味道·可能沾的也可能是他们的人）。Ch25 判定非目标后存档为"感知异常"。
7. **Ch32 雷纳德伪装接触阿尔温的动机**：Part 2 存档"值得留意" + Part 3 老狐狸说"两周前外地人（朋友）也问过 K" → 雷纳德意识到**阿尔温可能独立摸到 K 线** → Ch32 决定主动贴近验证。
