description: V13 细纲每章写作粒度——只写基础框架故事·不写旁白和心理独白
type: feedback
---

**规则：V13 细纲每章写作粒度 = 约 50-80 行 / 章·严格按 Ch18 模式。**

**不允许：**
- 大量括号内心独白（类似正文的心理描写）
- 完整对白台词堆砌
- 旁白性场景描写（用于烘托气氛的段落）
- Part 下的 # 分隔符块（# ──────────）
- 写到接近正文那种详细度

**允许：**
- 时间/地点/视角说明（1-2 行）
- 场景设定（1-3 句）
- 事件陈述（用动作动词推进）
- Bullet 列档案/线索/动作链
- **3 句左右**推动剧情的**关键对白**（多了就要砍）
- 标注参考作品章节结构对齐
- 章末伏笔列表（可详细）

**Why:** 我第一版写 Ch25 写了 468 行·大量括号内心独白+对白细节+场景描写·严重超出骨架粒度·用户明确指出"旁白和心理描写太多了·我们应该只写基础框架故事"。压缩版 72 行是正确粒度。

**How to apply:** 写任何 V13 细纲新章节时：
1. 先看该卷已有章节（如 Ch18/Ch25 骨架版）确认粒度
2. 每章目标行数：50-80 行·超过 100 行就要考虑压缩
3. 心理描写只保留决定性判断句（如"判定：非目标·存档"）·不写咀嚼式情绪独白
4. 对白保留≤3 句·其他都概括
5. 场景描写每处≤2 行·不铺环境氛围
6. 伏笔列表可详细·因为后续章节要引用
