**V13 四项根本规则锁定（V12 纠错后的核心）：**
1. 真灵 4 阶以下仅以纹身形态附着，无物质化；4 阶及以上才物理显现
2. 彼界=纯灵魂入境，物质不随·伤痛不跨界（唯一例外：精神疲劳、10 年瘫痪代价）
3. 蚀析现实能力=双重影像预见（主动·耗灵能）+ 重大死亡预警（被动）；**没有残留读取/考古能力**
4. 蚀析彼界能力=白蜘蛛全形态+八足丝线+三秒随机回放

**Why:** V12 因这四项规则错乱导致连续剧情崩坏。V13 全文严禁蚀析"读取物品历史/画框移动/印章签名"等考古式能力——改用单片镜（灵痕年轮）+ 分析仪（源流频谱）+ 灵痕频谱比对纯物证法。

**How to apply:** 任何涉及调查推理的章节，阿尔温能用的工具只有：单片镜/分析仪/录像手环/石墨粉/苯醌笔/灵痕频谱比对。**禁止蚀析读画框/印章/衣物历史**。
