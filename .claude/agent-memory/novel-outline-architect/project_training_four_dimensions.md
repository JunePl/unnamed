**用户 2026-04-17 锁定·V13 训练副线四维系统 + 支线终点：**

## 训练四维度（Q2.C 方案：维度在背景立住·每章聚焦一维的一个细节点）

1. **灵能训练**：觉醒等级提升+灵能容量扩展
   - 标准内容：冥想·契合仪·晨练灵力通路
   - 阿尔温处境：4.5 年空白后重返·蚀析已觉醒·练的是"稳定与扩展"（幅度两→三毫米稳定期）
2. **体力训练**：肉身素质（灵使不是纯灵能·肉身是承载）
   - 标准内容：晨跑·力量·柔韧·耐力·反应
   - 阿尔温处境：4.5 年前体能全优·28 岁底子在·需要恢复+配合灵能输出节律
3. **灵能运用**：能力施展精度
   - 标准内容：真灵在实战中的输出控制+精准度+节律
   - 阿尔温处境：蚀析用于"主动预见"（a 方向）·精度从两毫米推进到三毫米并稳定
4. **实战**：综合对抗
   - 标准内容：对练·模拟战·晋级赛
   - 阿尔温处境：列娅是主要实战搭档·帕维尔监督·Ch28 列车站战斗是首次实战验证

## 五件套超凡科技（Q3 全都有）

1. **真灵测量仪**：数值化显示觉醒等级/灵能强度·读数
2. **模拟战训练舱**：全息/立体环境模拟不同地形+对手
3. **体力增幅装备**：训练用的重力压制/节律引导器
4. **灵能回路导管**：将真灵运用可视化·帮助学员控制输出
5. **战斗记录仪**：对练画面+真灵波形保存·教练分析

每维对应的主用科技：
- 灵能训练 → 真灵测量仪+灵能回路导管
- 体力训练 → 体力增幅装备
- 灵能运用 → 灵能回路导管+战斗记录仪
- 实战 → 模拟战训练舱+战斗记录仪

## 支线终点：区域灵使大战（Q4.b·V1 直接参赛）

**规格：多城市/多训练营的区域锦标赛（类似竞技联赛）。**

时间线路径（锁定）：
- **Ch24**：训练副线启动·四维同步起步
- **Ch25-30**：早期阶段·蚀析觉醒+a 方向预见起点
- **Ch31/33/35**：呼吸章深化·每章聚焦一维的一个细节点
- **Ch35 前后**：【实战】维度·模拟战训练舱首次启动
- **Ch41**：核心训练章（四维综合测试/阶段评估）
- **Ch50+**：**区域预选赛首战**（列娅 Ch33 "想不想打比赛"即指此·V1 末时间线）
- **Ch55-60**：区域灵使大战主赛事（V1 末核心舞台）
- **V2**：区域大战后续影响+更高级别赛事

## 区域灵使大战的定位

不是单纯竞技——**比赛本身是阿尔温主线的一个聚光灯舞台**：
- 阿尔温以"无真灵但有蚀析"的身份参赛·身份异常
- 区域大战吸引三执政+家族+执法局多方势力关注
- 可能是德米特里/安东尼奥/瓦伦丁家族观察阿尔温的场所
- 阿涅丝/维辛妮/哈罗德/海伦娜 B 线配角的观赛位置各自承载戏份
- V1 末的通缉弧可能与区域大战的结果互相嵌套
