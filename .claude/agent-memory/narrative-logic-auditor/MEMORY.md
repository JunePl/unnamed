# Narrative Logic Auditor - Project Memory

## Project Overview
- Novel title: untitled (灵契纪元 spirit contract world)
- Genre: Mystery/Fantasy hybrid with political intrigue
- Structure: 5 volumes, ~140万字 total
- Setting: Post-imperial collapse, 6 nations, spirit contract system
- Main character: Alwin Edmond, spirit trace analyst (detective archetype)

## Key World Rules
- Spirit contract system: 10 tiers (序), from 初感 to 更迭
- Ranking system: 天秤 with 冠位/柱位/刃位/痕位
- Key mechanism: 引燃媒介 (ignition medium) - forced awakening, 30% success, failure = monster
- 窃相术: identity replacement, 72h limit
- 锚点 system: emotional/belief/identity/contract anchors resist spirit assimilation
- Higher tier = more assimilation, 7th tier+ needs organizational anchors
