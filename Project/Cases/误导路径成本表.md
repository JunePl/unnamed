# 编纂说明

> 本表记录每条误导路径的铺垫成本、读者误判概率、推翻时机和"废品利用"设计。

# 案件一：画商之死 -- 误导路径

## 误导A：学徒艾伦复仇

| 项目 | 内容 |
|------|------|
| **假设** | 艾伦因被马蒂亚斯克扣工资而谋杀老板 |
| **误导铺垫量** | Ch3-Ch12，约5个假线索，累计约3000字 |
| **假线索清单** | 1. 艾伦与马蒂亚斯的薪资争吵记录（Ch4）; 2. 艾伦案发当天行踪可疑（Ch5）; 3. 画廊员工暗示"艾伦最近怪怪的"（Ch6）; 4. 艾伦的住处发现灵力装置零件（Ch8）; 5. 艾伦在案发前一天深夜仍在画廊（Ch10） |
| **读者误判概率** | 70% |
| **推翻时机** | Ch12 |
| **推翻逻辑** | 艾伦的灵契位阶太低（1阶初感），根本不具备杀死马蒂亚斯的超凡能力。且他深夜在画廊是在帮马蒂亚斯整理"秘密画作"——他是知情的合作者，不是凶手 |
| **废品利用** | 推翻后，艾伦从"嫌疑人"变为"关键证人"→但他也是"弃子"→被引爆灵变为Ch20高潮提供核心素材 |
| **误导路径成本** | 低：5个假线索均在推翻后转化为真相拼图的一部分，无浪费 |

## 误导B：情杀·海伦娜外遇

| 项目 | 内容 |
|------|------|
| **假设** | 马蒂亚斯发现妻子海伦娜的外遇而被情杀 |
| **误导铺垫量** | Ch5-Ch14，约3个假线索，累计约2000字 |
| **假线索清单** | 1. 海伦娜与画廊常客格雷戈里的暧昧互动（Ch6）; 2. 马蒂亚斯死前一周与海伦娜激烈争吵（Ch7）; 3. 海伦娜案发当晚外出两小时行踪不明（Ch10） |
| **读者误判概率** | 50% |
| **推翻时机** | Ch14 |
| **推翻逻辑** | 海伦娜与格雷戈里的"暧昧"实为秘密商业合作——格雷戈里是塔玛拉（情报贩子）的下线，海伦娜在通过他传递马蒂亚斯掌握的信息 |
| **废品利用** | 推翻后揭示了情报网络的存在→连接到第二层（沉默兄弟会）和第三层（商人联盟） |
| **误导路径成本** | 极低：3个假线索全部转化为情报网络线索 |

## 误导C：商业暗杀·竞争对手

| 项目 | 内容 |
|------|------|
| **假设** | 画商市场竞争对手雇凶谋杀 |
| **误导铺垫量** | Ch7-Ch18，约4个假线索，累计约2500字 |
| **假线索清单** | 1. 竞争画商维托利曾公开威胁马蒂亚斯（Ch7）; 2. 四国展前的利益争夺（Ch9）; 3. 维托利雇佣的灵契者保镖可疑活动（Ch11）; 4. 商人联盟内部的派系斗争信息（Ch15） |
| **读者误判概率** | 60% |
| **推翻时机** | Ch18 |
| **推翻逻辑** | 维托利有完整不在场证明，且他虽然威胁过马蒂亚斯，但实际上两人在秘密合作（同为商人联盟中层）。维托利的"威胁"是演给联盟高层看的戏 |
| **废品利用** | 推翻后揭示商人联盟的分层结构→中层被当作工具的真相→直接连接第三层（德米特里体系） |
| **误导路径成本** | 低：4个假线索中3个转化为商人联盟结构揭示 |

## 误导D：替身老彼得（隐性误导·读者不自觉）

| 项目 | 内容 |
|------|------|
| **假设** | 老彼得是可靠的证人（读者默认假设） |
| **误导铺垫量** | Ch3-Ch25，贯穿全案，约10个隐性线索 |
| **隐性线索清单** | P01-P10伏笔链（见案件与伏笔集） |
| **读者误判概率** | 90%（绝大多数读者不会怀疑一个"忠诚老管家"） |
| **推翻时机** | Ch25 |
| **推翻逻辑** | 五段式揭示——阿尔温通过倒茶用手、颜料斑、摸下巴习惯等细节证明"这个人不是老彼得" |
| **废品利用** | 整个伏笔链的回收构成第一幕最大反转+引出沉默兄弟会 |
| **误导路径成本** | 最高但回报最高：22章的伏笔投资换取全卷最震撼的反转 |

# 案件二：伯爵之死 -- 误导路径

## 误导E：德米特里直接暗杀

| 项目 | 内容 |
|------|------|
| **假设** | 德米特里用织命·人脉型能力直接杀死安东尼奥 |
| **误导铺垫量** | Ch28-Ch38，约4个假线索 |
| **假线索清单** | 1. 安东尼奥体内织命系灵痕残留（Ch30）; 2. 德米特里近期与安东尼奥政治冲突升级（Ch32）; 3. 内线情报指向德米特里（Ch35）; 4. 德米特里案发当天的可疑行程（Ch37） |
| **读者误判概率** | 80% |
| **推翻时机** | Ch42 |
| **推翻逻辑** | 灵痕"太完美"——真正的柱位#67高手不会留下如此清晰的签名。这是栽赃 |
| **废品利用** | 推翻后揭示了一个更危险的存在：能模仿灵痕的顶级铸魂师 |

## 误导F：塔尔拉政变

| 项目 | 内容 |
|------|------|
| **假设** | 第三执政塔尔拉联合外部势力夺权 |
| **误导铺垫量** | Ch33-Ch40，约3个假线索 |
| **读者误判概率** | 40% |
| **推翻时机** | Ch43 |
| **推翻逻辑** | 塔尔拉是保守派，她最不希望三执政格局被打破——安东尼奥死对她毫无益处 |
| **废品利用** | 揭示塔尔拉的"保守"动机→她在第二卷成为阿尔温的临时盟友 |

## 误导G：安东尼奥自杀伪装

| 项目 | 内容 |
|------|------|
| **假设** | 安东尼奥自杀并伪装成他杀以栽赃德米特里 |
| **误导铺垫量** | Ch38-Ch45，约2个隐性线索 |
| **读者误判概率** | 25%（少数读者会猜到） |
| **推翻时机** | Ch46-47（部分正确） |
| **推翻逻辑** | 不完全推翻——安东尼奥确实"选择"了死，但不是自杀，是"让自己被杀"并在体内预埋证据。比自杀更复杂、更悲壮 |
| **废品利用** | 这条路径离真相最近，猜到的读者会获得"原来如此"的满足感 |
