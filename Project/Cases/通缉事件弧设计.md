# 第一部分：Chain6·替罪羊协议

## 安东尼奥预留的证据链（5项）

```
【Chain6设计哲学】
安东尼奥的Thanatos Gambit原有五条连锁引爆线（案件与伏笔集V7.0 L1074-1079）。
Chain6是第六条——安东尼奥**故意**制造证据链，让阿尔温在安死后被栽赃为杀人嫌犯。

关键前提：
  安东尼奥是誓约·契约型——不能说谎。
  因此五项证据全部基于**真实事件**——没有一项是伪造的。
  每一项单独看都是事实。五项组合在一起=完美的栽赃。
  执法局和塔玛拉基于证据链**自主判断**发出通缉令——非德米特里操纵。

【五项证据】

1. 出入记录
   安东尼奥故意不清除庄园安保日志。
   事实：阿尔温确实在安东尼奥死亡时段前到访庄园（秘密合作会面）。
   栽赃效果：嫌疑人在死亡时段前出现在现场=最基本的犯罪嫌疑成立。

2. 终焉灵痕伪层
   安东尼奥在书房结界上叠加模拟终焉源流的灵痕残留。
   事实：书房确实被攻破（德的织命·支配型是真正攻破者）。
   栽赃效果：标准执法局设备首先读取到终焉特征→阿尔温是已知的终焉源流灵使。
   设计精度：深层扫描仍可发现织命特征——但执法局不会在有明确嫌疑人时做深层扫描。
   → 安赌的是官僚惯性：有了嫌疑人=结案压力>真相追求。

3. 预植证人
   通过誓约·契约型能力在一名次要联络人意识中植入记忆碎片。
   该证人真诚地「回忆」看到与阿尔温描述相符的人从庄园离开。
   关键：证人不是在说谎——他真的"记得"看到了。
   → 誓约型的极端应用：不是制造谎言，是制造真诚的错误记忆。

4. 资金痕迹
   安东尼奥账户中一笔标注为「咨询费」的转账。
   事实：安东尼奥确实向阿尔温支付过灵痕分析咨询费（合法合作报酬）。
   栽赃效果：在通缉语境下，「执政官死前向嫌疑人转账」=暗杀佣金。
   → 安的精妙：用真实交易制造虚假叙事。

5. 可见笔记
   书房中未隐藏部分留有关于阿尔温的记录。
   全部是真话——但断章取义极具指控性。
   例："埃德蒙先生对终焉源流的痕迹读取能力令人惊异——
        他能在我的书房中发现我自己都忘记的灵痕。"
   在通缉语境下→"嫌疑人对受害者书房极度熟悉"=预谋踩点。
```

## Chain6与五连锁的交叉关系

```
Chain6不是独立存在的——它与前五条连锁形成交叉增幅。

C1(三密钥断裂) × Chain6：
  通缉转移注意力。所有人关注「谁杀了执政官」而非「防御缺口」。
  → 三密钥认证链断裂的事实被通缉的政治风暴掩盖。
  → 德有更多时间探索如何用两颗密钥强行控制傀儡灵。

C2(经济定时炸弹) × Chain6：
  通缉冻结阿尔温的合法账户。
  → 阿尔温被迫启用安东尼奥隐藏经济渠道（安的遗产网络）。
  → 使用遗产网络=激活更多安的后手=德更难追踪资金流向。

C3(誓约遗证) × Chain6：
  逃犯身份=无法向执法局提交任何证据。
  → 阿尔温手中的誓约遗证碎片无法通过法律渠道使用。
  → 必须拼完完整图景才能公开——延长了调查时间=其余四条链安全运行。

C4(情感武器) × Chain6：
  被导师栽赃=第六层情感复杂度。
  → C4的情感武器本是"失去导师的悲痛"。
  → Chain6加入后变成"失去导师+被导师栽赃"的双重认知冲击。
  → V2+回报更大：当真相揭示时，"他用栽赃保护了你"=情感核弹。

C5(Xanatos困境) × Chain6：
  Chain6为德创造第六条路径——无论维持还是撤销通缉都有代价。
  → 维持通缉：阿尔温从棋盘上消失→德难以公开追杀→遗产网络安全运行。
  → 撤销通缉：承认证据有误→执法局重新调查→织命灵痕可能被深层扫描发现。
  → 德被困在"追查一个逃犯"和"无视一个嫌疑人"之间——两个选项都不好。
```

## 为什么帮助阿尔温（安东尼奥L4-L5设计层）

```
安东尼奥设计Chain6的核心逻辑不是惩罚阿尔温——是保护他。

1. 消失效应
   逃犯身份让阿尔温「从棋盘上消失」。
   德的织命·支配型依赖人脉网络=合法身份系统内的感知。
   通缉犯脱离合法身份→德的网络感知大幅降低。
   → 安赌的是：一个「消失」的阿尔温比一个「可见」的阿尔温更安全。

2. 信息升级
   逃犯身份迫使阿尔温进入地下网络。
   莫哈夫的情报系统/维克托的安东尼奥旧网络/哈罗德的灵使地下圈。
   → 合法分析师无法触及的信息层=地下世界的真实情报。
   → 阿尔温的蚀析能力在地下世界更有用（没有官方设备竞争）。

3. 政治烟幕弹
   执法局资源被调往追查「杀害执政官的嫌疑人」。
   → 五连锁不受干扰地运行。
   → 德即使发现了五连锁的痕迹，执法局也没有多余资源去配合他调查。

4. V2+社会翻转回报
   「被自己导师栽赃的学生」→真相揭示后=社会同情大翻转。
   → 安东尼奥的情感武器(C4)在Chain6解除时达到最大效果。
   → "他用谎言保护了你"——但安不能说谎——"他用真话保护了你"更具冲击力。
```

## 引入期（Ch52·新增章节「警报」）

```
紧张值趋势：3→9→5
净章节占比：1章（~12%）
与日常融合：引入从日常被打断开始

【场景序列】

S1·莫哈夫的警报（压3/释2）
  深夜。阿尔温在夜莲酒吧，莫哈夫在柜台后擦杯子。
  莫哈夫忽然放下杯子——罕见的严肃。
  "楼下。"
  夜莲酒吧地下情报网截获执法局内部通讯。
  死信箱传递：一个地址+一个字——"走"。
  → 莫哈夫不解释更多："先走。等到了再想。"

S2·被堵（压9/释0）
  阿尔温试图从后门离开——执法局已部署封锁线。
  不是普通巡逻——是全城搜捕级别的封锁。
  殊死逃跑：穿巷/跳墙/蚀析两秒预判（白丝在前方织网→感知到3个执法使位置→选择路径）。
  被追至死胡同——执法局三人合围。
  阿尔温背靠墙壁，手按灵质枪但没有拔出来（对执法使开枪=死罪升级）。
  蚀析在绝望中给出最后一个信息："东边。有缺口。"

S3·维辛妮的放水（阿尔温不知道）
  维辛妮·蒙特在指挥中心：
  "封锁南街和码头路。所有人转移至西区。"
  → 东巷口未封。
  阿尔温视角：靠蚀析感知到东边封锁最薄弱→全力冲刺→穿过东巷口→脱出包围圈。
  → 阿尔温认为：蚀析的判断力+自己的速度=逃脱关键。
  → 读者二读视角：维辛妮的封锁精确遗漏了那个方向。
  → 设计铁律：叙事中绝不暗示维辛妮在帮忙。阿尔温的「脱险」必须看起来是他自己的能力。

S4·避难屋（压5/释4）
  抵达莫哈夫提供的地址——提耶林港区边缘一间废弃渔具仓库改造的密室。
  维克托已在等候。
  维克托没有寒暄：
  "坐下。听我说。"
  → 解释Chain6——安东尼奥预留的「保险机制」。
  维克托的台词：
  "伯爵大人在两年前就准备了这个。假身份、银行账户、旅行许可。
   他不是在预言你会被通缉——他是在确保如果你被通缉，你能活下来。"
  "第六条链。前五条用来对付德米特里。第六条用来保护你。"
  提供假身份文件+易容套件。
  → 阿尔温成为「菲利克斯·布兰特」。

S5·通缉公告（压7/释2）
  避难屋中的老式灵波收音器传出通缉公告：
  "卡萨林执法局紧急通告——灵痕分析师阿尔温·埃德蒙，涉嫌刺杀三执政安东尼奥·莫兰蒂，
   现已被列为一级通缉嫌犯。所有公民发现此人立即通报——"
  维克托关掉收音器。
  沉默。
  阿尔温盯着桌面——手指敲桌三下（习惯动作·与蚀析推理时一致）。
  "他用真话把我变成了杀人犯。"
  → 决心调查安东尼奥之死与D的关系——不是为了洗清自己，而是要搞清楚"为什么他需要我消失"。
```

## 展开期（~Ch53-62）

```
紧张值趋势：5→6（底层不安恒在）
净章节占比：~2-3章核心驱动（其余以状态层融入其他线路）
日常融合度：高——通缉状态渗透每一个日常行为

【新身份适应】
  学会压制分析师习惯——停止对所有事物的灵痕扫视。
  单片镜必须隐藏（最具辨识度的特征）——用普通眼镜替代。
  蚀析管理：白蛛绝不可在公共场合被看到→通过衣物缝隙操作丝线。
  维克托的指导（一句话定调）：
    "你走路像个检查一切的人。走得像个无聊的人。"

【日常场景中的通缉状态】
  买东西时：递钱的手必须稳——不能让店主注意到指尖的分析师老茧。
  过关卡时：心跳加速但表情必须无聊——菲利克斯·布兰特是个无聊的贸易清单员。
  染发维护：炼金染料2-3周褪色→定期联系莫哈夫地下渠道补充。
  → 通缉不是一个"事件"——它是一种"状态"，渗透到每一个呼吸里。

【展开期事件】
  以伪装身份参与四国展后期活动（Line F交叉）。
  通过地下渠道获取D相关情报：
    莫哈夫（情报掮客）→提供D网络的外围信息碎片。
    维克托（安东尼奥旧网络）→提供三执政内部政治动态。
    哈罗德（灵使圈子）→提供灵力异常的地下报告。

【阿涅丝隔离弧开始】（平行D线）
  执法局开始监控阿涅丝的住处——作为通缉犯亲属。
  阿涅丝的反应详见第五部分。
```

## 转折点（~Ch63）

```
紧张值：跳至7-8
净章节占比：~1章

【雷纳德看穿伪装】
  雷纳德引介时刻——某人（莫哈夫或地下渠道联系人）引介「菲利克斯」给雷纳德。
  雷纳德看了他三秒。
  "你染了头发。单镜也摘了。走路比平时慢了半拍——在刻意控制步频。"
  停顿。
  "无所谓。我不是执法局。"
  → 雷纳德几分钟内看穿伪装——但完全不在乎阿尔温是否被通缉。
  → 独立猎手/虚空圣座实验品的身份决定了他对「合法性」毫无兴趣。

【转折后强制消化】
  执法局巡逻险些暴露→雷纳德本能挡住视线。
  不是帮忙——是反射动作：有人在他面前被追捕→他的身体记忆做出反应（实验品经历）。
  "别误会。我只是不喜欢制服。"
```

## 深入期（~Ch64-72）

```
紧张值趋势：6→7
净章节占比：~2-3章核心驱动（其余状态层）

【地下身份调查D】
  以「菲利克斯·布兰特」在地下网络中活动。
  逃犯身份=地下世界的通行证——"被执法局追杀的人"在灰色地带反而更受信任。
  阿尔温发现：D（此时他还在用代号）的网络比他想象的更广——不只是犯罪网络，
  而是渗透到了商人联盟、部分执法局、甚至学术机构。

【D的名字揭示】
  德米特里这个名字不是在大场面中揭示的。
  是在简陋避难屋中——阿尔温拼完所有碎片后——
  维克托确认："是的。德米特里·卡拉维奇。三执政中最强的那个。"
  阿尔温的反应不是震惊——是疲惫的确认。
  "我早就猜到了。我只是希望我猜错了。"

【维辛妮关卡检查——三秒对视】
  阿尔温以「菲利克斯·布兰特」身份过关卡。
  维辛妮在关卡执勤。
  她接过证件。看了看文件。看了看脸。
  三秒。
  三英尺距离。
  "通过。"
  → V1最重要的暗线时刻。
  → 阿尔温视角：心跳加速但表情无波——菲利克斯的表演通过了审查。
  → 读者二读视角：维辛妮在三英尺距离内认出了他——然后放行——完美控制面部表情。
  → 设计铁律：叙事中绝不暗示维辛妮认出了他。阿尔温的「通过」看起来是伪装成功。

【碎片交易】
  苏暮雪以阿尔温本名称呼他。
  "卡萨林的通缉令在瓦尔德不具备效力。埃德蒙先生。"
  → 苏暮雪完全不在乎卡萨林内政——外交官的特权+瓦尔德联邦的立场。
  → 对阿尔温而言：世界上还有人用他的真名叫他——微小但重要的锚定。
```

## 高潮期（~Ch73-82）

```
紧张值趋势：8→10
净章节占比：~2章核心驱动（融入终幕引爆）

【通缉状态与终幕引爆碰撞】
  通缉令不只是栽赃——阿尔温意识到德在利用它。
  关键发现：执法局的搜捕部署模式异常——
  不像在找一个人——像在布置军事封锁线。
  蚀析读取封锁令上的灵痕残留→"这些部署不是塔玛拉签发的。有另一只手在移动棋子。"
  → 通缉令=德的物流工具：以搜捕通缉犯为名，将执法局兵力部署到围猎所需的位置。
  → PB21回收：阿尔温意识到通缉令的双重用途。

【东港区·伪装身份进入】
  以「菲利克斯·布兰特」进入东港区——围猎的核心区域。
  逃犯身份此刻成为优势：他不在德的合法感知网络中。
  但也成为劣势：如果被执法局发现，他将同时面对通缉追捕+德的围猎。

【战斗中身份暴露】
  围猎战斗中——阿尔温被迫全力使用蚀析。
  白蛛在战场上展开完整的丝网——所有在场者都看到了终焉源流的特征。
  菲利克斯的伪装不再可持续。
  → 身份暴露不是"被抓"——是"被所有人同时看到"。
  → 伪装期彻底结束。
```

# 第三部分：新身份机制

## 身份：「菲利克斯·布兰特」

```
【基础设定】
  全名：菲利克斯·布兰特
  封面故事：外省贸易清单员，安东尼奥商会三级外包雇员
  纸面背景：安东尼奥2年前创建的应急提取身份
    - 银行账户（小额定期存入·模拟工薪收入）
    - 住宿登记（提耶林港区边缘廉价旅馆·按月付费记录）
    - 雇佣记录（商会三级外包名册中存在菲利克斯·布兰特的条目）
    - 旅行许可（提耶林-王都区间·商务类·有效期6个月）
  → 安东尼奥的誓约·契约型精度：
    他不能伪造——但他能「创造」。
    菲利克斯·布兰特是一个真实存在的雇员——只是从来没有真人担任过这个职位。
    所有文件都是「真的」——签名是安东尼奥的，账户是商会的，记录是实际系统里的。

【物理伪装】
  染发：炼金染料（深棕色→暗金色），2-3周需续一次。
  服装风格：从灵痕分析师的实用装束→贸易清单员的廉价正装。
  面部微调：维克托提供的炼金软膏——微调下颌线和颧骨阴影。不是变形=长时间佩戴不适。
  最关键隐藏项：单片镜。
    阿尔温的单片镜是整个卡萨林灵痕分析圈最具辨识度的特征。
    灵质体融合后单片镜数据全乱——但外形仍在。
    必须完全隐藏→替换为普通眼镜（无灵力增强·阿尔温的分析能力大幅受限）。

【行为伪装】
  维克托的指导核心：
    "你走路像个检查一切的人。走得像个无聊的人。"
    "你进门先看角落。菲利克斯进门先找柜台。"
    "你喝水之前会看杯底。菲利克斯不会。"
  → 阿尔温必须压制蚀析赋予的痕迹感知本能——每个日常动作都是表演。

【蚀析管理】
  白蛛绝不可在公共场合被看到。
  操作方式：通过衣物缝隙操作丝线——袖口/裤脚/衣领内侧。
  限制：丝线范围被压缩到体表1米以内（正常状态可达5-10米）。
  → 态势感知从「全方位灵力雷达」降级为「近身微弱触须」。
```

## 限制/张力源（6项）

```
1. 染发褪色
   炼金染料2-3周褪色→根部原色开始显现。
   维护需要莫哈夫地下渠道的供应→每次接触=暴露风险。
   → 张力：物理伪装有保质期。时间是敌人。

2. 蚀析压制
   白蛛不能公开活动→态势感知大幅削弱。
   阿尔温从「读取一切痕迹的分析师」降级为「普通人的感知水平」。
   → 张力：最强的能力不能用=最脆弱的状态。

3. 单镜隐藏
   普通眼镜替代单片镜→分析能力受限。
   无法进行灵痕扫描→无法在日常中验证信息真假。
   → 张力：蚀析能力在但读取工具不在——像神枪手没了瞄准镜。

4. 知情人泄漏风险
   知道阿尔温真实身份的人：
     维克托（安东尼奥旧部·高度可信）
     莫哈夫（情报掮客·利益绑定=可信但非无条件）
     哈罗德（灵使圈子·人情关系=脆弱）
     雷纳德（独立猎手·不在乎但不可控）
     苏暮雪（瓦尔德外交官·国际政治=安全但有条件）
   → 张力：每多一个知情人=多一个泄漏点。但调查需要更多人的帮助。

5. 灵痕特征监控
   执法局有阿尔温的灵痕特征档案。
   监控区域（执法局/官方设施/灵力密集区）不能使用灵能。
   → 张力：进入核心调查区域时必须选择——用灵能获取情报/暴露灵痕特征。

6. 赤锁烙印
   维辛妮的赤锁烙印能力——她给过阿尔温一个（Ch4修理铺·公事标记）。
   赤锁=灵力追踪标记。理论上维辛妮随时可以定位阿尔温。
   → 她从不使用。但威胁始终存在。
   → 张力：阿尔温知道这个标记在→增加每次路过维辛妮时的紧张感。
   → 维辛妮暗线：她不用赤锁=她不需要用（因为她已经知道他在哪）。
```
