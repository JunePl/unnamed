## 一、双核命题体系

本项目采用 **双核命题** 结构，两条命题线如DNA双螺旋般交织贯穿全书：

### 命题α：真相与代价（贯穿线）

```
【核心命题】：每一个真相都有代价，而知道真相的人必须决定——这个代价由谁来承担？
【正论】：阿尔温——坚持"真相必须被揭示，无论代价"（V1-V3）
【反论】：德米特里——"控制真相才是保护秩序"；莱奥里克——"有些真相不该被所有人知道"
【命题演化】：
  V1 开篇立场："真相是正义的基础"（阿尔温的职业信条）
  V2-V3 动摇："知道真相的人反而最危险"（安东尼奥之死/帝国秘密）
  V4-V5 裂变："真相可以是武器，也可以是诅咒"（阿涅丝身份/穿越者身份）
  V6-V7 升华/颠覆：每个人都有权选择自己能承受的真相——但隐藏真相的权力不属于任何人
【日常映射】：
  · 阿尔温分析灵痕时的执念——"灵痕不说谎"
  · 阿涅丝的称呼变化——每次改口都是一个微型真相的揭示或掩盖
  · 德米特里温和笑容背后的信息操控——日常对话中的微型谎言
  · 市场上的假灵质检测——连商品真伪都是"真相"命题的日常投射
```

### 命题β：治世之道——锚点政治 × 超凡科技 × 制度博弈（三维命题）

```
【核心命题】：在超凡者可以灭城、科技可以改变权力结构的世界里，什么样的制度才能让人民幸福？
【正论】：没有单一正确答案——六国六种实验，各有代价
【反论】：每种制度都有致命缺陷，且所有制度最终服务于锚点需求而非人民需求
【命题演化】：
  V1 引子："这个世界的秩序看起来运转良好"（表面稳定）
  V2 揭幕："秩序的本质是强者的锚点需求"（真相浮现）
  V3-V4 实验场："推翻旧秩序之后呢？"（莱奥里克/阿尔温各自尝试）
  V5-V6 三维碰撞："科技、超凡力量和制度，三者如何共存？"
  V7 开放式结局：阿尔温的"第三条路"——不是答案，而是一种新的提问方式
【日常映射】：
  · 不同国家的面包价格差异——经济是制度的体温
  · 普通人对超凡者的态度——恐惧/崇拜/利用/漠不关心
  · 科技产品（符文家电/魔偶服务/灵能交通）在日常中的渗透差异
  · 边境线两侧的生活对比——制度差异的最直观展示
```

### 双核交汇点

命题α和β并非平行线——它们在关键节点交叉引爆：
- **V2交汇**：安东尼奥之死的真相（α）揭示了三执政的制度性腐败（β）
- **V5交汇**：阿涅丝身份的真相（α）与血月教利用锚点系统的政治阴谋（β）同时爆发
- **V6交汇**：锚点系统的真相（α）——"七阶以上者需要人民当锚点"——一旦公开将重写所有国家的政治结构（β）
- **V7终极交汇**："真相应该被揭示吗？"（α）与"什么样的制度能承受真相？"（β）合流为同一个问题

## 二、三维命题矩阵：锚点政治 × 超凡科技 × 制度博弈

### 维度一：锚点政治——"不是因为强大所以统治，是因为必须统治才能保持理智"

**核心机制**（源自世界通鉴·灵学典籍·残卷第三章）：

七阶（执律）以上超凡者面临灵的深度侵蚀（60%+灵魂同化率）。单纯的情感/信念锚点不足以抵抗。他们需要**国家锚点**——数以万计的人将其视为领袖/保护者/象征，集体认知在彼界形成屏障。

> "你不再只是'你'，你是'他们眼中的你'。" ——灵学典籍

**六阶可灭城，八阶可灭小国**——这意味着一个失去锚点的七阶超凡者不只是"疯了"，而是一个行走的大规模杀伤性武器。锚点政治不是哲学命题——它是生存问题。

**锚点质量阶梯**（从低到高）：

| 等级 | 锚点类型 | 稳定性 | 代价 | 代表 |
|------|----------|--------|------|------|
| 1 | 恐惧 | ★☆☆☆☆ | 需持续制造恐惧，稍有松懈即崩溃 | 历史上的暴君（全部短命） |
| 2 | 制度服从 | ★★☆☆☆ | 依赖权力结构，制度崩塌即崩溃 | 旧帝国中期皇帝 |
| 3 | 利益交换 | ★★★☆☆ | 持续提供利益，经济衰退即动摇 | 薇拉（商人依赖她的情报） |
| 4 | 军事忠诚 | ★★★☆☆ | 依赖军队的真实忠诚，败仗即动摇 | 阿德里安（军队即身体） |
| 5 | 民主尊重 | ★★★★☆ | 需真正回应民意，决策失误积累则衰减 | 格里高尔（联邦精神化身） |
| 6 | 文化认同 | ★★★★☆ | 文化解体则崩溃，但文化惯性极强 | 巴图尔（草原永远是草原） |
| 7 | 宗教信仰 | ★★★★★ | 最稳定但最危险——信仰狂热可走向极端 | 卡尔文（神的代言人） |
| 8 | 真心崇拜 | ★★★★★ | 理论最优，需要真正值得崇拜的品格 | 莱奥里克？奥里斯？ |

**补充机制**：灵主动保护宿主可大幅降低侵蚀（如「弑杀」与利姆尼，侵蚀度降至10%），但代价是灵切割与彼界的部分联系，丧失彼界权柄。

**核心悖论**：
> "我需要人民来维持理智，但我用什么方式获得他们的认可，就决定了我成为什么样的人。"
> 选择恐惧的人，最终被恐惧定义。
> 选择信仰的人，最终成为信仰的囚徒。
> 选择民主的人，最终发现民意可以被操纵。
> 选择利益的人，最终被利益绑架。

### 维度二：超凡科技——"科技可以被复制和传播，超凡力量不能"

**核心命题**（源自世界通鉴·科技文明典籍·序言）：

超凡者仅占人口0.3%。科技是99.7%普通人改善生活的唯一途径。帝国的强大"在于它找到了超凡与科技的平衡点"——但帝国崩溃后，这个平衡被打破了。

**科技对权力结构的冲击**：

| 科技领域 | 当前状态 | 对超凡者的影响 | 对锚点政治的影响 |
|----------|---------|---------------|----------------|
| **符文武器** | 普通士兵装备"准超凡"战力 | 低阶(1-3阶)超凡者战斗优势被削弱 | 军事锚点贬值——"为什么需要你保护？" |
| **魔偶制造** | 四级(学习型)全大陆<100台；三台五级(自我型) | 铸魂系的劳动可被机器替代 | 产业锚点动摇——"铸魂师还有价值吗？" |
| **灵能通讯** | 500km/100字符/0.5信用券每次 | 信息垄断被削弱 | 情报锚点面临挑战——薇拉的优势缩小 |
| **信息网络** | 4节点重建中，计划新历20年覆盖两国 | 所有基于信息差的权力面临透明化 | 全面冲击——"谁都能知道真相了" |
| **灵质能源** | 灵质结晶/液/合金/粉四形态民用化中 | 能源结构从超凡者依赖转向灵质依赖 | 资源政治取代人身依附——灵质产地成战略要地 |
| **五级魔偶** | 亚当/莉莉丝/该隐——拥有自我意识 | "人类"的定义被动摇 | 存在性冲击——"如果魔偶是'人'，锚点计算怎么办？" |

**灵质三角锁**（六国资源博弈的核心结构）：

```
      瓦尔德联邦（技术持有方：帝国科技70-80%）
           ↗ 技术输出          ↖ 灵质进口
    塞伦共和国（贸易中转方）       科尔曼汗国（灵质产地）
         ↕ 情报+金融            ↕ 原材料出口
    艾瑞克帝国（军用需求方）     米拉尔圣国（灵质="神赐"）
```

- 瓦尔德需要灵质发展科技 → 灵质来自科尔曼/米拉尔
- 科尔曼的灵质是草原的"血脉"——过度开采=文化/生态双重灭绝
- 米拉尔将灵质视为"神赐"——出口灵质=亵渎，但经济上又需要外汇
- 塞伦作为中间商获利——但信息网络将逐步绕过中间商
- **每一方的合理诉求都在伤害其他方**

### 维度三：制度博弈——"每种制度都是一场未完的实验"

六国 = 六种治世实验，各自服务于不同的锚点需求：

| 国家 | 制度类型 | 最强者/锚点 | 科技态度 | 人民幸福度 | 致命缺陷 |
|------|---------|-----------|---------|-----------|---------|
| **卡萨林** | 三执政寡头制 | 合力≈8阶 / 恐惧+服从 | 保守（科技威胁权力） | ★★☆☆☆ 底层压迫严重 | 无真正8阶个体→一人倒则全崩；锚点质量最低 |
| **瓦尔德** | 联邦议会制 | 格里高尔8阶 / 民主尊重 | 最前沿（帝国科技继承者） | ★★★★☆ 最高但贫富扩大 | **科技替代悖论**：科技越发达→人民越不需要超凡者→格里高尔锚点自我消解 |
| **塞伦** | 商会共和制 | 薇拉7阶(=8阶) / 利益交换 | 实用（利润驱动引进） | ★★★☆☆ 商人富裕底层尚可 | **信息透明死亡螺旋**：信息网络→情报垄断被打破→薇拉锚点崩塌 |
| **艾瑞克** | 军事帝制 | 阿德里安7阶(→8阶) / 军事忠诚 | 军用优先 | ★★☆☆☆ 非军人被忽视 | 和平时代→军队价值降低→军事锚点贬值→阿德里安危机 |
| **米拉尔** | 神权制 | 卡尔文8阶 / 宗教信仰 | 排斥（"对神的亵渎"） | ★★★☆☆ 信徒满足，异教者受压迫 | 科技传入→年轻人不再信神→信仰锚点缓慢侵蚀 |
| **科尔曼** | 部落联盟/汗制 | 巴图尔7阶(=8阶) / 文化认同 | 无感（游牧不依赖） | ★★★☆☆ 自由但资源匮乏 | **灵质诅咒**：灵质外销→草原"血脉"被抽干→文化/生态根基崩塌 |

**帝国（已崩溃）的警示**：

奥里斯以**整个帝国**为锚点，将四灵契约的代价分摊给全体臣民——"那些莫名其妙的瘟疫、歉收、生育率下降……不是天灾，是代价。"皇帝不是统治者，是**祭品**。帝国必须不断扩张（更多臣民=更小的人均代价），停止扩张后开始衰落。

这是锚点政治的终极案例：**最"完美"的制度，建立在对所有人最隐蔽的剥削之上。**

## 三、八个核心命题问题（Q1-Q8，跨卷递进）

### 第一层：锚点政治问题（V1-V3浮现）

**Q1**：统治者需要人民维持理智——这种"需要"是真正的"服务"还是"利用"？
- V1暗示：德米特里看似维护秩序，实则将人民视为棋子——但他的锚点确实需要他们
- V3爆发：莱奥里克推翻三执政后，他获得的是血洗之日的恐惧，还是解放的感恩？他的锚点质量取决于答案

**Q2**：人民有权知道自己是某人的锚点吗？有权拒绝吗？
- V2-V3展开：普通人不知道"国家锚点"的存在——他们的崇拜/恐惧/忠诚是"自发"的，但超凡者在利用这一点
- V6引爆候选：如果锚点真相被公开，所有国家的政治合法性同时动摇

**Q3**：奥里斯的帝国锚点——让帝国替他承担代价——是最伟大的契约，还是最大的剥削？
- V4-V5追溯：阿尔温发现帝国秘密（皇帝=祭品，代价分摊给臣民）
- V7终极提问：如果重建帝国锚点能拯救世界，但代价是让所有人不知不觉地牺牲一小部分——值得吗？

**Q4**：锚点崩溃 → 超凡者疯狂 → 大规模灾难——谁该为此负责？
- 霜棘伯爵：6阶，妻子死→锚点崩溃→屠杀1200人→被三位同阶联手击杀
- 预言者伊莎贝拉：7阶，发现信仰为假→锚点崩溃→永久疯狂，关押在圣国深层地牢
- 深渊之子科尔涅利乌斯：8阶，无法理解"爱"→锚点崩溃→沉入深海
- V3触发：莱奥里克的崛起导致三执政锚点体系崩塌——间接引发灾难
- V5升华：阿涅丝的故事——如果锚点是被操纵的（血月教），锚点本身就是牢笼

### 第二层：超凡科技问题（V3-V6浮现）

**Q5**：当科技让普通人拥有"准超凡"战力，超凡者的统治合法性建立在什么之上？
- V3-V4展开：瓦尔德的符文武器装备军队后，低阶超凡者开始"贬值"
- V5冲突：格里高尔两难——推动科技（削弱自身锚点价值）还是限制科技（背叛联邦精神）

**Q6**：灵质是超凡科技的血液——谁控制灵质，谁就控制文明走向。灵质该像空气一样自由，还是像武器一样管控？
- V4展开：灵质贸易战成为六国博弈的暗线
- V6引爆：灵质危机（真实枯竭或人为制造的枯竭假象）迫使所有势力摊牌

**Q7**：五级魔偶拥有自我意识——"亚当"问："你们怎么知道自己不是更高存在制造的'魔偶'？"——当非人类存在质疑存在本身，超凡者的"人类特权"还成立吗？
- V5侧线：魔偶权利成为政治博弈的棋子
- V7呼应：阿尔温作为穿越者——他是"人"还是某种更高存在的投射？穿越者身份与魔偶问题在哲学层面共振

**Q8**：信息网络重建将打破所有信息垄断——当所有人都能获取信息，"真相"还有力量吗？还是信息洪流本身就是最好的遮蔽？
- V4-V5展开：阿尔温利用信息网络建立新型势力基础
- V6反转：信息过载 → 人们反而更容易被操纵（假信息+情绪引导 > 真相）
- 与命题α交汇："真相必须被揭示"遇上"揭示后没人在乎"——哪个更可怕？

## 四、三重核心矛盾

### 矛盾一：科技替代悖论

```
科技进步 → 普通人能力↑ → 超凡者不可替代性↓
    → 锚点质量↓（"我为什么要崇拜/依赖你？"）
    → 超凡者理智风险↑
    → 要么限制科技（背叛文明进步）
    → 要么接受锚点衰减（自我毁灭）
```

**最尖锐的体现者：格里高尔**

"钢铁之父"格里高尔的锚点是"联邦精神的化身"——人民崇敬他，因为他代表了科技、进步、民主。但科技越发达，人民的生活越不依赖任何超凡者。他的锚点在**他最引以为傲的成就中缓慢死亡**。

格里高尔的选择映射Q5：他选择继续推动科技——"如果我的锚点注定消亡，那就让它消亡在正确的事业中。"这是本书中最高贵的悲剧之一。

### 矛盾二：灵质三角锁

```
瓦尔德需要灵质发展科技
    → 灵质来自科尔曼（草原的"血脉"）和米拉尔（"神赐"之物）
    → 科尔曼过度开采灵质 = 游牧文化灭绝 + 巴图尔的文化锚点崩塌
    → 米拉尔出口灵质 = 亵渎 + 卡尔文的信仰锚点面临"世俗化"侵蚀
    → 塞伦作为中间商 = 利润至上，但信息网络将绕过中间商
    → 每一方的合理诉求都在伤害其他方
    → 没有人是"坏人"，但局势必然恶化
```

**阿尔温的视角**（穿越者红利）：他看到了"资源诅咒"的本质——灵质之于奥里斯塔尼，如同石油之于中东。资源富国反而最脆弱，技术强国反而最依赖。

### 矛盾三：信息透明死亡螺旋

```
信息网络重建 → 信息民主化
    → 超凡者的秘密（锚点需求）面临曝光风险
    → 政治操控升级（假信息 + 信息洪流 = 新型武器）
    → 薇拉的情报垄断崩塌 → 塞伦商业模式崩塌
    → 圣国教义封锁被打破 → 年轻人流失 → 卡尔文锚点侵蚀
    → 所有基于"信息不对称"的权力结构同时动摇
    → 但信息透明不等于人民获得真相——信息过载 = 新形式的蒙蔽
```

**终极讽刺**：阿尔温的职业本质就是"揭示被隐藏的信息"（灵痕分析师——"灵痕不说谎"）。他天然站在"信息透明"一边。但当他发现"有些真相被隐藏是有原因的"（锚点崩溃→超凡者疯狂→大规模死亡），命题α和命题β在这里交汇：

> **揭示真相是正确的——但如果揭示的代价是让七阶者失去锚点并发疯呢？**

### 各卷命题焦点详述

**V1「裂缝」**——命题β以暗线存在

读者看到的是一个悬疑故事（D案件+画商之死），但制度问题的"日常温度"已经渗透在每个场景中：
- 德米特里对信息的操控 → 制度性压迫的微观体现
- 执法局的规则变更（新标准三天前发布=案发之后）→ 权力对专业的干预
- 提耶林城的阶层差异 → 普通人在超凡者阴影下的真实生活
- 普通读者不需要意识到这是"政治哲学"——他们只需感觉到"这个世界有什么地方不对"

**V2「棋局」**——锚点政治正面浮现

莱奥里克举旗。三执政的崩塌不只是政变——它是"没有真正八阶个体的国家"这一结构性缺陷的必然结果。三执政的锚点是"恐惧+制度服从"（质量等级1-2），本就是最脆弱的类型。
- Q1正面化：莱奥里克需要人民认可维持理智——血洗之日后，他获得的是恐惧还是感恩？
- 安东尼奥之死的真相（α）揭示三执政对国家锚点的寄生式利用（β）
- 阿尔温流亡瓦尔德——首次近距离观察"科技文明"的面貌

**V3「野心王」**——第一个实验场

莱奥里克在卡萨林建立新秩序。这是命题β的第一次正面实验：
- 一个理想主义者用暴力推翻旧制度后，如何建立新制度？
- 莱奥里克 vs 阿德里安的战斗不只是个人对决——它是"人格崇拜锚点"vs"军事忠诚锚点"的碰撞
- 瓦尔德科技（符文武器）开始改变力量格局——Q5初次触发
- 阿尔温在塞伦重建生活，开始接触信息网络、商业体系——为V4势力建设埋种

**V4「暗棋」**——命题β成为主导

阿尔温利用穿越者红利（现代管理学+信息战+社会科学）建立自己的势力。他必须正面回答："我要建立什么样的组织？"
- 灵质贸易战成为六国博弈暗线（矛盾二·灵质三角锁激活）
- Q5-Q6正面展开：科技正在改变权力结构，但谁控制科技？
- 阿尔温的"第三条路"开始显形——但远未完善
- 与格里高尔的深度交流——科技替代悖论的直面

**V5「面具」**——三维碰撞

阿涅丝面具身份揭示的情感核爆与命题β深度绑定：
- 阿涅丝选择血月教的原因与锚点政治相关——她不是被洗脑，是基于对锚点系统的理解做出的选择（或自以为理性的选择）
- Q7：魔偶权利问题浮上政治台面——"什么算'人'"成为各方博弈的棋子
- Q8：信息网络扩张引发连锁反应——薇拉的情报优势缩小
- 命题α和β首次激烈碰撞："真相该被公开吗？"+"公开后制度能承受吗？"

**V6「更迭」**——全部问题交汇

所有真相汇聚。命题α和β完全交融：
- "锚点的真相"本身就是最大的真相——如果所有人知道"七阶以上者需要人民当锚点"，整个世界的政治结构将被重写
- 灵质危机迫使所有势力做出终极选择
- 阿尔温必须决定：公开锚点真相（可能引发大规模锚点崩溃→超凡者疯狂→灾难），还是以另一种方式处理？
- 全部Q1-Q8在此卷中交汇——不是给出答案，而是将所有矛盾推到最尖锐的状态

**V7「诸王黄昏」**——开放式终局

终局不是"哪种制度胜出"——而是"能否超越锚点政治 × 超凡科技 × 制度博弈的三角困局"。
- 阿尔温的"第三条路"在这里接受终极考验
- 每个国家、每个角色都走向自己的命题结局——没有统一的"正确答案"
- 读者带着自己的思考离开——"如果是我，我会选择哪条路？"

## 七、阿尔温的"第三条路"（命题β的终极回应）

### 穿越者红利在命题β中的独特价值

阿尔温来自我们的世界。他的知识框架让他看到奥里斯塔尼各势力看不到的底层逻辑：

**1. 科技普及 = 对自己锚点最好的投资**

> "如果我帮助人民真正提升生活质量，他们的认可是建立在'我确实让他们过得更好'之上的。不是恐惧，不是信仰，不是利益交换——而是真实的感激和尊重。"

穿越者视角：前世"服务型政府"的理念——治理者的合法性来自实际贡献，而非身份或力量。在锚点政治语境下，这意味着：**主动帮人民获得不需要你的能力，反而让他们更愿意支持你——因为他们知道你是真心的。**

**2. 信息平台而非信息垄断**

> "薇拉靠垄断信息维持权力。但真正持久的影响力不来自垄断——而来自让信息流通，并在流通中创造价值。"

穿越者视角：互联网平台思维——信息的价值不在于独占，而在于连接和筛选。阿尔温不做"情报头子"，做"信息基础设施"。

**3. 制度设计而非个人统治**

> "莱奥里克靠个人魅力统治，但个人终将老去或被侵蚀。真正持久的是制度——法治、分权、监督、制衡。"

穿越者视角：政治学基础——权力需要制度化而非人格化。但在超凡者存在的世界，如何让制度约束"可以灭城的个体"？

**4. 社会科学 = "看不见的灵契"**

> "前世的经济学、社会学、心理学——它们就是'看不见的灵契'。不能毁城，但能建城。"

穿越者视角：阿尔温的真正"金手指"不是超凡力量——而是跨越文明的认知框架。他用前世的社会科学知识设计组织结构、激励机制、信息系统。

### 第三条路的致命缺陷（V6-V7必须面对）

阿尔温的路径不是完美的——它有自己的自毁种子：

| 缺陷 | 根源 | 叙事展开点 |
|------|------|-----------|
| **科技替代反噬** | 帮得越多→被需要越少→锚点自我消解（重蹈格里高尔覆辙） | V6：阿尔温的势力壮大后，发现自己的"被需要度"在下降 |
| **信息可被灵性干扰** | 彼界的存在意味着信息可以被"灵性操纵"——前世的互联网没有这个问题 | V5-V6：信息网络被灵性力量入侵/干扰 |
| **分权制衡的超凡困境** | 当一个人可以灭城时，"三权分立"怎么制衡？宪法约束不了八阶超凡者 | V7核心困境：制度可以约束普通人，但如何约束比制度更强大的个体？ |
| **前世理论的适用性** | 这个世界有灵、有彼界、有锚点——前世的社会科学需要大幅修正，不能照搬 | 贯穿V4-V7：每次"照搬"都失败，逼迫阿尔温创造性地融合两个世界的智慧 |
| **穿越者的锚点悖论** | 阿尔温的"真实身份"是什么？穿越者的记忆是"真我"还是"植入"？他的锚点建立在哪个自我之上？ | V7终极提问：与Q7（魔偶自我意识）哲学共振 |

### 第三条路的开放性

V7的结局不是"阿尔温成功建立了完美制度"——而是：

> 他证明了一种**可能性**：锚点可以建立在"真实的服务"而非"权力的索取"之上；科技可以是平等的工具而非统治的手段；制度可以容纳超凡者而非被超凡者践踏。
>
> 但这条路是否可持续、是否可推广、是否经得起时间考验——留给读者思考。

## 八、命题表达方式——"日常即哲学"原则

### 禁止清单（红线）

| 类型 | 具体禁止 | 替代方案 |
|------|----------|----------|
| 说教 | 角色直接发表政治/哲学演说超过200字 | 通过行动、选择、后果展示 |
| 概念倾倒 | 一次性引入3个以上新政治/哲学概念 | 每章最多自然引入1个新概念 |
| 二元化 | "X制度好Y制度坏"的简单判断 | 每种制度至少展示1个真实优点和1个致命缺陷 |
| 作者传声筒 | 阿尔温成为作者的政治观代言人 | 阿尔温的每一个观点都必须被至少一个角色有力地反驳 |
| 现实直射 | 直接影射现实国家/政治事件/政治人物 | 保持虚构世界的独立性，读者自行联想 |
| 结论先行 | 先给结论再找论据 | 先展示矛盾，让读者和角色一起困惑 |

### 主题的日常温度计

命题不是通过"讨论"传递的，而是通过**日常生活的温度差**让读者自行感受：

| 日常场景 | 暗含的命题 | 读者感受 |
|----------|-----------|---------|
| 不同国家面包价格差异五倍 | 制度影响经济→经济影响民生 | "这个世界的不平等是真实的" |
| 普通人见到超凡者的反应（恐惧/崇拜/躲避/麻木） | 锚点政治的民间投射 | "权力有很多种面孔" |
| 符文家电在富人区普及但贫民区用不起 | 科技进步的分配不均 | "科技不自动带来平等" |
| 魔偶在街上被人踢但不反抗 | 非人类存在的权利问题 | "什么算'活着'？" |
| 两个老人争论"帝国时代是不是更好" | 历史记忆与制度评价 | "怀旧是因为现在不够好" |
| 边境检查站的不同态度（联邦友善/圣国盘问/汗国放行） | 制度差异的日常体感 | "过个边境就是另一个世界" |
| 孩子问"为什么圣国的人不能读这本书" | 信息管控的日常面 | "自由不是理所当然的" |
| 咖啡馆里灵质价格的涨跌引发骂声 | 灵质政治的民生投射 | "资源战争就在你的餐桌上" |
