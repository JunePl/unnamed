# 编纂说明

```
本文档是小说的"心电图"——控制读者情绪起伏如同控制心跳。

核心指标：
· 压抑值(0-10)：紧张、焦虑、悲伤、愤怒的累积
· 释放值(0-10)：爽感、感动、释然、胜利的释放
· 钩子密度：每千字的悬念/伏笔数量
· 断章指数：章末留悬念的强度(1-5)

商业铁律（不可违反）：
· 首章800字内出现第一个冲突
· 每3章至少一个情绪释放点
· 压抑值连续>8不得超过2章
· 每卷至少3个"可截图传播"的名场面
```
