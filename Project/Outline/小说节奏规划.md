# 编纂说明

```
本文档是小说的"心电图"——控制读者情绪起伏如同控制心跳。

核心指标：
· 压抑值(0-10)：紧张、焦虑、悲伤、愤怒的累积
· 张力值(0-10)：悬念、期待、信息密度的累积
· 爽感值(0-10)：爽感、感动、释然、胜利的释放
· 断章指数(1-5)：章末悬念强度
· 线索标记：[A]=画商线 [B]=政治线 [C]=彼界线 [D]=血月暗线

章节类型标记（小说编辑·呼吸系统）：
· [核心章]：重大剧情推进、真相揭露、高潮对决（紧张上限10）
· [过渡章]：衔接事件、伏笔埋设、情绪缓冲（紧张上限6）
· [日常章]：纯角色互动、世界观展示、读者休息（紧张上限3）
· [暗流章]：表面日常但暗藏伏笔（紧张上限4）
· [微谜章]：日常中的小谜题（紧张上限4）

商业铁律（不可违反）：
· 首章800字内出现第一个冲突
· 每3章至少一个情绪释放点
· 压抑值连续>8不得超过2章
· 每卷至少3个"可截图传播"的名场面
· 连续核心章不得超过2章
```
