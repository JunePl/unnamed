# 平台特调策略

## 起点适配

```
目标读者：20-35岁男性，偏好严谨世界观+智力博弈
适配重点：
· 世界观渗透密度保持高位（伏笔密度5+/千字）
· 三执政政治博弈线需要充分展开
· D模式悬念的逻辑严谨性是核心竞争力
· 允许3章慢热开局（但V2.0已改为尸体开场，不再慢热）
· 每5章至少1次推理展示
调整项：
· Ch1尸体开场后，场景B酒馆可适当增加政治信息密度
· 增加阿尔温的内心独白量（起点读者喜欢"看主角在想什么"）
```

## 番茄适配

```
目标读者：18-30岁，快节奏偏好，需要强首章
适配重点：
· Ch1必须800字内出现第一个强冲突 → 已满足（尸体开场）
· 每章末必须有悬念钩子（断章指数3+）
· 幽默密度可适当提高（兄妹互动是天然优势）
· 减少纯世界观说明段落，通过对话和场景自然渗透
调整项：
· 酒馆场景的哥布林闲聊可以增加趣味性
· 阿涅丝的"伏击"场景可以加入更多动作描写
```

## 刺猬猫适配

```
目标读者：16-25岁，ACG文化背景，强角色偏好
适配重点：
· 角色标签化："戴单镜的侦探"+"怪力少女妹妹"
· 每章1个可制成表情包的桥段（"汪""1号按摩技师"）
· Gen-Z梗的嵌入需要谨慎——不硬塞，通过角色自然说出
· 单镜+手套+乱发=优秀的角色立绘素材
调整项：
· 强化阿涅丝的"反差萌"特质
· 小织作为"灵宠"可以增加互动戏份
· 哥布林角色可以增加"吐槽担当"功能
```

# 基调双轨标记系统（V2.0新增·冲突#9解决）

```
【基调双轨标记说明】

从V2.0起，每章的压抑值/释放值区分两条线：

案件线基调（A轨）：严肃悬疑为主
· 案件现场/调查/推理/对峙场景
· 幽默元素不超过5%
· 压抑值通常较高

日常线基调（D轨）：温馨幽默为主
· 回家/酒馆/朋友互动/日常琐事
· 允许高密度轻松元素
· 释放值通常较高

标注格式：(A:压X/释Y  D:压X/释Y)

示例：
Ch1 = (A:压5/释3  D:压1/释8)
→ 案件线严肃紧张，日常线温馨释放
→ 两条线交织形成"重压中的微光"效果
```
