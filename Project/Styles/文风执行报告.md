# 一、正文文风参数提取

## 1.1 叙事视角与时态

```
【正文实际使用】
· 视角：第三人称限制视角（以阿尔温为主视角锚点）
· 时态：过去式为主，关键时刻切现在式增加临场感
· 视角切换：暂未出现切换（Mian.novel仅覆盖阿尔温视角）
· 内心独白：极少，以行为和对话暗示心理
  → 示例：阿尔温"叹了口气，没有说什么"（不写内心）
          阿尔温"眨了下眼"（微动作代替心理描写）

【执行规则】
· 案件线：第三人称限制视角（阿尔温），禁止全知叙述
· 日常线：第三人称限制视角（可短暂切阿涅丝，但需章节单元内保持一致）
· 内心独白使用频率：每千字不超过1次，且必须精准——不解释已通过行为展示的情绪
```

## 1.2 描写密度与节奏

```
【正文实际参数】
· 平均句长：15-25字（短句为主，紧凑感强）
· 对话比例：约55-60%（对话驱动型叙事）
· 环境描写密度：低，但精准——每个环境细节都服务于氛围或暗示
  → 示例："蜡烛在画框之间摇曳，火焰像是在犹豫要不要继续燃烧"
          （仅一句，但同时建立了画廊氛围+暗示"生死"主题）
· 动作描写密度：中等，但细节感强
  → 示例："一只手细致的调整镜片，一只手轻轻托起死者的下巴"
          （不写"他开始分析"，写具体动作）
· 留白技巧：大量使用（正文最大特色之一）
  → 示例："阿涅丝的声音卡住了。""沉默在客厅里蔓延了三秒。"

【执行规则】
· 句长控制：案件线平均18字/句，日常线可放宽至22字/句
· 对话驱动：每章对话比例不低于50%
· 环境描写：每场景不超过3句，必须兼具氛围+功能
· 动作描写：以具体小动作代替抽象心理描写
· 留白：每章至少2处留白设计（沉默/省略号/转场空白）
```

## 1.3 场景基调分层（冲突#9执行方案）

```
【案件现场·严肃基调】
正文示例（画廊）：
"但尸体从来不在乎这种气氛。"
→ 冷幽默切入（仅此一句），之后全程严肃
→ 灵痕分析过程：精确、专业、无废话
→ 与哈罗德的对话：直接、有力、各自立场清晰
→ 幽默元素：≤5%（且必须是角色性格自然流露，不是叙述者的调侃）

【酒馆·半放松基调】
正文示例（夜莲酒吧）：
"莫哈夫咧嘴笑了笑，转身进了后厨。"
→ 阿尔温在熟悉环境中略微放松
→ 对话简短、默契——"老一套吧"
→ 背景信息通过旁人闲聊渗透（哥布林讨论太阳）
→ 幽默元素：约20%

【回家·温馨基调】
正文示例（兄妹夜谈）：
全段从"咚……咚"到"时钟的指针划过十点四十分"
→ 幽默密度最高：伏击、蛋糕贿赂、"汪"、1号按摩技师
→ 但穿插了真实情感时刻："恭喜你。你做到了。"
→ 暗线种子在温馨中几乎不可见
→ 幽默元素：约40-50%（仅限兄妹互动场景）

【执行总则】
案件线幽默占比 ≤ 5%
过渡线幽默占比 ≤ 20%
日常线幽默占比 ≤ 50%
兄妹互动幽默占比 ≤ 50%
随剧情推进（Ch50后），日常线幽默占比逐步下降至 ≤ 30%
```

# 二、角色语言基因 vs 正文对照

## 2.1 阿尔温·语言基因验证

| 人设集定义 | 正文实际表现 | 匹配度 | 备注 |
|-----------|-------------|--------|------|
| 冷静、精确 | "他的表情不是恐惧，是惊讶。所以他认识凶手。" | 完全匹配 | 推理简洁有力 |
| 干燥幽默 | "卡萨林皇家学院里的知名帅哥的颜值好" | 匹配 | 自嘲式幽默 |
| 思考时停顿 | "嗯……"（分析尸体瞳孔时） | 匹配 | 自然停顿 |
| 放松时调戏 | "汪""我心爱的妹妹" | 完全匹配 | 仅对亲密之人 |
| 压力下句子变短 | 暂未出现高压场景 | 待验证 | Ch16-20需要验证 |
| 命令式关心 | 连珠炮追问阿涅丝进阶代价 | 完全匹配 | 核心特征 |

**阿尔温文风执行规则**：
```
案件场景：
· 句式：短句为主（"这案子，是谋杀。"）
· 对白：直接、不绕弯（"你想说什么，现在说。"）
· 分析时：列举式（"第一……第二……"）→ 但不超过3条
· 幽默：极少，且必须是冷幽默（"那没办法"式轻描淡写）

日常场景：
· 句式：可适当延长，允许语气词（"啊……哈哈哈"）
· 对白：带调侃、带撒娇式讨好（仅对阿涅丝/莫哈夫）
· 叙述者视角下的阿尔温：通过微动作展示情绪
  → 揉头发 = 尴尬/不知所措
  → 手指敲桌面 = 在下决心
  → 叹气 = 无可奈何的温柔
```

## 2.2 阿涅丝·语言基因验证

| 人设集定义 | 正文实际表现 | 匹配度 | 备注 |
|-----------|-------------|--------|------|
| 活泼、直接 | "准备好去死吧！""你再提他一次试试。" | 完全匹配 | 语气凌厉 |
| 语速快 | 连珠炮式追问/控诉 | 匹配 | 对话密度高 |
| 带小性子 | 撅嘴/偏头不看/气鼓鼓的裙摆 | 完全匹配 | 身体语言丰富 |
| 委屈时声音变小 | "我等了你一整天，想看你惊讶的表情。" | 完全匹配 | 声音变轻=真心话 |
| 喜欢比喻 | 暂未大量出现 | 待补充 | 正式写作时增加 |

**阿涅丝文风执行规则**：
```
· 语速标记：正常对话无标注；兴奋/愤怒时加注"连珠炮"
· 称呼变化：
  "哥哥——"（拖长=撒娇）
  "阿尔温！"（全名=生气）
  "阿·尔·温·埃·蒙·德！"（一字一顿=非常生气）
· 身体语言优先于台词：
  "阿涅丝的嘴角微微抽动了一下"比"阿涅丝觉得有趣"强100倍
· 小织互动：
  小织的动作反映阿涅丝的真实情绪（阿涅丝嘴硬时小织在偷笑）
  → 这是读者"读懂"阿涅丝的第二通道
```

# 三、特殊文风元素清单

## 3.1 哥布林对话风格

```
正文参照：
胖哥布林："我亲眼见到太阳的战斗！""你这辈子都想不到！"
耳坠哥布林："吹吧你。""真的假的！听说霍恩家那位，可帅了！"

特征提取：
· 嗓门大（"！"使用频率高）
· 信息传播者（酒吧闲聊=世界观渗透工具）
· 夸张表达（"我就在城北教区！"）
· 彼此吐槽但关系融洽
· 八卦属性强

执行规则：
· 哥布林对话用于：世界观信息自然渗透、增添场景烟火气
· 不可让哥布林说出超出其认知范围的信息
· 哥布林对话幽默度可以较高（它们是天然的喜剧角色）
· 但不可沦为纯搞笑——偶尔通过哥布林的视角展示阶级问题
  → 示例：西普的权利运动、灵魔器（灵力装置）的黑市交易
```

## 3.2 关键意象复用

```
正文中建立的意象（需在后续写作中保持一致）：

1. 蛋糕意象 = 兄妹关系的温度计
   · 熔岩蛋糕 → 和解、温暖
   · 阿尔温做的蛋挞 → 失败、笨拙的爱
   · 随剧情发展，蛋糕场景的频率下降 = 日常温暖在流失

2. 单镜意象 = 阿尔温的"真实之眼"
   · 戴上单镜 = 进入分析模式 = 看见真相
   · 摘下单镜 = 回到普通人视角 = 不想面对
   · 单镜碎裂（某个关键时刻）= 分析能力的局限性

3. 灯光意象 = 世界的明暗
   · 画廊蜡烛 = 犹豫的真相
   · 酒馆灯火 = 脆弱的安全感
   · 回家后的水晶灯 = 日常的光明
   · 夜色/浓雾 = 外部世界的不确定
```

# 四、文风禁区

```
【绝对禁止】
1. 上帝视角评论角色心理（"他心想""她感到"→改为微动作/微表情）
2. 大段世界观说明文（必须通过对话、告示、书籍片段渗透）
3. 角色在案件场景说与性格不符的笑话
4. 用现代网络用语（除非是哥布林特有的市井腔调，且需控制）
5. 叙述者主观评论（"这很讽刺""命运弄人"→不说，让读者自己感受）

【谨慎使用】
1. 感叹号：案件线中仅限极端情绪爆发；日常线中可用于阿涅丝
2. 省略号：正文中使用较多，需控制频率→每千字不超过3处
3. 括号注释：正文中未使用，不建议在正式写作中引入
```
