## 一、基础档案

```
姓名：格里高尔（Grigor）——只用单名，联邦传统
年龄：41岁
身份：瓦尔德联邦事实上的最高领导者/"联邦精神的化身"
灵契：铸世·造物型（第八位阶·执律）
天秤排名：冠位区间（具体排名待定，前10）
国家：瓦尔德联邦

【六国志验证·L397】
格里高尔：铸世·造物型，第八位阶——已确认

【为什么他是最特殊的8阶】
六国有多个8阶强者（卡尔文、巴图尔等）。
但格里高尔是唯一一个——明确选择"不以力量统治"的8阶。
卡尔文用信仰锚点。巴图尔用文化锚点。阿德里安用军事锚点。
格里高尔用的是：民主尊重——他让联邦公民真正选择他。
这意味着他的锚点质量是5级（民主尊重），不是最高的——
但他认为这是"唯一正当的方式"。

【核心定位】
格里高尔不是一个"强者选择谦卑"的故事。
他是一个"明知民主锚点会削弱自己，依然选择民主"的故事。
他的克制不是性格——是信念。
但信念有代价：科技正在让普通人越来越不需要他，
而他的锚点（民主尊重）恰好建立在"被需要"之上。
当人民不再需要你时，你的理智怎么维持？
这是格里高尔的终极困境。
```

## 二、弧线设计

```
【弧线类型】：平弧（信念不变，改变世界）
格里高尔的信念始终如一："力量属于人民，不属于个人。"
他不会变。世界会因为他而变。
但"不变"本身的代价会越来越大——
这就是平弧的悲剧性：越坚守，越痛苦。
```

## 三、核心矛盾

```
【灵魂撕裂点】

"我可以一个人保护整个联邦。
 但如果我这么做了——联邦就不再是联邦了。"

矛盾A：力量 vs 克制
  8阶可以灭小国。他每天都在克制这种力量。
  不是因为不想用——是因为用了就等于证明"力量比制度更可靠"。
  而他一生都在证明相反的事。

矛盾B：科技替代悖论（主题框架·三重核心矛盾之一）
  他支持科技普及——因为普通人应该有自我保护的能力。
  但科技越普及，人民越不需要超凡者的保护。
  人民越不需要他，他的锚点（民主尊重）就越弱。
  锚点越弱，他越接近失控。
  失控的8阶=行走的灭国武器。
  所以：他支持的事业正在杀死他。

矛盾C：公共 vs 私人
  他的公共形象是"联邦精神"——一个象征，不是一个人。
  但他是一个人。他也会疲惫、孤独、怀疑。
  只是没有人被允许看到这一面——
  因为如果联邦精神动摇了，联邦就动摇了。
```

## 四、怪癖系统

```
1. 每周六去工厂当工人
   8阶最强者，每周六穿工装去联邦的某个工厂干一天体力活。
   不是作秀——没有记者跟拍，没有安保，没有人知道。
   他去的是不同的工厂：灵质加工厂、魔偶装配车间、港口仓库。
   工人们不认识他（他不用真名）。
   他在流水线上干活，吃食堂的饭，听工人聊天。
   "如果我不知道普通人的一天是什么样的，
    我怎么替他们做决定？"
   -> 叙事功能：通过他的工厂体验展示联邦底层生活

2. 保留旧伤手疤
   左手背有一道很深的疤——年轻时在工厂事故中留下的。
   8阶的恢复力完全可以让它消失。他选择保留。
   "提醒自己：力量解决不了所有问题。
    这道疤是机器留的，不是灵契者留的。"

3. 站在画廊看台而非主席座
   联邦议会大厅里，最高领导的座位在正中央主席台。
   格里高尔从不坐那里。
   他站在二层的公众看台——和旁听的市民站在一起。
   在看台上听完辩论后才下来发言。
   "我是来听的。不是来命令的。"

4. 不戴任何徽章/勋章
   联邦有各种荣誉勋章。格里高尔一个都不戴。
   穿着简朴到像个普通教师——深色外套，白衬衫，没有装饰。
   和他站在一起，你猜不到他是联邦最有权势的人。
   "勋章是给需要证明自己的人的。"

5. 修东西
   办公室里总有他正在修的东西——旧钟表、坏了的灵能台灯、裂了的椅子。
   铸世·造物型的本能——他手痒。
   但他不用灵契修——他用工具，手工修。
   "用灵力太快了。手工修的时候能想事情。"
```

## 五、语言基因档案

```
基调：朴实、直接、偶尔带着工人阶层的粗粝
语速：中等，不快不慢，像在和你平等对话
特征：
. 从不用"我命令""我要求"——总是"我建议""大家觉得呢"
. 偶尔冒出工人俚语——联邦元首说工人话，这在议会里很有辨识度
. 说到技术问题会突然变专业（铸世·造物型的本能）
. 面对挑衅时不怒——但声音会变得更沉、更慢

示例（议会发言）：
"各位议员，我听了三个小时的辩论。
 你们说了很多关于'联邦精神'的话。
 但我今天早上在第七区的供水厂看到——
 三个工人用扳手修一根管道，修了四个小时。
 他们没提过一次'联邦精神'。
 但那根管道修好了。
 也许联邦精神不在这个大厅里。
 在那根管道里。"

示例（与凯撒的对话——罕见的私密时刻）：
"你问我怕不怕？
 ……怕。
 不是怕死。是怕有一天——
 我站在看台上，底下的人不再抬头看我。
 不是因为他们不尊重我。
 是因为他们不再需要我。
 那一天……我的锚点就没了。
 你知道一个8阶失去锚点意味着什么。"
```
