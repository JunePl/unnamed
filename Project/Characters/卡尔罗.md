## 二、真灵档案：渡物·手型

### 能力规格

| 维度 | 规则 |
|---|---|
| **核心能力** | 将自己亲手接触过的物品·传送到自己亲手接触过的坐标点 |
| **质量上限** | 约 10 公斤（能放入标准信封/手提箱的体积） |
| **距离上限** | 同城半径约 15 公里（卡萨林市内全覆盖·不可跨城·不可跨国） |
| **前置条件** | 目标坐标点必须是他**亲手接触过**的位置——"摸过墙"或"坐过椅子"等级的物理接触·记住坐标 |
| **冷却时间** | 每次传送需约 2 分钟冥想锁定坐标·每天使用上限 5-8 次（取决于状态） |
| **代价** | **单向传送**（只能传出·不能把东西传回自己手里）/ 传送瞬间自己不能移动 / 每次消耗中度灵力（连续多次使用后半天无法再用） |

### 灵痕签名

- **淡蓝色边缘的"传送余波"**——物品被传送到目的地时·表面会留下一层极淡的蓝色残留（边缘特征明显）
- 三序以上的灵使借单片镜可辨识
- **不可伪造**——每个渡物型灵使的余波色调略有不同·卡尔罗的是"冷蓝偏灰"
- 这个签名**在卡萨林本地只有卡尔罗一个人**——因为渡物型真灵极其罕见·整个卡萨林只有他

### 他为什么被安东尼奥用了 20 年

卡萨林没有第二个渡物型三阶灵使。整个卡萨林地区这类真灵的现役持有者不超过两人·能够稳定为一个人服务的只有卡尔罗。**他不是强战力·他是不可替代的渠道**——这就是安东尼奥看重他的原因。

（伏笔：这也是德米特里买通他的原因·垄断了卡尔罗=垄断了安东尼奥的"安全通讯渠道"）

## 四、个性与对白模式

### 性格特质

- **精确**：从不说不必要的话·每个字都是他斟酌过的
- **克制**：20 年从没在客户面前表达过情绪·包括现在
- **冷外热内**（与巴克利相似·但比巴克利更深）——他是内应·但他对安东尼奥确实有几分真心·只是这几分真心从不会动摇他的立场
- **职业荣誉感**：他给自己的辩护是"我只是在两边都尽职"

### 对白模式

**A 模式·公事公办**：
- "埃德蒙先生。公爵让我问您·这份文件您收到了？"
- "公爵早就预料您会这么说。"
- "我建议您先把卡萨林的事安排好。"

**B 模式·委托人秘密**：
- "您不告诉他吗？"（对安东尼奥·关于是否警告阿尔温被跟踪）
- "如果他出事呢？"
- （安东尼奥沉默三秒后·他也就不再问）

**关键动作标记**：
- 收起眼镜 = 要说重话
- 白手套调整指节 = 在压住情绪
- 说话前先低头看表 = 他已经计算好时间

## 八、设计注记·Why

**为什么选渡物·手型·三阶？**

1. **三阶** = 工具人·不是战力·这匹配他的"内应"设定（不够强所以容易被德米特里渗透·没办法反抗）
2. **渡物·手型** = 特化型·不战斗·极稀缺·这匹配安东尼奥 20 年用他的原因
3. **前置条件"必须亲手接触过目标位置"** = 这是他的**结构性限制**·让能力不显得万能
   - 他为什么能传到阿尔温办公室？因为他之前来过一次·摸过桌子
   - 这也是伏笔：Ch21 之前他就**已经来过卡萨林探过路**
4. **淡蓝色余波灵痕** = 给阿尔温的推理留了线索·但线索要登门配合才能锁定身份
5. **单向传送** = 让能力不能用来"传送证据回来"或"紧急救援"·避免能力被滥用成剧情开挂

**为什么让他是内应但安东尼奥知道？**

这是 PB24 原始设计。物品传送能力完美契合这个设定：
- 传送文件 = 德米特里可以要求看副本·传送=可被监控的信息渠道
- 安东尼奥故意用他 = 让德米特里以为自己掌控了通讯·实际上安东尼奥通过传送给德"安全的真实信息"来操纵德的判断
- 安东尼奥的真正后手从不经过卡尔罗 = 关键证据/遗嘱的真正传递路线另有其人
