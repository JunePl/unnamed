## 一、基础档案

```
姓名：薇拉（Vera）——从不使用姓氏
称号："幻弄师"
年龄：外表30岁左右（实际年龄不详——幻象源流的副作用）
身份：塞伦共和国隐性最高权力者/情报垄断者
灵契：幻象·支配型（7阶·主宰）
天秤排名：柱位区间（系统篇记载）
国家：塞伦共和国

【六国志验证·L514-519】
塞伦权力结构：执政官(中等) < 议会主席老安东尼奥·梅迪奇(较高) < 薇拉(最高·隐性)
薇拉掌控情报垄断——塞伦真正的王是她。

【核心定位】
薇拉不是一个"情报贩子"。
她是一个"信息系统"。
塞伦共和国的所有信息流动都经过她——
谁知道什么，谁不知道什么，全由她决定。
她不需要军队，不需要暗杀，不需要恐吓。
她只需要知道一切——然后选择性地告诉每个人不同的"一切"。
```

## 二、弧线设计

```
【弧线类型】：堕落弧 or 觉醒弧（取决于最终选择——暂不写死）

路径A·堕落弧：
  信息民主化浪潮（阿尔温的"第三条路"中的信息平台计划）威胁她的根基。
  她选择对抗变革→越来越极端→从"信息管理者"变成"信息独裁者"。

路径B·觉醒弧：
  在信息民主化的冲击中，她被迫面对：
  "我的权力建立在信息不对称之上。如果每个人都能获取信息——我是谁？"
  从"控制者"转变为"导航者"——不再垄断信息，而是帮助人们理解信息。

先按两条路径并行设计。在V4-V5之间做最终决策。
```

## 三、核心矛盾

```
【灵魂撕裂点】

"我知道所有人的秘密。
 但没有人知道我的。
 这不是胜利——这是最完美的牢笼。"

矛盾A：控制 vs 孤独
  她知道一切——所以她不信任任何人。
  不信任任何人——所以她永远孤独。
  永远孤独——所以她更需要控制一切来获得安全感。
  这是一个闭合的痛苦循环。

矛盾B：信息透明死亡螺旋（主题框架·三重核心矛盾之一）
  阿尔温推动的信息平台如果成功——
  塞伦的信息垄断将被打破。
  薇拉的权力基础将崩塌。
  但如果她阻止信息透明化——
  她就站在了"人民的敌人"那一边。
  而她的锚点（利益交换·第3级）本就脆弱。

矛盾C：面具 vs 真我
  幻象·支配型——她的灵契本质就是"制造幻象"。
  她活在层层面具之中。
  到最后，她自己都不确定哪个是"真的薇拉"。
  那份空白档案——是她保留的最后一个谜。
```

## 四、怪癖系统

```
1. 镜墙
   她的私人居所四面墙都是镜子。
   不是自恋——是"自我监控"。
   幻象·支配型的副作用：她需要不断确认"这个是真的我"。
   每天早上起来的第一件事是站在镜墙前，确认自己的真实面孔。
   "如果有一天我照镜子看到的不是我——那我就完了。"

2. 倒酒但从不喝
   在任何社交场合，她都会为自己倒一杯酒。
   然后整场都端着——从不喝。
   倒酒是社交礼仪（融入人群）。不喝是原则（保持清醒）。
   如果有人注意到她没喝——她会微笑说"在等一个值得干杯的时刻"。
   那个时刻从未到来。

3. 保存一份关于自己的空白档案
   薇拉的档案室里有每个重要人物的详细情报。
   只有一份档案是空的——标签上写着她自己的名字。
   空白不是因为没有信息——是故意保持空白。
   "如果连我自己的档案都填满了——那就没有真正的我了。"

4. 从不在同一个地方睡两晚
   她在塞伦城有七处安全屋——每晚轮换。
   不是因为安全（7阶不怕暗杀）——是因为不安。
   如果她在同一个地方停留太久，她会开始觉得"被锁定了"。
   幻象源流的人天生恐惧"被确认"——因为确认=可被攻击。

5. 收集他人丢弃的信件
   薇拉有一个奇怪的爱好：在废纸篓里翻找别人扔掉的信件。
   不是为了情报——她有更高效的获取方式。
   她只是想看："人们会扔掉什么样的文字？"
   那些被扔掉的往往是最私密的——表白信、道歉信、未寄出的求助。
   她把这些信收藏起来。
   -> 这是薇拉唯一的"感性"行为。她在别人丢弃的情感碎片中
      寻找某种她自己无法拥有的东西。
```

## 五、语言基因档案

```
基调：优雅、模糊、像一面会说话的镜子
语速：中等偏慢，每个词都精确选择
特征：
. 从不给出明确答案——总是用问题回答问题
. "你确定你想知道吗？" "那取决于你问的是哪个版本的真相。"
. 偶尔会复述对方的话——但微妙地改变了含义
. 沉默时比说话时更可怕——因为你知道她在"处理"你说过的一切

示例（情报交易）：
"你要的信息我有。
 但你确定你想听吗？
 有些东西，知道了就退不回去了。
 ……好吧。
 那你用什么来换？"
（她不要钱。她要"等价信息"。）

示例（被问及私人问题）：
"我？你想了解我？
 你现在看到的就是我。
 ……不满意？
 那也许你该问：你想看到哪个版本的我？"
```
