## 一、基础档案

```
姓名：伊索尔德·温特（Isolde Winter）
代号：第一星（六穿越者中最早到达此世界）
年龄：28岁（外表因预言消耗而显老，看起来三十多）
身份：圣国隐居的"神秘预言者" → 教廷秘密顾问
前世：重病患者。大部分人生在病床上度过——
  "上辈子被困在床上。这辈子被困在预言里。"
灵契：织命·预言型（第七序·主宰）——最高位阶的穿越者
天秤排名：未公开上榜（圣国通过特殊阵法隐瞒其存在）
国家：米拉尔圣国（隐居）

【世界观验证·G确认】
. 人物图鉴L269：米拉尔圣国绑定——织命·预言灵契，教廷秘密顾问
. 塞巴斯蒂安·光明为其保护者（人物图鉴L393-396）

  这种信仰构成了她的锚点基础（锚点质量阶梯第7级：宗教信仰）
. 注意：她的排名不公开，但实力在七序——如果上榜应在冠位区间

【核心定位】
伊索尔德不是"全知者"。
她是"看到碎片的痛苦者"。
预言不是"看电影"——是"看到碎片化的可能性"。
她知道很多人的命运……但不完整。
她知道如何改变命运……但不确定改了会不会更糟。
她只能看着、等待、祈祷——然后在正确的时刻说出正确的碎片。
这是世界上最孤独的工作。

【外貌特征】
. 身高165cm，纤弱——长年不见阳光+预言消耗
. 银白色长发（预言代价导致早白，不是天生——和阿涅丝的白发成因不同）
. 紫色眼睛，预言状态时变为全白（瞳孔消失，全眼白光）
. 皮肤苍白，有病态的美感——但不是脆弱，是"燃烧中的宁静"
. 温和的微笑中藏着无尽的悲伤——她知道太多人的结局
. 穿着：宽松的紫色/白色长袍，圣国风格但不华丽
  （不是教廷正式服装——她有意和教廷保持距离）
. 手腕处有预言消耗留下的灵痕烧灼痕迹——像枯萎的藤蔓纹路
. 随身携带一块磨得光滑的河石——"回到当下"的触觉锚点
```

## 二、弧线设计

```
【弧线类型】：觉醒弧（从"被困者"到"选择者"）

核心觉醒：
  前世她被困在病床上，没有选择权。
  今生她有了力量——但预言的负担让她再次"被困"。
  觉醒不是获得更大的力量——是学会"选择"：
  选择什么时候说、什么时候沉默、什么时候放手。

V1·彼界线登场（Line C·中后期）：
  阿尔温探索彼界时遇到她——
  她在彼界的边缘地带以某种方式存在（预言者可在彼界与现实交界处"观察"）。
  V1中她是"谜一样的引导者"——给阿尔温碎片信息但不解释。
  阿尔温好奇+警惕。"你是谁？" "一个看过你很多结局的人。"

V2·旁观者的痛苦：
  预见阿涅丝的堕落——想干预但不能。
  因为她看到：如果干预，更多人会死。
  "选择沉默和选择杀人有什么区别？——区别是沉默更慢。"

V3·正式会面 + 揭示穿越者真相：
  与阿尔温面对面。终于可以倾诉。
  V3是她弧线的转折点——从"被困的观察者"变成"主动的参与者"。

V4-V5·燃尽：
  预言能力开始衰退——生命在消逝。
  V5进行最后一次重大预言——燃尽最后的生命力。
  死前的最后一个微笑：
  "这辈子……我终于做了选择。不是被迫的。"
```

## 三、核心矛盾

```
【灵魂撕裂点】

"我看到了所有人的命运，却不能说出口。
 不是因为不想——是因为说了可能更糟。"

矛盾A：知道 vs 沉默
  她看到了未来——包括很多人的死亡。
  但说出预言可能改变未来，改变后可能更糟。
  "蝴蝶效应"让她被迫沉默。
  她必须眼睁睁看着悲剧发生——这是最大的折磨。
  "每一次我选择沉默，都像是在杀死一部分自己。"

矛盾B：活着 vs 消耗
  每一次预言都在消耗她的生命。
  小预言：轻微头痛。中预言：虚弱几天。大预言：永久失去部分寿命。
  她知道自己活不到最后。
  但为了"让正确的未来发生"，她必须继续预言。
  这是一种自愿的慢性自杀。

矛盾C：前世 vs 今生
  前世她被困在病床上——无法选择任何事。
  今生她有了力量，但预言的负担让她再次"被困"。
  她真的获得了自由吗？还是只是换了一种牢笼？
  觉醒弧的终点：不是逃出牢笼——是在牢笼中找到自由。
```

## 四、怪癖系统

```
1. 触摸河石回到当下
   随身携带一块磨得光滑的灰色河石。
   每次预言结束后（或接收到碎片时），会紧握河石。
   "感受石头的温度、重量、纹理——这是现在。不是未来。不是过去。"
   → 如果她掉了河石=她在恐慌/失控（极罕见）。
   → 阿尔温注意到后会帮她捡起来——V3的信任建立细节。

2. 编织
   她总在编织——围巾、手套、毯子。
   不是爱好——是"锚定"。织命·预言型的副作用：
   大脑常年处于"接收碎片"状态，编织的重复动作帮她保持理智。
   她编的东西从不给自己——全送人了。
   "编织是唯一能让我想'现在'而不是'以后'的事。"
   → 塞巴斯蒂安的书房里有她编的毯子。阿尔温后来也收到了一条围巾。

3. 在每个房间的角落放一杯水
   圣国的居室、她隐居的小屋——每个角落都有一杯水。
   不是喝的——是"时间标记"。
   水蒸发的速度告诉她"过了多久"。
   预言状态下她会失去时间感——醒来时不知道过了几小时还是几天。
   看水位=知道现实时间。

4. 哼一首没有歌词的曲子
   旋律很简单，像摇篮曲。
   没有人听过完整版——因为她自己也记不全。
   那是前世的记忆——病床上母亲哼给她的。
   跨越两个世界都没有消失的唯一的东西。

5. 对第一次见面的人微笑——因为她可能已经"见过"他们的结局
   见到新面孔时的温柔微笑不是客套——
   是因为她可能在预言碎片中见过这个人的死亡/痛苦/幸福。
   微笑是她能给的最廉价也最珍贵的礼物。
   "至少在我们认识的这一刻，你不知道将来会发生什么。
    保持这种无知吧。它比任何预言都珍贵。"
```

## 五、语言基因档案

```
基调：温和、带着疲惫的悲悯、偶尔空灵
语速：慢，像在斟酌每个字的重量
特征：
. 说话像在"选择"——她知道哪些词可以说、哪些不能
. 常用隐喻和比喻——预言者的职业病，习惯间接表达
. 沉默的时间比说话的时间长——"沉默也是一种选择"
. 偶尔会突然停顿——接收到碎片了

三种模式：

=== 模式A：神谕（对外人/圣国教廷） ===
使用圣国的宗教术语，模糊、充满隐喻。
"光明会在黑暗中诞生……但黑暗也会在光明中孕育。"
"你问的答案……已经在你心中。我只是镜子。"
→ 不是她本来的说话方式——是保护自己的面具。

=== 模式B：真实声音（对亲近的人——塞巴斯蒂安/阿尔温） ===
温柔、略带疲惫。句子更长，更完整。会表达情感。
"我看到了你的未来。很多个版本。"
"有些版本里，你是英雄。有些版本里……"
"但每一个版本里，你都在试图保护你爱的人。"
"这就是我相信你的原因。"

=== 模式C：预言状态 ===
声音变得空灵、不像自己。句子断裂、碎片化。
"看到了……影子……不，是光……光里有影子……"
"他会……不，她会……不对，是他们……"
"结局是……（突然停止，眼睛恢复正常，手抓紧河石）"

示例（V1彼界线·与阿尔温初遇）：
阿尔温："你是谁？"
伊索尔德："……一个看过你很多结局的人。"
阿尔温："什么意思？"
伊索尔德："有些人用分析理解世界。你就是。
 我用……另一种方式。"
（停顿。摸河石。）
"你不需要相信我。你只需要记住一件事：
 你追查的那个东西——比你以为的大得多。"
```

## 六、预言能力的严格局限

```
【重要设定——解决"为什么不全说"】

1. 看到的是"可能性"而非"确定性"
   同一事件可能有多个版本。她无法确定哪个会发生。

2. 看到的是"结果"而非"过程"
   能看到"雷纳德会回来"但看不到"雷纳德是假死"。
   所以她没有阻止——因为她知道结果是好的。

3. 越远越模糊
   近期（几天内）：清晰。
   中期（几周到几月）：模糊。
   远期（几年以上）：只有碎片。

4. 看不到"自己的选择"的后果
   一旦她决定干预，那条时间线就变得不可见。
   这让她无法确定"干预后会不会更糟"。

5. 每次预言都有代价
   小预言：轻微头痛（几分钟恢复）。
   中预言：虚弱几天（需要休息）。
   大预言：永久失去部分寿命（不可逆）。
```
