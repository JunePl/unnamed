## 核心矛盾与心理档案

```
【核心矛盾】：秩序 vs 亲密关系
· 阿尔温信仰秩序和规则——犯罪必须被揭露，真相必须被追查
· 但追查真相的代价是：伤害他在意的人、破坏他珍惜的日常
· 最终极的矛盾：当真相指向阿涅丝时——

【情感进度条】：当前值 25%
· 0% = 理性机器
· 100% = 情感核爆
· 第一卷起始：25%（控制情绪已成习惯，但内心不是冰冷的）

触发事件链：
├─ 画商案发现D模式 → +10%（旧伤口被撕开）
├─ 艾伦被引爆变怪物 → +15%（亲眼目睹灵契被武器化的恐怖）
├─ 自己觉醒灵 → +5%（恐惧：我会不会也失控？）
├─ 卷末血夜事件 → +20%（第一次面对"保护所有人"的无力感）
├─ 莱奥里克出现 → -5%（获得了盟友，压力释放）
└─ 第一卷结束时：约70%

【D模式动机线】（V0.5新增·核心）
三年前阿尔温在执法局时，发现11起表面无关的案件共享一个隐藏特征：
每一起案件中，都有一个"不应该出现在那里"的人或物——
精确地在案件的关键节点出现，引导事件走向特定结局。

经过曾经的蛛丝马迹，他确认这个幕后存在的称呼为"D"。

当他向上级报告时，被要求立即停止调查。
随后被以"不当使用职权"为由"自愿"离职。
11份案件档案被密封。

D模式的特征：
1. 目标人物在死亡前3-7天内出现异常行为变化
2. 案发现场存在"不属于任何当事人"的第三方灵痕残留
3. 关键证据总在调查初期就被"意外"破坏
4. 每起案件都有一个看似合理的表面动机——但真实动机被掩盖

当画商案出现时，阿尔温识别出了D模式：
· 马蒂亚斯死前行为异常（符合特征1）
· 现场有无法归属的灵痕（符合特征2）
· 老彼得"不认识"他这个两周前的委托对象（新的异常）

→ 这不是一个普通的画商谋杀案。
→ "D"又出现了。
→ 这是阿尔温三年来第一次抓住线索。
→ 他不会放手。不是因为职责，是因为执念。
```

## 人设警戒线

```
【OOC高危区】
1. 阿尔温绝不会无故对陌生人展示脆弱面
   → 如需展示，必须有至少2章的信任建立铺垫

2. 阿尔温不会为了"正义"牺牲阿涅丝
   → 当正义与阿涅丝冲突时，他会选择阿涅丝
   → 这个选择本身就是他最大的痛苦
   → 如需改变此倾向，必须通过完整的价值观转变弧

3. 阿尔温面对危险时不冲动
   → 他是分析师不是战士
   → 即使觉醒了灵，他的第一反应仍是"分析"而非"战斗"
   → 如需战斗场景，必须是逼不得已的情况

4. 阿尔温不会轻易信任权威
   → 被执法局"清退"的经历让他对组织有深刻不信任
   → 即使面对莱奥里克，他的信任也是"验证后的信任"

5. 阿尔温的保护欲不等于温柔
   → 对阿涅丝的保护往往表现为"过度控制"
   → 这让阿涅丝感到"被保护但不被看见"
   → 这是兄妹关系的核心张力
```

## 语言基因档案

```
【语言基因·阿尔温】
基调：冷静、精确、偶尔干燥的幽默
语速：中等偏慢，思考时会停顿
习惯用语：
· "让我看看……"（分析时的口头禅）
· "有意思。"（发现异常时的反应，频率不可过高）
· 喜欢用排比式推理："第一……第二……所以……"

情绪变化时的语言变异：
· 放松时：喜欢调戏和开玩笑，通过调戏的话语来确认对方心里的底线并保护自己
· 压力下：句子变短，停顿增多，有时会冒犯到别人，手会无意识握紧
· 愤怒时：声音反而更低更平（压抑型）
· 面对阿涅丝时：语气变柔软，但说出的话经常是"命令式关心"
  （例："回去睡觉。""别在外面待太晚。""你今天吃了几顿？"）

创伤后（第一卷中后期）：
· 句子开始出现碎片化（如果压力值超过60%）
· 会在分析案件时突然走神——因为联想到了D模式
```

```
· 喜欢画画——手指常常沾着颜料
· 穿着比阿尔温讲究得多（会跟潮流）
· 笑容很灿烂——但越到后期，这个笑容下面藏着越多东西
```

## 核心矛盾与暗线

```
【核心矛盾】：被保护 vs 被看见
· 阿尔温保护她，但从不"看见"她的力量和独立意志
· 她渴望被哥哥当作"同伴"而非"需要保护的妹妹"
· 这种不被看见的挫败感，是血月教会能够渗透的心理缺口

【堕落六阶段】（跨越五卷的暗线）
1. 识别目标：血月教会的人注意到她的异常天赋
2. 建立信任：提供"理解"和"认可"——恰好是阿尔温没给的
3. 提供归属：让她觉得"这里有人把我当大人看"
4. 揭示"真相"：告诉她"世界的秩序是假的"
5. 契约：在她主观认为是"自由选择"的情况下接受血月之力
6. 无法回头：发现代价时已经太晚

★ 关键原则：阿涅丝的堕落不是"被洗脑"
她的每一步都有真实的心理动机。
读者在第一遍阅读时会震惊——
但第二遍阅读时会发现所有线索都在那里。

【情感进度条】：当前值 35%
触发事件链：
├─ 毕业典礼哥哥缺席 → +5%（又一次不被看见）
├─ 第一卷前半段日常 → -5%（短暂的幸福稳定期）
├─ 卷末血夜事件 → +15%（第一次见到真正的暴力和恐惧）
├─ 第一卷结束时：约50%
```

# 三、莱奥里克·霍恩

## 基础档案

```
【基础信息】
姓名：莱奥里克·霍恩（Leoric Horn）
年龄：32岁
身份：卡萨林王国·霍恩公爵嫡子，前第一骑士
灵契：璀耀·太阳型（开始时为柱位，故事结束时为最高阶，冠位）
称号："太阳"
天秤排名：柱位63

【核心定位】
太阳 + 莫里亚蒂的集合体。
表面：光明正大、声望极高的英雄人物
暗面：布满整个王国的情报网络，正在谋划"取而代之"

他不是单纯的好人，也不是单纯的阴谋家。
他是一个"为了正确的目的使用一切手段"的人。
他认为德米特里的统治正在毁掉这个国家，
而唯一的出路是由他来主导变革——哪怕代价巨大。
```

## 第一卷出场设计

```
【第一卷·莱奥里克的存在方式】

Ch1-80：莱奥里克不直接登场。
存在方式：
· 通过信件与阿尔温建立联系（匿名信→暗号→逐步信任）
· 新闻/传言中偶尔出现他的名字
· 其他角色谈论他时带着敬畏/恐惧
· 他的暗线（情报网络）通过间接方式帮助阿尔温

Ch81-100（卷末危机）：
· 阿尔温陷入血夜围猎的绝境
· 所有退路被切断
· 在最黑暗的时刻——莱奥里克亲自出现
· 不是偶然路过，而是"他早就在等这一刻"
· 展示压倒性的力量（天秤#2不是虚名）
· 但他不是纯粹的救世主——他救阿尔温，是因为阿尔温对他的棋局有用

→ 爽点+悬念双击：
  "太阳"在最黑暗的时刻降临。
  但这个太阳的光芒照亮的不只是黑暗——
  也照亮了他自己更深层的阴影。
```

## 语言基因档案

```
【语言基因·莱奥里克】
基调：温和、从容、偶尔带着意味深长的微笑
语速：慢，每个字都像是经过计算
习惯用语：
· 极少说"我"，多用"我们""人们"
· 喜欢用提问代替命令："你觉得呢？"
· 偶尔引用古籍或诗歌——不是装腔作势，是真的读过

关键特征：
· 他说话时让你觉得他完全在倾听你
· 但事后你会发现，他引导了整个对话的走向
· 莫里亚蒂式的控制力——包裹在太阳般的温暖中
```
