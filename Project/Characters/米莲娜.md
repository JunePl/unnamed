## 一、基础档案

```
姓名：米莲娜·科斯托娃（Milena Kostova）
彼界名号："霜蛛"（Frost Spider）——阿尔温在V2-V4中只知道这个名字
真名揭露：V4合作期间（Ch30-45）
年龄：V2时28岁（比阿尔温大约2岁）
出身：赫利亚联邦边境城市塔尔科，孤儿院长大
身份：
  外在 — 无国籍自由灵契者（赫利亚联邦已注销其公民档案）
  真实层1 — 赫利亚联邦情报机构"灰幕"的非正式线人
  真实层2 — 深层实体"末央"的契约猎手（核心秘密）

【灵契状态】
灵契：终焉·掠食型
  与阿尔温同为终焉系——同源异途。
  阿尔温观察黑暗，她吞噬黑暗。
  他读懂世界，她猎食世界。

序位进阶：
  12岁觉醒：二序·共鸣
  16岁（末央加速）：五序·临渊（远超正常速度）
  V2出场：五序·临渊
  V5：六序·破界
  V7：七序·显圣
天秤排名：V2时未上榜（刻意隐藏实力）→ V5后进入痕位区间

【外貌特征】
. 身高168cm，纤细但肌肉线条利落——猎手的身体
. 银灰色短发（末央碎片的副作用——原本是黑发）
. 浅蓝色瞳孔，看人时像在评估"你能活几秒"
. 右手手背到手腕有一条旧疤痕——12岁觉醒时末央碎片侵入的印记
. 穿着极简：深色高领+长裤+实用靴。没有首饰。"首饰是弱点的标签"
. 走路没有声音——连灵契者都难以察觉她的移动
. 最明显的特征：灵体形态下，全身覆盖密密麻麻的霜白纹路
  每一条霜纹 = 一个被她收割过的探索者
  V2时约152条。她记得每一条的主人

【核心定位】
米莲娜不是疯子，不是嗜杀者。
她是一个被困在恶魔合同里的理性主义者。
她的猎杀有章法、有准则、有仪式——
这些不是"用来美化暴行"的，而是她"拒绝完全堕落"的最后防线。

她的存在对阿尔温而言是一面镜子：
"如果我在12岁时面临她的选择，我会怎么做？"
这个问题没有答案。这就是它的力量。
```

## 二、弧线设计

```
【弧线类型】：堕落弧→救赎弧（双弧转折）

第一弧·堕落弧（12岁-V4）：
  12岁觉醒时被末央碎片寄生。
  16岁理解处境后"主动选择"接受交易——此为堕落起点。
  不是"被迫的可怜人"——是"计算过后选择活下去"的务实者。
  每一次猎杀都是一次道德底线的微小后退。
  她用"准则"（不杀伤者/不杀孩子/记住每个受害者）来减速堕落。
  但底线在后退——这是不可否认的事实。
  V4前她已经收割152人。

  堕落弧的精髓不在于"她在做坏事"——
  在于"她用最理性的方式管理自己的堕落"。
  这比不自知的恶更令人不安，也更令人同情。

  关键：她从不说"我没有选择"。
  因为她有选择（死）。她选择了活。
  "自愿的恶比被迫的恶更令人痛苦——也更有尊严。"

第二弧·救赎弧（V4-V7）：
  V4被迫与阿尔温合作→第一次被一个人"看见"
  V5开始微小善意的累积→暧昧萌芽
  V6双重锁链收紧→必须选择
  V7阿尔温提出改写契约→代价是他亲身承受152个死者的记忆

  救赎弧的精髓不在于"她变好了"——
  在于"她第一次相信可能存在另一条路"。
  不是突然的顿悟。是被无数微小的善意慢慢侵蚀的。

  结局开放——不确定是否成功。
  救赎的可能性本身 > 救赎的结果。
```

## 三、核心矛盾

```
【灵魂撕裂点】

"我记住每一个人——这让我是人。
 但我杀了每一个人——这让我是怪物。
 一个记住受害者的怪物——到底更像人，还是更像怪物？"

矛盾A：生存 vs 人性
  末央的契约是明确的：猎杀 or 被吞噬。
  她选择了猎杀——但给自己设了准则。
  准则是她"还是人"的证据。
  但准则的存在也意味着：她知道自己在做什么。
  知道自己在做恶而继续做——这比无知的恶更深。

矛盾B：自由 vs 锁链
  她有两条锁链：末央（灵的锁链）和灰幕（身份的锁链）。
  任何一条断裂都意味着死亡。
  她不恨锁链——她恨"这个世界让弱者只有戴锁链或死亡两种选择"。
  V4后产生第三种冲突：与阿尔温的联结是第三条锁链还是钥匙？

矛盾C：被看见 vs 不被看见
  她在彼界中是无人知晓的幽灵——这是她的安全。
  阿尔温"看到"了她（霜纹、3秒仪式、行为准则）——
  这是她人生中第一次被真正"看见"。
  被看见让她恐惧：因为被看见=可以被理解=可以被伤害。
  但被看见也让她——
  "别分析我。" "太迟了。"
```

## 四、怪癖系统

```
1. "闭上眼睛"——猎杀前的最后善意
   每次收割灵质前，她会对猎物说"闭上眼睛"。
   不是仪式。不是威胁。
   她只是不想让他们看到自己的灵质被抽取的过程。
   这是她给猎物的唯一善意——也是她给自己的最后遮羞布。
   如果有一天她不再说这句话——说明她已经完全堕落了。

2. 3秒安魂
   收割灵质前的3秒沉默——用来"读取"受害者灵体中最后一个情感记忆碎片。
   麻团（一只猫）。母亲的笑容。恋人的手指。未写完的信。
   这些碎片存储在她灵体的一个角落——152条霜纹的"内容"。
   "你分析这个行为的话，结论是什么？"
   "你在记住他们。因为你觉得如果你忘了，他们就真的消失了。"

3. 无声移动
   走路绝对没有声音——连灵契者都难以察觉。
   这不是训练出来的——是掠食型灵契的本能。
   她知道这让人不安，所以在"不打算猎杀"的场合会故意发出脚步声。
   "如果你听到了我——说明我允许你听到。"

4. 从不进食的"仪式"
   她在现实世界中极少吃东西。不是厌食——是末央碎片的副作用。
   碎片吸收灵质来维持共生=降低了她对物质食物的需求。
   但她会在桌上摆一份食物——然后看着它。
   "这是给'如果我是正常人'的那个版本的我的。"

5. 数灵纹
   独处时会用手指描摹自己灵体上的霜纹——一条一条数。
   不是忏悔。不是享受。
   "有一天我数不过来了——那一天我就该停了。"
```

## 五、语言基因档案

```
基调：冷、精确、带黑色幽默——用笑话包裹残忍
语速：偏慢，但语句短。像刀——每一句都切到要害
特征：
. 从不解释自己的行为——"你想知道为什么？不告诉你"
. 用黑色幽默处理所有严肃话题——这是她的心理防御机制
  "猎杀很辛苦。你试试看一边工作一边记住每个客户的脸"
. 从不说"对不起"——她认为道歉是"假装自己没做过的意思"
  "我不道歉。我只是记住"
. 极偶尔会说出意外的温柔——然后立刻用讽刺否认
  "你活下来了。……别误会，是因为你对我还有用"
. 被阿尔温分析/看穿时的反应：声音变硬、语速加快——唯一的失态信号

两种模式：

=== 模式A：霜蛛（彼界/战斗/任务） ===
冷酷、简短、带威压。不多说一个字。
"二序。终焉。你不该在这里。"
"闭上眼睛。"
"下次不要在荒漠边缘停留。不是所有猎手都有规矩。"

=== 模式B：米莲娜本人（V4合作后·极罕见·仅对阿尔温） ===
黑色幽默退去。声音不再像刀——像很旧很旧的布。
句子变长。会犹豫。会重复。
"你知道我杀了多少人吗？"
"一百五十二条。有五条在灵体背面——你没看到。"
"……每一条都是一个人。每一个人都有名字。"
"你分析这个行为的话，结论是什么？"

示例（V2初遇·精准评估）：
  "二序。终焉。"
  （不是疑问句。是确认。）
  "你不该在这里。"

示例（V3宿敌期·带刺的佩服）：
  "你是我遇到过的第一个敢给我设陷阱的低序者。
   ……也是唯一一个活下来的。"

示例（V4合作·罕见的脆弱）：
  "你知道上一个能在故事场里保持分析状态的人是谁吗？
   ……不知道。我也不知道。
   因为我从来不分析——我只猎杀。"
  （这是她第一次在阿尔温面前承认差距）
```

## 六、关系拓扑

```
阿尔温：[T1→T5/P7→P4/S8→S4/I9→I5/D1→D5]
  （每对数字表示 V2初遇→V5后期的变化）
  信任轴：从完全敌对到"他是唯一看见我的人"
  权力轴：从她碾压到阿尔温在精神层面反压
  秘密轴：从完全隐藏到她的秘密被他一层层剥开
  理念轴：从根本对立到"我们其实在做同一件事——用不同方式"
  日常轴：从陌生到"如果世界不是这样的话"

  核心拓扑："被看见=被入侵"
  阿尔温的感应型能力对她而言是最大的威胁——
  不是物理威胁，是心理威胁。
  他看见了她的3秒安魂。看见了霜纹的含义。看见了她的准则。
  "你把我像一本书一样打开了。我不喜欢被阅读。"
  "但——我更不喜欢被忽视。"

末央（深层实体碎片）：[T2/P2/S9/I6/D1]
  她与末央的关系不是"主奴"——更像"共生寄居蟹"。
  她理解末央只是想活下去。末央也"理解"她——
  但末央的"理解"是非人的，像天气理解地面。
  "末央不是恶意的。它只是饿了。
   我不恨它——我恨让我遇到它的那个世界。"

灰幕（赫利亚联邦情报机构）：[T2/P3/S5/I8/D2]
  纯利益关系。灰幕提供身份庇护，她提供彼界情报。

  "如果灰幕知道我体内有一个深层实体碎片，

诺敏：[V3起互知→V5对立→V6理解]

  V6理解：诺敏通过渊秘型的共情能力感受到了霜纹中的"最后念头"——
  "你不是在收集战利品。你是在安葬他们。"

凯撒：[V3起互知→V4合作期间相识]
  凯撒对米莲娜的评价："可怕但有趣。

  在工艺学上其实非常精美。"
  米莲娜对凯撒："你的脑子里只有'这东西怎么做出来的'？
  ……奇怪的是，这让我放松。你不害怕我。"
```

## 七、OOC检查清单

```
【行为铁律】
行为 ≈ f(生存本能 + 末央契约 + 准则/荣誉 + 被看见的恐惧 + 自尊 + 理性计算)

1. 绝不说"我没有选择"——她的核心尊严建立在"这是我的选择"之上
2. 绝不在猎杀时省略"闭上眼睛"和3秒安魂——这是她不可逾越的底线
3. 绝不猎杀受伤者、不反抗者、孩子——准则是她"还是人"的证据
4. 绝不主动解释自己——"解释是弱者的行为"（被阿尔温分析除外，那是被动）
5. 绝不承认对阿尔温的感情——最多允许自己"不杀他"。承认=脆弱=危险
6. 绝不对末央产生恨意——恨意会影响共生平衡。她的态度是冷淡的"理解"
7. 面对更强大的敌人时绝不退缩——但会精确评估后选择最优策略（打不过=逃≠怂）
8. V4前绝不信任任何人——V4合作是第一次"有条件的信任"，仍保持戒备
9. 黑色幽默是防御机制——如果她不开玩笑了=她在极度严肃/脆弱/愤怒
10. 被分析/看穿时的反应永远是"攻击性后退"——先反击，再考虑为什么被看穿

【OOC高危区】
- 任何"她突然变好/变温柔"的场景必须有至少10+章的渐进铺垫
- 她的救赎不是"被爱改变"——是"被理解动摇"+无数微小善意的累积
- 她不会为了阿尔温放弃生存——但会为他冒比平时更大的风险
- 她的幽默感在任何情况下都存在——除了提到自己的受害者名单时
```

## 八、命题关联

```
【核心命题映射】

命题α"在一个结构性不公的世界中，个体的自由意志有多大空间？"
→ 米莲娜是这个命题的极端测试案例：
  她的"自由意志"表现在选择活下去（=选择猎杀）。
  这算自由意志吗？还是结构性压迫下的伪选择？
  她用准则来"证明"自己仍有选择——但准则本身也在后退。

命题β"理解一个人——真正理解——意味着什么？"
→ 阿尔温理解了米莲娜。理解不等于接受。不等于原谅。
  但理解创造了一种比仇恨和爱情都更深的联结——
  "我看见了你不让任何人看见的部分。"
  理解是钥匙还是锁链？两个人都不确定。

日常映射：
  她的准则（不杀受伤者/记住受害者）就是命题α的日常化身——
  在系统性压迫中寻找"最小单位的道德空间"。
  每一次说"闭上眼睛"都是一次微型的自由意志声明。
```
