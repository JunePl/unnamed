## 一、基础档案

```
姓名：塔玛拉·塞维林（Tamara Severin）
代号：第三星（六穿越者之一）
年龄：31岁
身份：塞伦共和国地下女王 / 六穿越者中的"蛛网"
前世：商业精英，死于商业暗杀——"上辈子信错了人，这辈子不信任何人"
灵契：缚誓·契约型（第六序·化身）
天秤排名：柱位#89
出身国/活动中心：塞伦共和国
势力范围：塞伦地下经济约50%——贸易、走私、情报、黑市

【人物图鉴验证】
. 人物图鉴L186：塞伦绑定——地下女王，缚誓·契约灵契
. 她与薇拉（镜之眼）是竞争者也是合作者，两人瓜分塞伦的情报市场

【核心定位】
塔玛拉不是"情报贩子"。
她是"规则制定者"。
塞伦地下世界的每一笔交易、每一条走私路线、每一个秘密联盟——
都建立在她的契约之上。
缚誓·契约型的灵契让她的每一个承诺都有灵力约束力：
违约者不是被人追杀——是被灵力本身惩罚。
这意味着：在塔玛拉的领地里，契约比法律更可靠。
这也意味着：她自己也被契约束缚。
"我给别人套上了锁链。但我自己戴着最多的锁链。"

【外貌特征】
. 身高170cm，身材高挑——地下女王的气场不是靠体格
. 深红色卷发，像燃烧的火焰——最醒目的特征
. 琥珀色眼睛，看人时像在计算价值——"在她眼里你是一个人还是一个数字？"
. 嘴角常带似笑非笑的弧度——这个弧度让人不安
. 穿着：华丽但不张扬，暗色系为主，首饰精致但每件都有功能
  （如：耳坠内嵌灵质结晶——紧急通讯器；戒指可展开微型契约书）
. 手指修长，指甲保养完美——但左手中指有一个旧墨水渍洗不掉
  （签契约时的职业病）
. 走路时脚步声精确——不大不小，刚好让人知道她来了
```

## 二、弧线设计

```
【弧线类型】：试炼弧（信念被反复考验）

塔玛拉的信念很简单："我不信任任何人。"
这个信念被反复考验——每次考验都更艰难。

V1·远程铺垫（不直接登场）：
  通过情报交易、雷纳德的回忆、她的地下网络间接呈现。
  读者在V1结束时知道：塞伦有一个"地下女王"，她和雷纳德有过去。

V2·正式登场 + 塞伦弧：
  V2中阿尔温的调查扩展到塞伦。
  必须经过塔玛拉的领地——她是塞伦地下的"关卡"。
  第一次试炼：是否信任阿尔温？（最终答案：有条件的合作）

V3·雷纳德归来——最大试炼：
  她以为雷纳德死了。V3他归来。
  "你知道我最恨什么吗？不是你走了。是你让我……在乎你走了。"
  她必须面对：不信任任何人的信条，被一个人打碎了。

V4-V5·选择参与 + 放手：
  从"这不关我的事"到"也许有些事比我的帝国更重要"。
  V5雷纳德牺牲——她的试炼弧的终极考验。
  她没有崩塌。她选择继续。
  "他教会了我一件事：有些东西不是用来交易的。"
```

## 三、核心矛盾

```
【灵魂撕裂点】

"我控制了一切，却控制不了自己的心。"

矛盾A：控制 vs 情感
  她习惯用"交易"定义所有关系。
  "你想要什么？我想要什么？好，成交。"
  但对雷纳德，她无法用"价值"来衡量。
  这是她唯一的弱点，她恨这个弱点——
  因为弱点在上辈子杀了她一次。

矛盾B：地下 vs 地上
  她在地下世界无所不能——规则由她定义。
  但真正的权力在"地上"——政治、军事、锚点。
  她能跨越这条线吗？
  还是说她永远是"在暗中影响世界，但永远不能走到阳光下"的人？

矛盾C：前世 vs 今生
  前世她死于"信任错误的人"。
  今生她建立了一整套系统来确保"不用信任任何人"。
  但这个系统也把她自己困住了——
  契约可以确保交易公平，但契约无法创造友情、爱情、归属感。
  "我建造了一座完美的牢笼。然后发现自己在里面。"
```

## 四、怪癖系统

```
1. 在任何谈判前倒茶——然后数对方拿起茶杯的时间
   拿起得快=急切，好压价。
   慢慢端起=有底气，需要认真对待。
   不碰茶=不信任/敌意。
   "茶是最好的测谎器。"

2. 左手中指的墨水渍
   签了太多契约留下的。洗不掉。
   她从不试图遮掩——反而当作一种标记。
   "这只手签过的契约比你活过的年份还多。"

3. 收集不同城市的地图
   不是为了情报——她有更高效的方式。
   她只是喜欢看地图。看那些她去过和没去过的地方。
   "上辈子没来得及旅行。这辈子……也没时间。"
   → 地图册是她仅有的"浪漫"——一个从不旅行的旅行家。

4. 从不让别人碰她的头发
   深红色卷发是她最明显的特征。
   有人试图碰她的头发时，她会本能后退——即使是善意的。
   唯一的例外：雷纳德。（他不在时，这个例外不存在。）

5. 微笑时最危险
   她的"真实感情"不通过微笑表达——微笑是武器。
   真正高兴/放松时，她反而面无表情或轻轻皱眉。
   如果她在谈判中笑了——对面应该开始害怕。
```

## 五、语言基因档案

```
基调：精确、商业化、像在签合同
语速：中等，有节奏感——每句话都有标点
特征：
. 每句话都像合同条款——主语、谓语、条件、后果，清清楚楚
. 从不说废话——"时间是唯一不能交易的东西"
. 喜欢用"交易"比喻一切关系
. 从不提高音量——声音变低=你真的惹到她了
. 唯一会"打破格式"的时刻：提到雷纳德时

两种模式：

=== 模式A：地下女王（商业/权力场合） ===
冷静、精确、不可动摇。
"我们来谈谈价码。"
"你有我想要的东西。我有你需要的东西。"
"公平交易。如果你试图欺骗我——
 你会发现，违约的代价比你想象的高得多。"

=== 模式B：塔玛拉本人（极罕见·仅对雷纳德/极信任的人） ===
声音变轻。语速变慢。句子变短。
不再像合同——像一个不知道怎么说话的人在尝试说话。
"……你还活着。"
"你知道我以为……"
"算了。"
"你还欠我一个解释。不急。"

示例（与阿尔温的第一次会面·V2）：
"阿尔温·埃德蒙。灵痕分析师。前执法局。被革职。
 目前在追查一个你称为'D'的东西。
 你来我的地盘——需要什么？"
阿尔温："你怎么知道——"
塔玛拉："这是塞伦。在塞伦，我什么都知道。
 ……好吧。不是什么都知道。
 但关于你的事，我知道得够多了。"
"说吧。你想要什么。我来定价。"

示例（提到雷纳德时——V2·以为他死了）：
"雷纳德？"
（停顿。面无表情。声音变低。）
"他死了。"
"还有别的问题吗？"
（转身走了。没有人敢追问。）
```

```

  "这条消息的源头是塞伦。
   ……算了，你不需要知道源头是谁。
   只需要知道——塞伦的地下，有一个不能招惹的人。"

Ch40附近（雷纳德登场后·回忆闪现）：

  雷纳德："塞伦。贫民窟。有个人把我捡起来。"

  "塞伦的地下女王——塔玛拉·塞维林。
   她的情报网比一般国家的官方情报机构更准确。
   如果你将来需要塞伦方面的帮助——
   先准备好等价交换的筹码。
   她不接受金币。她接受'有价值的信息'。"

Ch91附近（莱奥里克登场后对话）：
```

## 七、关系拓扑

```
雷纳德：[T5/P4/S3/I5/D3（冻结——V1不直接互动）]
  最复杂的关系。
  她培养了他——看出他是穿越者，给了他第一份"工作"。
  他选择了阿尔温——选择了自由而不是她的帝国。
  她恨他走了。但她更恨自己在乎他走了。
  V3重逢时的第一句话不是"你好"或"你怎么可以"——
  是"你变瘦了。"（然后转身走了。）

阿尔温：[V1不直接互动→V2起+30→+50]
  起初视为"可利用的有趣棋子"。
  逐渐认识到阿尔温的价值不在于能力——在于他的"不可被收买"。
  "你是我见过最不像塞伦人的人。这是夸奖。"

薇拉：[-20·竞争+合作]
  两个女王的博弈。
  薇拉控制情报的"来源"，塔玛拉控制情报的"流通"。
  她们互相需要但互相提防。
  "薇拉知道一切。但她不做任何事。
   我做一切。但我不知道所有事。
   我们加在一起——才是完整的塞伦。"

莱奥里克：[+25·有条件合作]
  "太阳？太阳照不到地下。
   但他知道地下有什么——这说明他比大多数人聪明。"

德米特里：[-60]
  德米特里的网络和她的地下帝国有冲突区域。
  "他想把手伸进塞伦的地下。
   我切了三根他的手指。
   他会再长出来——但每次都痛。"
```

## 八、OOC检查清单

```
1. 绝不免费帮忙——一切都有价码（唯一例外：雷纳德，但她永远不会承认）
2. 绝不在雷纳德面前完全放下防备——防备是她最后的盔甲
3. 承认在乎雷纳德的时刻极罕见——声音变化（变轻/变慢）是唯一标志
4. 不会在公开场合流露脆弱——微笑是武器，面无表情才是真实
5. 前世创伤让她对"信任"有极深的恐惧——任何要求她"盲目信任"的场景都需要慢慢铺垫
```
