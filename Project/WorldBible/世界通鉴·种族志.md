## 一、种族总览

奥里斯塔尼大陆是一个多种族共存的世界。旧帝国本身就是多种族帝国——精灵、矮人（深岩民）、灵木族、兽灵族都在帝国治下生存了近两千年。大崩溃后的国家分裂不完全按种族线划分，而是按地理和政治势力划分。因此，大部分国家内部都是多种族混合的——差异在于主体种族比例和文化主导权。

## 二、九大种族详述

### 2.1 人类

| 项目 | 数据 |
|------|------|
| 生物学特征 | 标准人类 |
| 寿命 | 70-90年 |
| 灵契亲和 | 全源流均等——人类是唯一对所有源流无偏好的种族 |
| 帝国时代地位 | 统治种族。旧帝国的皇室、贵族、官僚体系以人类为主体 |
| 崩后地位 | 各国主体或重要组成。虽然总占比仅34%，但在政治权力上仍占据优势地位 |

**文化特征**：
- 人类的核心优势是**适应性**——没有天然的种族偏好，反而使人类能在任何环境中生存
- 人类是唯一一个"哪里都有但哪里都不是绝对主体"的种族
- 帝国时代人类的"统治种族"遗留心态在某些地区仍然存在（尤其是卡萨林和卡拉赞）

**叙事中的核心角色**：阿尔温、莱奥里克、德米特里、安东尼奥、奥古斯都等主要角色均为人类。

---

### 2.2 精灵族

| 项目 | 数据 |
|------|------|
| 生物学特征 | 尖耳、高挑、眼色异常（多为银/金/紫色） |
| 寿命 | 300-500年 |
| 灵契亲和 | 织命·预言型偏高——长寿带来的模式识别能力使精灵在预言类灵契上有天然优势 |
| 帝国时代地位 | 被征服种族。帝历356年（帝国扩张纪早期），人类帝国征服了精灵王国 |
| 崩后地位 | 薇恩涅独立，各国少数族群 |

**文化特征**：
- 精灵族是大陆上**最古老的文明种族**——他们的文明比人类帝国更早出现
- 帝国征服精灵后，大量精灵族被同化或流散。大崩溃后，精灵族在薇恩涅重建了独立国家
- 精灵族的长寿意味着他们的"记忆"就是活历史——月之长老亲眼见证了大崩溃
- 精灵族生育率极低——这是他们人口稀少的主要原因
- "活木"培育技术是精灵族独有的文明遗产

**种族歧视问题**：
- 帝国时代的征服留下了深刻的历史创伤
- 某些人类社会仍然视精灵为"被征服的民族"
- 精灵族内部对"纯血"问题极为敏感——半精灵在薇恩涅被视为"不够纯粹"

**叙事中的核心角色**：月之长老（七序·柱位约#30，薇恩涅最强者，V7 R-8死亡）

---

### 2.3 半精灵

| 项目 | 数据 |
|------|------|
| 生物学特征 | 人类与精灵混血。外貌介于两者之间——耳朵略尖但不如纯精灵明显 |
| 寿命 | 150-250年 |
| 灵契亲和 | 混合亲和——既有人类的全面性，又有精灵族的织命偏好 |
| 帝国时代地位 | 两边不讨好——人类社会视为"精灵的残余"，精灵社会视为"不够纯粹" |
| 崩后地位 | 各国普遍存在，琥珀海岸为主体 |

**文化特征**：
- 半精灵是帝国时代精灵与人类通婚的产物
- 沿海地区（尤其琥珀海岸）通婚率最高，因此半精灵比例也最高
- 半精灵的困境是"双重归属缺失"——在人类社会不完全被接受，在精灵社会被视为混血
- 琥珀海岸是唯一一个半精灵不受歧视的地方——因为在那里他们是主流

**叙事中的核心角色**：薇奥莱塔·灰叶（V4间谍盟友，半精灵·织命·共感型，V4 L1死亡——种族歧视的间接产物）

---

### 2.4 灵木族

| 项目 | 数据 |
|------|------|
| 生物学特征 | 树纹覆盖皮肤（灵纹状的天然纹路），发色偏绿或棕绿，指尖常有木质化特征 |
| 寿命 | 200-400年 |
| 灵契亲和 | 织命·感应型天生——灵木族对"记忆"有天然的感知和存储能力 |
| 帝国时代地位 | 学者/记录员。帝国皇家学院中灵木族学者占重要比例 |
| 崩后地位 | 陶格林/风歌绿洲/各国学术界 |

**文化特征**：
- 灵木族与植物有天然共鸣——不是"变成植物"，而是能感知植物的生长状态和"记忆"
- "活档案馆"能力：灵木族可以将记忆存储在木质物体中，类似天然的信息载体
- 灵木族不分性别——他们使用中性代词，生殖方式类似植物的分蘖
- 两大灵木族文化圈：
  - **森林/学术灵木族**（陶格林/沃伦）：知识导向，与各国学术界深度融合
  - **沙漠灵木族**（风歌绿洲）：适应沙漠环境，发展出独特的"沙漠灵木"培育术，维持绿洲生态

**叙事中的核心角色**：伊万·长忆（V5核心同伴，灵木族·织命·感应型——活档案馆，V5 L1死亡："记住我记住的"）

---

### 2.5 兽灵族

| 项目 | 数据 |
|------|------|
| 生物学特征 | 动物特征混合——耳（兽耳形态）、尾（对应动物的尾巴）、爪（指甲硬化为类爪状）。特征程度因血统浓度而异 |
| 寿命 | 60-80年（略短于人类） |
| 灵契亲和 | 终焉·万兽型偏高——与动物有天然共感能力 |
| 帝国时代地位 | 游牧民/骑兵。帝国征用兽灵族组建骑兵军团，但社会地位低下 |
| 崩后地位 | 科尔曼/索尔海姆/各国军事 |

**文化特征**：
- 兽灵族与动物的"共感"不是魔法——而是一种天然的灵契亲和，类似本能
- 科尔曼的萨满传统就是基于兽灵族的共感能力发展而来的
- 兽灵族的寿命较短，这使得他们的文化更强调"当下"和"行动"，而非精灵族的"沉思"和"等待"
- **分支差异**：
  - **草原兽灵族**（科尔曼）：与马、鹰等草原动物共感，骑术超群
  - **冻原兽灵族**（索尔海姆）：与冻角麋、雪狮等冻原动物建立更深层的灵契，比草原兽灵族更"原始"——他们不骑乘而是"同行"

**叙事中的核心角色**：
- 哈什克·风语（V3战友，科尔曼兽灵族·兽灵·群感型，V3 L1死亡）
- 诺敏（科尔曼萨满学徒——人类，但在兽灵族文化中成长，Line C超长线核心角色）

### 2.6 半兽灵

| 项目 | 数据 |
|------|------|
| 生物学特征 | 人类与兽灵混血。兽类特征减弱但仍可辨识——略尖的犬齿、隐约的斑纹、增强的嗅觉/听觉 |
| 寿命 | 65-85年 |
| 灵契亲和 | 混合亲和——动物共感能力弱于纯兽灵族，但比人类强 |
| 帝国时代地位 | 边缘群体。既不被人类社会完全接纳，也不完全融入兽灵族部落 |
| 崩后地位 | 各国少数，科尔曼/艾瑞克/诺德占比较高 |

**文化特征**：
- 半兽灵在军事领域有独特优势——兼具人类的纪律性和兽灵族的战斗直觉
- 艾瑞克帝国的军队中半兽灵占重要比例——帝国的"人人皆兵"制度为半兽灵提供了平等的上升通道
- 诺德海岸的海盗中半兽灵比例较高——海盗世界不看血统看能力

**叙事中的核心角色**：芬恩·晨歌（V6组织成员，半兽灵·辉耀·旋律型，V6 L1死亡——"他一直在唱歌"）

### 2.7 矮人族（深岩民）

| 项目 | 数据 |
|------|------|
| 生物学特征 | 矮壮（平均身高约140-150cm），须浓（男女皆有，但程度不同），夜视力极强，骨骼密度高于人类 |
| 寿命 | 150-200年 |
| 灵契亲和 | 铸世系天生——矮人族对金属/矿物/锻造有天然的灵契亲和 |
| 帝国时代地位 | 矿工/锻造师。帝国强制定居化了原本生活在地下的矮人族，但也利用他们的锻造天赋 |
| 崩后地位 | 岩砷独立/各国矿区工业区 |

**文化特征**：
- 矮人族的文化核心是"锻造即信仰"——他们不崇拜神，崇拜"完美的工艺"
- 岩砷共和国的工匠议会制是矮人族独有的政治形态——最好的锻造师就是最好的领导者
- 矮人族的地下城市文明与地表文明有显著差异——他们的建筑、艺术、音乐都在地下环境中发展出独特形态
- 矮人族生育率低于人类但高于精灵族
- **"永锤"布罗克**与联邦的**格里高尔**形成"铸世系双雄"——一个锻武器，一个造魔偶

**叙事钩子**：V4-V5中可能需要矮人族的锻造技术来对抗德米特里的傀儡灵网络。

### 2.8 深渊民（火山之子）

| 项目 | 数据 |
|------|------|
| 生物学特征 | 暗红色灵纹覆盖皮肤（天然灵纹，非纹身），体温高于人类约3-5°C，眼瞳偏红/橙色 |
| 寿命 | 100-150年 |
| 灵契亲和 | 渊秘/铸世偏高——与火山灵质有天然联系 |
| 帝国时代地位 | 偏远岛民。帝国晚期才将深渊列岛纳入版图（名义上） |
| 崩后地位 | 深渊列岛封闭/塔肯尼少数 |

**文化特征**：
- 深渊民的文明与火山密切相关——他们的城市建在火山内部或岩浆通道之上
- 暗红色灵纹是深渊民的种族标记——灵纹的复杂程度反映灵契者的力量等级
- 深渊列岛的封闭性使得外界对深渊民了解极少——他们偶尔出现在塞伦港口进行贸易时，总是引起注目
- 塔肯尼联盟中的深渊民保留了较多原始传统——火山列岛与深渊列岛曾属同一文明圈，分裂后走上不同道路
- 深渊民的"深海崇拜"暗合彼界深层的某些秘密——这是V5-V7的悬念储备

### 2.9 石心族（岩语者）

| 项目 | 数据 |
|------|------|
| 生物学特征 | 硅基半生物——不是纯粹的石头，而是灵质与矿物融合的生命体。外貌类似粗糙的人形岩石雕像，但有灵活的"石肌"组织 |
| 寿命 | 500年以上（已知最长者超过800年） |
| 灵契亲和 | 铸世·构筑型天生——石心族本身就是"灵质与物质融合"的活体验证 |
| 帝国时代地位 | 稀有种族。帝国时期仅有学术记录，未被系统纳入社会体系 |
| 崩后地位 | 灰脊山脉深处隐居/极少数下山 |

**文化特征**：
- 石心族的思维速度远慢于人类——一个"快速决定"可能需要他们数天。但他们的思维极度精确，几乎不犯逻辑错误
- 石心族的"寿命"概念与碳基生物不同——他们可能进入数十年的"沉眠期"（类似休眠），在矿脉中汲取灵质
- 石心族不需要食物——他们从矿物中吸收灵质和养分
- 石心族的存在挑战了"什么是生命"的定义——与心域灵/傀儡灵的讨论形成主题呼应
- 目前已知的石心族个体不超过几千人，绝大部分隐居在灰脊山脉深处从不与外界接触
- 岩砷共和国是唯一与石心族建立了稳定交流的国家——少数石心族在岩砷的深矿区工作（人类和矮人无法承受的极端环境）
