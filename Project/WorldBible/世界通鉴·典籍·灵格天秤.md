### 序言：型态不等于个体

> *"十源分类法和六型态框架是这个世纪最重要的学术成就。但任何优秀的框架都有一个致命的副作用——让人误以为框架就是真相本身。"*
> *——科瓦廖夫部长原稿序言（帝历1856年，当时为九源分类法）*

当我们说一个灵是"圣序·毁灭型"时，我们只说清楚了两件事：它操控的是**秩序法则**（源流），它的法则表达方式是**直接破坏**（型态）。

但这**远远不够**。

两个同为圣序·毁灭型的灵，战斗方式可能天差地别。一个以白焰焚烧一切"不洁"，火焰自动绕开纯洁之物；另一个则制造灼热光柱从天而降精确打击，如同天罚之矛。它们的源流相同、型态相同，但它们是**完全不同的个体**。

本补遗卷的目的，就是解释这第三层差异从何而来。

> **【重要说明】** 关于十源流与六型态的完整定义、能力特征与代价征兆，请查阅《灵格特性全典》（灵学典籍·卷三）。本卷不重复型态定义本身，仅探讨**型态内部的个体差异机制**。

### 第一章：灵的三层结构

经过数百年研究，学界将灵的构成分为三个层次：

```
┌─────────────────────────────────────┐
│       第三层：契约烙印（动态）      │  ← 契约者的经历、情感、意志
│  ┌─────────────────────────────┐    │
│  │  第二层：型态倾向（半固定） │    │  ← 六型态之一（毁灭/支配/共鸣/织构/感应/变异）
│  │  ┌─────────────────────┐   │    │
│  │  │ 第一层：源流本质    │   │    │  ← 五道裂隙·十源流之一（不可变）
│  │  │    （固定）         │   │    │
│  │  └─────────────────────┘   │    │
│  └─────────────────────────────┘    │
└─────────────────────────────────────┘
```

**第一层：源流本质（不可变）**

灵最根本的属性。决定它属于五道裂隙中哪一侧的哪个源流——铸世或终焉、圣序或蜕变、织命或心域、誓约或辉耀、幻象或渊秘。源流本质从灵诞生之日起注定，**永不改变**。一个圣序灵永远是圣序灵。

> *已知唯一的"例外"是传说中的奥里斯事件——试图让灵跨越裂隙本身。但那不是改变源流，是毁灭一切分界线。详见五道裂隙概论。*

**第二层：型态倾向（半固定）**

在源流之上的能力表达方向——六种型态（毁灭/支配/共鸣/织构/感应/变异）决定了灵"如何使用"它的源流法则。型态在灵诞生时就已确定，但可以通过与契约者的长期互动而**缓慢偏移**，极端情况下甚至发生型态变异。

> *【研究员注】型态偏移与型态变异不同。偏移是同一型态框架内的微调（如毁灭型从"范围爆发"偏向"精准打击"）；变异是跳出型态框架本身（详见本卷第四章）。*

**第三层：契约烙印（动态变化）**

契约者在灵身上留下的印记。当人与灵签订契约后，双方相互影响——灵被契约者的经历、情感、意志所塑造，形成独特的"契约烙印"。这一层**高度动态**，随契约深化不断变化，直接影响灵的进化路径和能力表现方式。

> *契约者死亡后，灵回到彼界时仍携带这份烙印。这就是为什么在彼界偶尔能遇到"带着人类记忆碎片"的灵——那是前任契约者留下的痕迹。*

**核心结论**：源流决定"操控什么法则"，型态决定"怎么操控"，契约烙印决定"以什么风格操控"。三层叠加，造就了灵的完全独特性。

### 第二章：型态内的个体差异——为什么同一格子里没有两个相同的灵

六型态框架将灵的能力表达分为六个方向，但每个方向内部的变化空间仍然巨大。

#### 2.1 个体差异的四个来源

| 差异来源 | 机制 | 示例 |
|----------|------|------|
| **法则偏好** | 同一源流法则有多种"子法则"，灵天生偏好其中某些 | 圣序·毁灭型：A灵偏好"焚烧"子法则，B灵偏好"光击"子法则——同样是毁灭，一个用火，一个用光 |
| **媒介亲和** | 灵表达法则时偏好的物理媒介不同 | 终焉·织构型：A灵通过"黑暗"编织领域，B灵通过"寂静"编织领域——同样是终焉织构，场景观感截然不同 |
| **规模倾向** | 同型态的灵天生偏向精密或广域 | 铸世·织构型：A灵擅长微观精密铸造（一颗完美螺丝），B灵擅长宏观快速铸造（一面粗糙但巨大的城墙） |
| **性格染色** | 灵自身的"性格"影响法则表达的风格 | 渊秘·感应型：A灵沉默寡言，信息以碎片意象呈现；B灵多话絮叨，信息以完整句子灌入契约者脑中 |

#### 2.2 战斗差异实例

> **实例：两名五序·圣序·毁灭型灵使对决**
>
> 灵使A——灵名「审判之矛」：法则偏好=光击；媒介=凝聚光柱；规模=精准单体。
> 战斗方式：从天而降的白色光柱，直径仅十厘米，精确命中目标弱点。一击一杀型。
>
> 灵使B——灵名「荒火」：法则偏好=焚烧；媒介=高温火焰；规模=范围广域。
> 战斗方式：整个战场化为火海，焦土政策，不分精确——烧光一切不洁之物。
>
> 同源流、同型态、同位阶。但A是狙击手，B是轰炸机。

> **实例：两名四序·织命·感应型灵使的信息输出**
>
> 灵使C——灵的性格温和、有条理。预视呈现为清晰的"分叉路径图"，概率以数字标注。
> 灵使D——灵的性格暴烈、碎片化。预视呈现为高速闪过的画面碎片，需要契约者自行拼凑。
>
> 同样的能力，使用体验天差地别。C更适合战术规划，D更适合临场直觉。

### 第三章：契约烙印——人与灵的相互塑造

#### 3.1 三阶段烙印演化

**阶段一：初始烙印（一序至二序）**

灵根据契约者的灵魂特质选择显现哪些能力。同一个灵给不同契约者，可能走向完全不同的能力分支。

> *例：一个圣序·毁灭型灵，遇到暴躁的契约者发展出"爆炸火焰"（范围瞬爆），遇到冷静的契约者发展出"精准灼烧"（单点高温）。灵的型态没变——变的是个体表达。*

**阶段二：深度烙印（三序至五序）**

契约者的重大人生经历在灵身上留下永久烙印，催生**新的能力分支**——这些分支在灵格特性全典中找不到对应条目，因为它们是独一无二的"契约产物"。

> *例：一个渊秘·感应型灵使，亲眼目睹家人被淹死。悲剧经历在灵身上烙下"窒息"印记——灵的感应能力从"秘密窃取"偏移出一个新分支"溺亡预感"：能感知目标是否正在被某种力量"淹没"（物理或精神层面）。这个能力不在任何手册里。*

**阶段三：融合边界（六序以上）**

人与灵的边界开始模糊。契约烙印可能引发**型态变异**——灵的型态倾向发生根本性改变，甚至跨越到变异型领域。

> *详见第四章。*

#### 3.2 反向烙印——灵对人的塑造

这是大众更熟悉的方向：灵的存在改变契约者。

| 影响层面 | 表现 | 不可逆程度 |
|----------|------|-----------|
| **身体变化** | 代价征兆（金属化/灰化/兽化/等） | 高阶后多不可逆 |
| **性格偏移** | 向灵的"性格"方向缓慢移动 | 缓慢但累积 |
| **感知重塑** | 看世界的方式发生变化（铸世使者看到结构、终焉使者看到终结信号） | 深度影响后不可逆 |
| **本能植入** | 灵的本能反应渗入契约者（如蜕变使者在危险中自动兽化） | 高阶后成为"第二天性" |

### 第四章：型态变异——当框架容不下个体

> *"变异型不是第七种型态。它是前五种型态的边界溶解后产生的东西。"*
> ——超凡研究所首席研究员·分类委员会内部备忘

#### 4.1 变异条件

| 条件 | 说明 |
|------|------|
| 契约深度 ≥ 六序 | 人与灵的边界必须足够模糊 |
| 极端存在性危机 | 信仰崩塌、挚爱背叛、濒死重生等 |
| 契约者意志强度 | 必须强到能"引导"变异方向而非被变异吞噬 |
| 灵自身的变异潜质 | 并非所有灵都有——这取决于灵在彼界诞生时的"完整度" |

#### 4.2 变异类型

| 类型 | 定义 | 发生率 |
|------|------|--------|
| **型态进化** | 同型态内升级为更强版本（如毁灭→概念级毁灭） | 较常见 |
| **型态融合** | 两种型态特征合并为新复合型态 | 稀少 |
| **跨源流染色** | 灵受其他源流影响产生"染色"效果 | 极罕见 |
| **全新灵种** | 变异达到临界点，灵不再属于任何已知分类 | 传说级 |

> **关于变异型的完整档案（已知9种变异案例）**，请查阅灵学典籍·卷三·变异型特别记录。本卷不重复其内容。

### 第五章：源流相性——五道裂隙下的对立与共鸣

> *"旧的相性理论认为相性关系是'克制'。五道裂隙理论告诉我们——不是克制，是**对立面的天然张力**。"*
> ——超凡研究所·裂隙理论组·内部研讨会发言

#### 5.1 对立轴相性（五道裂隙对立面）

| 裂隙 | 对立双方 | 相性关系 | 战术含义 |
|------|----------|----------|----------|
| ①创灭 | 铸世 ←→ 终焉 | 创造vs终结：天然拮抗。铸世的创造可被终焉消解，终焉的虚无可被铸世填补 | 攻守互克，持久战倾向 |
| ②序变 | 圣序 ←→ 蜕变 | 秩序vs混变：圣序的净化直接压制蜕变的失控进化，但蜕变的适应性可逐渐绕过圣序的固定规则 | 圣序初期占优→蜕变后期反超 |
| ③因心 | 织命 ←→ 心域 | 因果vs意志：织命操控客观因果链，心域改写主观认知。织命看穿心域的幻觉，心域扰乱织命的精密计算 | 信息战层面的终极对决 |
| ④律越 | 誓约 ←→ 辉耀 | 规则vs突破：誓约的约束力越强，辉耀的突破欲望越大。誓约能封锁辉耀的超限能力，辉耀能打碎誓约的规则框架 | 力量vs规则的哲学对抗 |
| ⑤显隐 | 幻象 ←→ 渊秘 | 映现vs深隐：幻象制造表象，渊秘揭露本质。幻象看不穿渊秘隐藏的真实，渊秘看不破幻象伪造的表层 | 情报战层面的终极博弈 |

#### 5.2 非对立轴相性

对立轴之外的源流组合没有天然的强克制关系，但存在**战术级优劣**：

| 组合 | 关系 | 说明 |
|------|------|------|
| 铸世 + 蜕变 | 互利 | 铸世造装备增强兽化形态；蜕变提供天然材料 |
| 圣序 + 誓约 | 协同 | 圣火为见证、誓约为枷锁——"神圣契约"极难违抗 |
| 终焉 + 渊秘 | 暗面共鸣 | 终结与深渊——黑暗系双源流联手压制感极强 |
| 心域 vs 所有 | 特殊 | 精神攻击对所有源流有效，但被"专注状态"反制——心域是最强辅助/最脆弱正面战斗者 |

#### 5.3 型态克制——真正决定胜负的层面

> 源流相性是"方向"，型态克制是"技术"。实战中后者往往更重要。

| 克制方 | 被克制方 | 克制原理 |
|--------|----------|----------|
| 感应型（任何源流） | 支配型 | 看穿控制，提前规避 |
| 毁灭型 | 织构型 | 破坏快于建造 |
| 共鸣型 | 毁灭型 | 硬抗伤害，消耗对方 |
| 织构型 | 感应型 | 领域封锁信息传递 |
| 支配型 | 共鸣型 | 直接剥夺强化/融合 |

> *【研究员注】以上为一般规律。在高阶战斗（六序以上）中，个体差异和契约烙印的影响往往大于型态克制——框架是参考，不是定论。*

### 第六章：战斗变数总论

决定一场灵使战斗结果的**八大变数**，按影响权重排列：

| # | 变数 | 权重 | 说明 |
|---|------|------|------|
| 1 | 位阶差 | ★★★★★ | 两序以上的位阶差几乎无法逆转 |
| 2 | 灵格个体特性 | ★★★★ | 同源流·同型态但个体差异可造成质变 |
| 3 | 契约烙印 | ★★★★ | 独特能力分支可能是致胜关键 |
| 4 | 源流相性 | ★★★ | 对立轴克制约15-25%战力加减 |
| 5 | 型态克制 | ★★★ | 技术层面的优劣，可被经验弥补 |
| 6 | 战斗经验与技巧 | ★★★ | 超凡能力不替代战术智慧 |
| 7 | 环境因素 | ★★ | 草原利蜕变、城市利铸世、黑暗利终焉 |
| 8 | 灵的"状态" | ★★ | 灵有活跃度波动，受时间/环境/情绪影响 |

> **结论**：两个同位阶、同源流、同型态的灵使对战，结果可能完全不同——因为**世界上没有两个相同的灵**。每个灵都是在彼界独立诞生的个体，带着自己的偏好、性格和潜力。分类体系帮助我们理解灵，但**永远不要把框架当成灵本身**。

### 天秤规则补充说明

> **灵契天秤的双轨制**：
> 
> 血月之夜后，学者们发现天秤实际上记录两种信息：
> 
> **1. 契约者排名（主榜）**
> 衡量的是契约者的"命运重量"——影响力、行为涟漪、潜力。
> 不完全等于战斗力。
> 
> **2. 灵格强度排名（副榜）**
> 衡量的是灵本身的强度和独特性。
> 变异灵、古老灵、特殊灵会有更高排名。
> 
> **重要发现**：契约者排名和灵格排名并不总是一致。
> 
> 一个拥有强大灵的契约者，可能因为缺乏行动而排名较低。
> 一个灵格普通的契约者，可能因为重大行为而排名靠前。
> 
> **这就是为什么"灵能越强不一定越厉害"——天秤衡量的是综合影响力，而非单纯战力。**

### 【关键设定说明】

> **为什么契约者排名和灵格排名不一致？**
> 
> 以"原初之焰"为例：它是灵格排名第一的灵，但它的契约者已经死亡（或消失），所以没有出现在契约者榜上。
> 
> 以"永夜"为例：TA的契约者排名第一，但其灵的具体信息无法探知，所以灵格排名只能标记为"第六"（推测）。
> 
> **这也解释了"灵能越强不一定越厉害"**：
> 
> 一个强大的灵如果遇到一个平庸的契约者，可能发挥不出全部实力。
> 一个普通的灵如果遇到一个天才的契约者，可能创造出惊人的战绩。
> 
> **契约是"双人舞"，缺一不可。**
> 
> **另一个重要因素：灵格强度≠战斗适配性**
> 
> 例如："万象书库"灵格排名第12，但它的契约者塞西莉亚在战斗力上并不突出。因为这个灵的特长是"知识存储"而非"战斗"。
> 
> 相反，"夜之遗忘"灵格排名仅第19，但它的契约者伊格纳特被认为是大陆最可怕的刺客。因为这个灵的能力与刺杀完美契合。
>
>**强大的灵是否可能有一个弱小，更低阶的的主人**
>
>答案是可能，高阶而强大的灵通常都有着自己的思维和追求，因此可能通过主动寻找并签订契约的方式，离开彼界进入现世达成某些目标，但即便灵收敛自己并尽全力不去侵蚀，低阶的人类也通常无法长时间持有高与自己两阶以上的灵，否则必然会被侵蚀失去理智或是爆体而亡
>例一：战场之上，一名仅仅五阶的暗杀者进行自杀式死亡任务妄图斩杀米拉尔圣国大祭司，名为【弑杀】排名前三十灵格的【终焉源流·毁灭型】【能力为：看破破绽后，可一击致命】，主动签约为前任主人报仇，暗杀者主动放弃自己的躯体只求一击，【弑杀】只差一击就将圣国大祭司重伤垂死。心满意足后回到彼界。
