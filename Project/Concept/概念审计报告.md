# 审计方法

对Project全部文件执行概念锚定者三重检验：
1. **五感测试**：这个概念发生时，角色的五感会体验到什么？
2. **小学生测试**：能否用10岁小孩能听懂的话解释？
3. **场景演示测试**：用50字场景让读者明白，不能用解释，只能用展示。

判断标准：如果让演员表演这个概念，他们知道该怎么演吗？不知道就不能用。

## 漏洞 #C-01：D模式（D Mode）——【V2.0重大修正】

### 当前状态

"D模式"在项目中出现136次，分布于人设集、案件集、章纲、节奏规划、正文等几乎所有文件。但至今没有任何文件回答一个基本问题：**"D模式"这三个字是什么意思？**

### 三重检验结果

| 检验 | 结果 | 原因 |
|------|------|------|
| 五感测试 | **失败** | "D模式"发生时读者看到/听到/感受到什么？没有定义。人设集只说"四特征"，但那是调查结论的行为模式归纳，不是概念本身的五感体验 |
| 小学生测试 | **失败** | "D模式是什么？""是一个杀人模式。""为什么叫D？""因为......没有解释。" D为什么是D？是谁命名的？为什么不叫X模式或者"影子模式"？ |
| 场景演示测试 | **失败** | 无法用50字场景让读者理解"D模式"。读者看到的只是"阿尔温说这是D模式"——但D模式到底长什么样？ |

### 具体问题清单

1. **命名来源不明**：阿尔温三年前发现了案件的共性，命名为"D模式"。为什么是"D"？D代表什么？
2. **概念的物理表现不明**：D模式发生时，现场到底是什么样的？
3. **概念触发的体验不明**：阿尔温"识别出D模式"的那一刻，身体发生了什么？
4. **读者认知路径断裂**：读者在Ch9才"正式"了解D模式，但此前Ch1-Ch4的"极轻触"到底轻触了什么？

## 漏洞 #C-02：灵痕 / 灵痕分析——【V2.0灵值体系修正】

### 当前状态

"灵痕"在项目中出现200+次，是阿尔温的核心能力。世界通鉴·系统篇中有灵痕检测器和灵痕分析单镜的技术定义。但正文中灵痕的实际表现有多处模糊。

### 三重检验结果

| 检验 | 结果 | 原因 |
|------|------|------|
| 五感测试 | **部分通过** | 世界通鉴定义了"灵痕颜色（对应源流类型）、密度（对应位阶）、衰减曲线（对应时间）"——有视觉。但缺少触觉、听觉、嗅觉的描述。正文中也只写了颜色（"灿黑与星蓝"），其他感官为空 |
| 小学生测试 | **通过** | "灵痕"="灵留下的脚印"。人走过会留脚印，灵用过力气也会留痕迹。阿尔温的工作就是读这些脚印 |
| 场景演示测试 | **部分通过** | 正文L22-23有"灿黑与斑驳的星蓝交织"的视觉描写。但"灵值113灵点"是什么意思？正文L26的数字突然出现，没有解释 |

### 灵痕概念登记卡

```

### 灵值/灵点修复方案——【V2.0重大修正：质数级递增体系】

> **V1.0方案（线性递增0-10/10-50/50-100...）已被否决。** 理由：灵值应该是质数级/数量级递增的，反映灵力本身的指数增长特性。

```
【概念名称】：灵值（Spirit Level Reading）
【一句话定义】（15字内）：单片眼镜测出来的灵力强度数字

【为什么有这个数字】：
沃伦理工学院开发的标准化测量单位。就像温度计测温度一样，灵痕分析单镜
可以把肉眼看到的灵痕浓度转化成数字。

【数量级递增体系·V2.0】（需写入世界通鉴·系统篇）：

灵值不是线性增长的。它和灵契的位阶一样，是数量级跳跃的。
就像地震的里氏震级——4级和5级之间不是差了1，而是差了10倍能量。

  ┌──────────────────────────────────────────────────────┐
  │ 序位    │ 灵值范围        │ 含义                      │
  ├─────────┼─────────────────┼──────────────────────────┤
  │ 非灵契者│ 0               │ 无灵力（正常人基线）      │
  │ 残留/微弱│ 1-10           │ 环境残留/极微弱波动       │
  │ 一序    │ 1-100           │ 初感阶段，刚能感知灵      │
  │ 二序    │ ~101-1,000      │ 共鸣建立，灵力初步运用    │
  │ 三序    │ ~1,001-10,000   │ 灵显现，能力成型          │
  │ 四序    │ ~10,001-100,000 │ 临渊，深度运用灵力        │
  │ 五序    │ ~10万-100万     │ 烙印，灵力质变            │
  │ 六序    │ ~100万-1000万   │ 破界，可触及彼界          │
  │ 七序+   │ 1000万+         │ 单镜无法测量，直接过载    │
  └──────────────────────────────────────────────────────┘

  核心规则：
  · 每个序位的灵值上限大约是下一个序位的下限
  · 同一序位内的灵值差异反映个体差异和状态波动
  · 灵值不是固定的——同一个人在战斗态和平静态时灵值不同
  · 单片眼镜测量的是"环境灵力残留的灵值"，不是测人本身

  关键：读者不需要记住这些数字。
  作者需要记住的是比较关系——"大一个数量级=碾压性差距"。

【正文中已出现数值的V2.0解读】：

  马蒂亚斯（非灵契者）身上有113灵点：
  → 113处于一序范围内（1-100的上端甚至刚溢出）
  → 对一个非灵契者来说，应该是0
  → 113不高——一个一序灵契者日常残留的水平
  → 但对"不是灵契者"的马蒂亚斯来说——这就像在滴酒不沾的人血液里发现了酒精
  → 说明有一个低阶灵契者最近在他身上使用了灵力

  老彼得（缚誓源流）身上有73灵点：
  → 73处于一序范围（1-100）
  → 如果老彼得是缚誓系灵契者，这个数值偏低
  → 但"老彼得"实际上是替身——真正的老彼得不是灵契者
  → 73的灵值来自替身自己的灵力（伪装用的缚誓系能力的残留）
  → 伏笔：为什么一个非灵契者的画廊经理身上有73灵点？

  艾伦身上有1灵点：
  → 1灵点——比0多一点点
  → 正常人应该是0。1说明有极微弱的灵力残留

## 绝对禁止在正文中出现的表述

| 禁止 | 原因 | 替代 |
|------|------|------|
| "灵力频率" | 伪科学 | "灵力脉动" / 去掉 |
| "频率签名" | 伪科学 | "灵痕纹路" / "灵痕笔迹" |
| "D模式激活" | 空洞游戏术语 | 用场景展示阿尔温的识别过程 |
| "同步率XX%" | 游戏面板 | 展示角色行为同步的具体画面 |
| "灵能" | 术语不统一 | "灵力" |
| "灵魔器" | 旧术语 | "灵力装置" |
| "灵魔晶" | 旧术语 | "灵质结晶"（口语"结晶"） |
| "灵石" | 修仙味太重 | "灵质结晶"或口语"结晶" |
| "X阶Y者"（如"3阶唤灵者"） | 旧称 | "三序·显现" |

## 正文中概念引入的黄金规则

1. **首次出现时不解释**——先让读者看到效果，3-5页后再自然解释
2. **通过对比引入数字**——"113灵点"单独出现没意义，但"非灵契者应该是0"之后说"113"，读者立刻理解
3. **用角色的反应代替作者的注释**——不写"这是异常的"，写"阿尔温皱了下眉"
4. **概念密度控制**——每2000字最多引入1个新概念，引入后至少500字内不再引入下一个
