### C-01 灵痕

```

【体系位置】：灵契体系·基础概念（阿尔温的职业工具）

  最弱：普通人无法感知（肉眼不可见）

## Ch5：装备概念层（阿尔温的非法武装）

### C-04b 灵质枪

```
【概念名称】：灵质枪（Spirit Gun）
【一句话定义】：将使用者的灵能压缩为高密度灵质弹丸射出的近战武器——一阶灵使的力量倍增器
【体系位置】：装备体系·特级管制武器（一阶可用，可跨阶威胁二阶）
【首次出现章节】：Ch5（阿尔温进入画廊地下室时持枪）
【上下文解释方式】：■叙述说明 □角色对话 □系统提示 □旁白补充
  → Ch5：阿尔温进入未知暗道时拔枪——通过动作描写自然展示外观（灵能导流管、保险拨片）
  → Ch6：艾伦变异时使用——展示射击效果（短促玻璃碎裂声、枪口光弧、对变异体的有限杀伤）
  → 后续章节逐步展示：灵能消耗、弹药焦虑、高阶面前的无力感

【核心机制】：
  动力来源：消耗使用者自身灵能（非外部弹药）
  每发消耗：10灵能点
  弹容量：取决于使用者灵能总量（阿尔温一阶顶峰100点 = 满载10发）
  充能方式：一阶使用者无灵、灵能不可自然恢复→必须通过吸收灵晶补充
  使用门槛：一阶以上（需要灵能接入才能击发，普通人无法使用）
  保险机制：物理拨片 + 灵能接入双重保险（防止普通人误用/偷用）

【杀伤力刻度】：
  对普通人：致命
  对一序：有效（可击伤/击杀）
  对二序：可伤（灵质弹丸能突破低阶灵力防护）
  对三序以上：基本无效（灵力防护可轻松抵消弹丸）
  对七阶力场：毫无意义（苏暮雪的灵质枪在德七阶力场中射击效率下降三成→Ch90+终幕）

【管制原因】：
  灵质枪让低阶灵使拥有跨阶威胁力——一个一阶拿着枪可以伤到二阶。
  这种"等级均衡器"如果泛滥，灵使社会的阶级秩序会崩塌。
  因此被列为特级管制武器——仅执法局在编人员、军方特许、高层护卫可合法配备。
  非法持有 = 重罪。

【外观】：
  尺寸：比普通手枪略短，单手可持，可藏于大衣腰后
  材质：深灰色金属，枪身有磨砂质感
  灵能导流管：枪身侧面一条细长透明凹槽——充能时使用者的灵能以微光形式在管中流动
    满载时微光明亮；弹药不足时闪烁减弱；将尽时熄灭
  枪柄底部：灵能接入点，握枪时掌心贴合此处，灵能自动导入
  枪口内壁：刻有极细的灵纹阵列（压缩灵能用），肉眼几乎看不见
  射击表现：无火药枪声——发出一声短促的、类似玻璃碎裂的轻响，枪口闪过一道淡色光弧

【阿尔温的枪】：
  来源：D案件中殉职同事的遗物。阿尔温离开执法局时自己的配枪已被缴回，这把是他私自保留的
  状态：枪身有磨损痕迹（使用多年），灵能导流管略微发黄（老化）
  携带方式：藏在大衣腰后，用暗扣固定（非法持有，不能被人看到）
  情感重量：同事牺牲的唯一遗物——阿尔温愿意为了保护身边的人承担违法风险
  灵能补给：每次出任务前用灵晶充能。100灵能=10发，必须省着用

【与已有概念关联】：灵能（动力源）→灵质枪（应用）；灵晶（充能介质）→灵质枪（续航）；管制体系→阿尔温非法持有（角色张力）
【必要性检验】：■不能删。阿尔温前中期核心战斗手段+非法持有=角色深度+D案件情感锚点。
```

```

【一句话定义】：一种人造物质，可强制触发觉醒——但成功率只有30%，失败就变成怪物
