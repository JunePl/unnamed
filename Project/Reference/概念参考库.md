## 常见伪概念改写对照表

| 伪概念 | 问题 | 改写方案 | 改写理由 |
|--------|------|----------|----------|
| "A级能力者" | 空洞等级 | "能单独猎杀一条龙的人" | 有画面感，有重量感 |
| "S级任务" | 空洞等级 | "需要至少五人组队才能接的委托" 或 "十个人去，能活着回来三个就算成功" | 让读者感受危险 |
| "D模式" | 空洞代号 | "猎杀状态：瞳孔变金，心跳降到每分钟20下，失去恐惧和痛觉" | 具体的身体变化 |
| "灵力频率" | 伪科学 | "灵力像心跳一样有节奏，节奏越快，释放的力量越猛烈，但也越难控制" | 用心跳类比，可感知 |
| "同步率83%" | 伪精确 | "他们的呼吸开始一致，甚至眨眼的节奏都同步了" | 可观察的具体表现 |
| "意识同步" | 抽象 | "她害怕的时候，他的胃会像被人攥紧；她开心的时候，他会莫名其妙地想笑" | 情感具象化 |
| "战斗力提升2.4倍" | 游戏数值 | "他一拳打碎了自己以前需要十拳才能打碎的石头" | 对比展示 |
| "进入第三意识层" | 抽象境界 | "他第一次听到了树在说话，每一棵都在低语着几百年前的事" | 具体的新能力 |
| "精神污染" | 抽象 | "那个声音会在你睡着的时候钻进来，让你做关于它的梦，一开始只是梦，后来你醒着也能听到它在叫你的名字" | 恐怖具象化 |
| "契约代价" | 抽象 | "每使用一次这个能力，他的头发就会白一缕，等头发全白了，命也就没了" | 可见的倒计时 |
| "时间流速降低" | 伪科学 | "雨滴悬停在空中，他甚至能看清每颗水珠里倒映的自己的脸" | 诗意画面 |
| "觉醒" | 玄虚 | "那天晚上他发现自己能看到别人头顶的字——是他们今天心里想得最多的那句话" | 具体能力展示 |

## 优秀能力系统参考

| 来源 | 能力系统 | 为什么优秀 |
|------|----------|-----------|
| 《咒术回战》 | 术式 | 每个术式都有明确规则、视觉效果、使用限制 |
| 《JOJO的奇妙冒险》 | 替身 | 具体形态+明确能力范围+"5秒"这样的简单限制 |
| 《鬼灭之刃》 | 呼吸法 | 水之呼吸像水流，火之呼吸像火焰——直觉上成立 |
| 《钢之炼金术师》 | 炼金术 | "等价交换"一个原则解释一切，简单而深刻 |
| 《猎人×猎人》 | 念能力 | 六系分类清晰，"制约与誓约"让代价有重量 |
| 《物语系列》 | 怪异 | 每个怪异都基于日本民俗/心理学，有文化根基 |
| 《夏目友人帐》 | 妖怪与名字 | "名字即力量"——规则简单，情感深厚 |
| 《进击的巨人》 | 巨人化 | 咬手指、有时间限制、用多了会变形——代价具体可见 |

## 优秀等级系统参考（不要用SABC）

| 来源 | 等级系统 | 为什么优秀 |
|------|----------|-----------|
| 《猎人×猎人》 | 猎人星级 | 用"发现新大陆"等具体成就定义，不是抽象数字 |
| 《咒术回战》 | 特级咒灵 | "需要整个咒术界联手才能封印"——用具体场景定义等级 |
| 《海贼王》 | 悬赏金 | 数字有意义是因为有"能买什么"的参照系 |
| 《一拳超人》 | 英雄等级 | 用讽刺的方式展示等级系统的荒谬，反向设计 |
| 冰与火之歌 | 无等级系统 | 用身份、权力、军队数量等具体事物代替抽象等级 |

## 优秀代价系统参考（代价必须可见）

| 来源 | 能力与代价 | 为什么优秀 |
|------|----------|-----------|
| 《钢之炼金术师》 | 人体炼成 → 失去身体部位 | 代价永久可见，爱德华的机械臂时刻提醒读者 |
| 《进击的巨人》 | 始祖巨人继承者 → 寿命只剩13年 | 倒计时创造紧迫感 |
| 《东京喰种》 | 使用赫子 → 需要吃人维持 | 道德代价，让能力变成诅咒 |
| 《Fate》 | 令咒 | 只有三次机会，用完就失去控制权——数量限制 |
| 《Re:Zero》 | 死亡回归 → 无法告诉任何人 | 心理孤独作为代价，比肉体痛苦更深刻 |
| 《约定的梦幻岛》 | 农场契约 | 用具体的"交出食用儿童"展示交易的恐怖 |

## 概念命名参考（避免中二病）

| ❌ 坏命名 | ✅ 好命名 | 改进原因 |
|----------|----------|----------|
| 暗影歼灭模式 | 猎杀状态 | 简洁，不堆砌形容词 |
| 超维度精神共振 | 心意相通 | 用日常词汇表达奇幻概念 |
| 绝对零度冰封领域 | 凝冬 | 两个字胜过八个字 |
| 无限剑制·终极形态 | 剑冢 | 有意象，不需要"无限""终极"这种空洞修饰 |
| 第七感·神圣小宇宙 | 神识 | 简洁，有东方韵味 |
| 量子态意识投射 | 分身 | 读者本来就懂分身是什么 |
