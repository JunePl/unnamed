# 第一部分：种族机制调研

## 调研背景

本项目世界中非人类种占总人口66%，但当前所有主要事件/角色均为人类。种族机制刚解冻。本调研收集跨媒介的种族设计成功案例，提取可迁移机制，评估对本项目六国设定的适配性。

**已有种族相关设定摘要**（来自六国志/彼界）：
- 彼界实体化生物已确认5种（哥布林/石蛙/雾蛾/棘鼠/影鹿），均为低智力非人形或低智力人形
- 科尔曼汗国：万灵信仰+萨满议会+兽灵型灵契+游牧文化
- 艾瑞克帝国：军事秩序，可能按战斗力分层
- 米拉尔圣国：信仰治理，可能存在种族歧视
- 瓦尔德联邦：科技民主，种族多元
- 塞伦共和国：商业实用主义，不在乎种族只在乎利润

### 来源1：《风暴光典》(Stormlight Archive) — 歌者/聆听者种族设计

**调研范围**: Brandon Sanderson的歌者(Singer)/聆听者(Listener)种族体系
**来源**: [Singers | Stormlight Archive Wiki](https://stormlightarchive.fandom.com/wiki/Singers) / [Listener Forms](https://stormlightarchive.fandom.com/wiki/Listener_Forms) / [Parshendi](https://stormlightarchive.fandom.com/wiki/Parshendi)

**核心机制拆解**：

1. **形态转换(Form-Changing)机制**：歌者通过与不同类型的灵(spren)结合，可以切换形态——战争形态(warform)、工作形态(workform)、交配形态(mateform)、学者形态(scholarform)等。**每种形态会改变思维方式和性格倾向**，但核心自我不变。
   - 这是"异质思维"的最佳范例：不是"长得不一样的人"，而是**思维模式本身随形态变化**
   - 形态选择=社会角色分配=个人牺牲（选战争形态就失去温柔）

2. **韵律(Rhythms)系统**：歌者能听到世界的内在韵律，不同情绪对应不同韵律。人类完全感知不到这一层。歌者的对话中会"跟随某种韵律说话"。
   - 这创造了**不可翻译的感知差异**——人类永远无法完全理解歌者的沟通方式

3. **被奴役历史的政治叙事**：歌者在故事开始时是被人类奴役的"帕什人"(parshmen)，被视为无意识的劳动工具。后来恢复意识后的身份危机和复仇叙事，直接映射现实种族压迫议题。

**可迁移机制**：形态/状态影响思维+不可翻译的感知层+被奴役/被歧视的历史作为政治冲突源
**适配度**：高
**适配方案**：本项目的灵契系统已有源流分类（9大源流），可以设计不同种族对不同源流的**天然亲和度差异**——非人种不是"也能用灵契"，而是"以完全不同的方式与灵互动"。科尔曼的兽灵型已有雏形。

### 来源2：《全职猎人》嵌合蚁篇 — 蚁王梅路艾姆的非人种角色弧

**调研范围**: 嵌合蚁的种族设计+蚁王从"绝对非人"到"质疑自身"的转变弧线
**来源**: [梅路艾姆_百度百科](https://baike.baidu.com/item/%E6%A2%85%E8%B7%AF%E8%89%BE%E5%A7%86/6631720) / [如何评价蚁王 - 知乎](https://www.zhihu.com/question/292111574) / [蚁王的恶意 - 知乎](https://www.zhihu.com/question/31697339)

**核心机制拆解**：

1. **"吞噬继承"的种族基因**：嵌合蚁通过吞食其他物种获得其特性。蚁王继承了所有被吞食人类的碎片记忆/情感。这不是"学习"人性——是**基因层面的强制融合**。
   - 蚁王的人性不是"被感化"，而是"被污染"——他体内有人类的碎片在呼唤

2. **异质价值观的展现**：蚁王最初的价值体系是纯粹的种族生存逻辑（强者支配弱者，一切服务于蚁族延续）。这不是"邪恶"，是**另一套完全自洽的伦理体系**。
   - 他对人类生命的漠视不是残忍——是蚂蚁看待食物的正常态度

3. **小麦(棋手盲女)的催化作用**：蚁王的转变不是通过战斗/说教，而是通过**一个完全不在乎他的力量的人**。小麦只关心棋盘，这让蚁王第一次遇到了"力量无法解决的事"。
   - 核心机制：**用非人种不擅长的领域去撬动他的世界观**

4. **悲剧内核**：蚁王选择了人性，但人类用核弹（穷人的玫瑰）杀了他。人类证明了他们配不上他选择的信任。
   - 机制：非人种的"人性觉醒"不被人类接受——种族和解的不可能性

**可迁移机制**：非人种拥有完全自洽的"非人伦理"+通过意外领域(而非说教)触发价值观松动+种族和解的不可能性作为悲剧引擎
**适配度**：中高
**适配方案**：本项目可设计某个非人种角色，其价值体系完全不同于人类，但在特定情境下与人类角色产生化学反应。关键是：不要把"接受人类价值观"当作正面结局。

### 来源3：《巫师》系列(The Witcher) — 种族歧视作为政治剧情引擎

**调研范围**: 精灵/矮人作为二等公民的政治叙事+松鼠党起义+灰色道德
**来源**: [巫师3种族介绍 - 游民星空](https://www.gamersky.com/handbook/201506/597718_28.shtml) / [巫师3背景解析 - 游民星空](https://www.gamersky.com/handbook/201510/672693.shtml) / [CDPR的选择设计 - 知乎](https://zhuanlan.zhihu.com/p/194327741)

**核心机制拆解**：

1. **"非人种族"标签本身就是压迫工具**：人类用"非人种族"这个称呼来定义所有其他种族，暗含"人类是默认/正常"的预设。精灵和矮人被强制缴纳"非人种族税"。
   - 机制：**制度性歧视通过法律/经济手段实现，而非纯粹的暴力**

2. **松鼠党的灰色道德**：被压迫的精灵组建武装反抗组织"松鼠党"，但松鼠党本身也犯下暴行。游戏中玩家需要在"帮助被压迫者"和"阻止恐怖主义"之间做选择。
   - 机制：**反抗者不等于好人——被压迫不等于正义的通行证**

3. **两难选择设计**：每个涉及种族冲突的任务都没有"正确答案"。帮助精灵=助长极端主义，帮助人类=维护不公正制度。
   - 机制：**种族冲突提供的不是"解谜"而是"道德困境"**

4. **杰洛特的"局外人"视角**：猎魔人本身就是"非人"（变异人类），被人类和非人种族同时歧视。这让他成为种族冲突的完美观察者。
   - 机制：**主角自身的种族模糊性使他成为种族叙事的棱镜**

**可迁移机制**：制度性歧视(法律/经济/语言)替代纯暴力+反抗者的灰色道德+种族冲突提供道德困境而非简单善恶+局外人视角
**适配度**：高
**适配方案**：本项目中米拉尔圣国的信仰治理可以包含制度性种族歧视（如非人种不能担任某些圣职）。艾瑞克帝国的军事分层可以基于种族战斗力。阿尔温作为灵契者/前执法局探员，可以成为类似杰洛特的"观察者"角色。

### 来源4：《龙族》(江南) — 混血种的身份危机与血统等级

**调研范围**: 混血种设定+秘党+血统等级体系+种族歧视的叙事功能
**来源**: [龙族 - 维基百科](https://zh.wikipedia.org/zh-hans/%E9%BE%99%E6%97%8F_(%E6%B1%9F%E5%8D%97)) / [秘党 - 萌娘百科](https://zh.moegirl.org.cn/%E7%A7%98%E5%85%9A) / [龙族混血种视角 - 知乎](https://www.zhihu.com/question/1886002416287188365)

**核心机制拆解**：

1. **混血种的"原罪"叙事**：混血种的诞生方式本身就是耻辱（人类女性被献给龙族）。秘党认为"混血种的诞生是原初人类卑劣的盗窃"。
   - 机制：**种族身份的起源就是创伤——不是可以骄傲的遗产，而是需要面对的伤疤**

2. **血统等级(D/C/B/A/S)与能力绑定**：龙族血统越纯，能力越强。这创造了"天生的不平等"。
   - 机制：**生物学决定论作为世界观基石——它是否公正？**

3. **言灵系统**：混血种通过觉醒龙族基因获得"言灵"（特殊能力），属五大元素系。121种已知言灵各不相同。
   - 机制：**种族血统决定能力类型，但个体差异巨大**

**可迁移机制**：混血/跨种族身份的创伤起源+血统与能力的不公正绑定+种族身份是伤疤而非勋章
**适配度**：中
**适配方案**：本项目中，不同种族对灵的亲和度不同——这可以成为一种"天生的不平等"。但关键是：谁定义了"强"？科尔曼萨满的"兽灵共鸣"在学术体系中可能不被视为"正统灵契"，但在草原上它更有效。

### 来源5：《来自深渊》(Made in Abyss) — 环境强制异化与身份丧失

**调研范围**: 成骸(Narehate)机制+娜娜奇的种族异化+探索代价
**来源**: [娜娜奇_百度百科](https://baike.baidu.com/item/%E5%A8%9C%E5%A8%9C%E5%A5%87/22411253) / [来自深渊 - 维基百科](https://zh.wikipedia.org/zh-hans/%E4%BE%86%E8%87%AA%E6%B7%B1%E6%B7%B5)

**核心机制拆解**：

1. **环境导致的不可逆变异**：深渊的"上升负荷"（诅咒）会导致探索者身体变异。第六层的诅咒可以直接导致"成骸"——丧失人形，变成非人存在。
   - 机制：**环境/力量本身改变物种——这不是选择，是代价**

2. **娜娜奇的双重身份**：她保留了人类的智慧和情感，但外形已经是兔形的非人。她既不是人也不是怪物，是一种"中间状态"。
   - 机制：**"几乎是人"比"完全不是人"更令人不安**

3. **代价即叙事**：每一次深入探索都伴随不可逆的身体/精神损伤。探索的动力（好奇心/使命/爱）与代价之间的张力就是叙事。

**可迁移机制**：环境/力量导致不可逆种族异化+中间状态的身份危机+"几乎是人"的恐怖谷效应
**适配度**：中高
**适配方案**：本项目彼界的深层探索可以设计类似机制——长期深入彼界的灵契者可能发生"灵质侵蚀"，外形/思维逐渐偏离人类。这为非人种的"起源故事"提供了一种可能：某些非人种可能是古代人类在彼界深层探索后异化的后裔。

### 来源6：《异兽迷城》(彭湃) — "一万只兽中只有一个人"的反转设定

**调研范围**: 非人种伪装成人类的社会+信任危机+身份揭露机制
**来源**: [异兽迷城 - 中国作家网](https://www.chinawriter.com.cn/n1/2023/0721/c404027-40040597.html) / [异兽迷城书评 - 知乎](https://zhuanlan.zhihu.com/p/699135550)

**核心机制拆解**：

1. **"你身边的人可能不是人"**：核心设定是"一万只兽中只有一个人"，而非传统的"一万个人中有一只兽"。主角的家人、朋友、同学都可能是伪装成人类的"兽"。
   - 机制：**多数即非人——当非人种是常态时，"人类"反而是异类**

2. **日常生活中的身份怀疑**：不是通过战斗/对抗来展示种族差异，而是通过**日常互动中的微小违和感**。配角的行为看似正常，但细节不对。
   - 机制：**种族身份的悬疑化——伪装/揭露的张力**

3. **"世界是假的，但爱是真的"**：即使对方不是人类，建立的情感联系仍然真实。
   - 机制：**跨种族关系的核心问题不是"对方是什么"，而是"我们之间的关系是真的吗"**

**可迁移机制**：非人种伪装融入社会+日常违和感替代直接展示+跨种族情感的真实性问题
**适配度**：中
**适配方案**：本项目中，某些高智力非人种可能已经以人类形态融入六国社会（特别是塞伦共和国——不在乎种族只在乎利润）。身份揭露可以成为政治危机的触发器。

### 来源7：《赤心巡天》(张临川) — 多种族的独特文化生态

**调研范围**: 圣族/海族/曳落族/犰狳族的文化差异设计
**来源**: [赤心巡天 - 百度百科](https://baike.baidu.com/item/%E8%B5%A4%E5%BF%83%E5%B7%A1%E5%A4%A9/24349830) / [赤心巡天种族设定 - 起点](https://m.qidian.com/wenshu/qwzmirh/)

**核心机制拆解**：

1. **每个种族有独特的文化仪式**：圣族的"相狩"(战士继承逝者之名)和"魂育之刑"(活埋+龙木种子吸取生命精华)——这些仪式从人类视角看是野蛮的，但在圣族文化中有深刻的宗教意义。
   - 机制：**"野蛮"是视角问题——每个种族的仪式在其文化内部完全合理**

2. **海族的"万瞳"能力**：海族有背部的交叉纵目(万瞳)，可以跨越玄界观察所有海族并促进种族进化。这是一种**人类完全没有的集体意识**。
   - 机制：**非人种的独特生理机制创造独特的社会结构**

3. **犰狳族的"祭妖天决"**：将衰老濒死的族人转化为"祭妖"（战争主力）。从人类角度是"用族人当炮灰"，从犰狳族角度是"赋予老者最后的荣耀"。
   - 机制：**战争观/死亡观的种族差异——不同种族对"牺牲"的定义不同**

**可迁移机制**：文化仪式的视角相对性+非人生理创造非人社会结构+死亡观/战争观的种族差异
**适配度**：高
**适配方案**：科尔曼汗国的萨满传统可以包含类似的"外人看来野蛮、内部完全合理"的仪式。不同种族对灵契/彼界的态度差异可以映射其文化核心。

### 来源8：《虚构推理》(城平京) — 非人种与人类的仲裁者角色

**调研范围**: 岩永琴子作为"妖怪的知慧之神"+人类与非人种的中间人
**来源**: [虚构推理 - 百度百科](https://baike.baidu.com/item/%E8%99%9A%E6%9E%84%E6%8E%A8%E7%90%86/16836751) / [虚构推理 - 萌娘百科](https://zh.moegirl.org.cn/%E8%99%9A%E6%9E%84%E6%8E%A8%E7%90%86)

**核心机制拆解**：

1. **仲裁者角色**：岩永琴子被妖怪选为"知慧之神"，专门为妖怪们解决争端。她是人类，但被妖怪社会接受为仲裁者。
   - 机制：**跨种族仲裁者——在两个世界之间行走的人**

2. **代价交换**：琴子成为仲裁者的代价是失去右眼和左脚。与非人种建立联系必须付出身体代价。
   - 机制：**跨种族交流有不可逆的代价——不是免费的双重国籍**

3. **"虚构推理"的核心**：琴子用"合理的虚构故事"来解决问题——不追求真相，而追求"所有人都能接受的解释"。
   - 机制：**人类和非人种之间需要的不是"真相"，而是"双方都能接受的叙事"**

**可迁移机制**：跨种族仲裁者角色+跨种族交流的不可逆代价+"双方都能接受的叙事"替代绝对真相
**适配度**：中
**适配方案**：本项目中，诺敏作为科尔曼萨满学徒，可以发展成类似的"跨文化/跨种族仲裁者"角色。她的萨满传统本身就是"人类与灵/自然/非人存在之间的桥梁"。

### 来源9：托尔金中土世界 — 语言驱动的种族设计范式

**调研范围**: 精灵/矮人/霍比特人的文化差异设计方法论
**来源**: [J.R.R.托尔金 - 澎湃](https://www.thepaper.cn/newsDetail_forward_28210645) / [矮人(中土大陆) - 维基百科](https://zh.wikipedia.org/zh-hans/%E7%9F%AE%E4%BA%BA_(%E4%B8%AD%E5%9C%9F%E5%A4%A7%E9%99%B8))

**核心机制拆解**：

1. **语言→文化→历史→地理的反向构建**：托尔金先创造语言，从语言特征推导出使用这种语言的种族的文化特征、历史进程和地理分布。
   - 精灵语优雅流畅→精灵文化重视美与永恒→精灵的历史是"缓慢衰退的挽歌"
   - 矮人语粗犷有力→矮人文化重视工艺与坚韧→矮人的历史是"被压迫但不屈服"
   - 机制：**语言/表达方式的差异是种族差异的最直觉入口**

2. **犹太人对矮人创作的影响**（托尔金本人确认）：矮人是"在自己的住处既是原居者又是外人"，使用本地语言但带有母族口音。
   - 机制：**现实民族经验作为种族设计的情感基底**

3. **种族特征的一致性**：每个种族的行为逻辑、审美偏好、社会组织方式都从核心文化基因推导而出，形成自洽的体系。
   - 矮人：坚韧+固执+重承诺+重工艺+对地下世界的执念 = 完整的文化包
   - 机制：**文化包设计——从一个核心特质推导出全部表现**

**可迁移机制**：语言/表达方式作为种族差异的第一入口+现实民族经验作为情感基底+从核心特质推导完整文化包
**适配度**：高
**适配方案**：本项目中，每个非人种应有独特的语言特征（至少是口音/特殊词汇/表达习惯），科尔曼语已设定为"与帝国通用语完全不同"。可从现实游牧民族文化中提取情感基底。

---

### 来源10：《术师手册》— 多种族共存社会的制度设计

**调研范围**: 血月王国的种族平等制度+社会化养育
**来源**: [术师手册 - 百度百科](https://baike.baidu.com/item/%E6%9C%AF%E5%B8%88%E6%89%8B%E5%86%8C/57202471) / [术师手册虚境世界观 - QQ阅读](https://ubook.reader.qq.com/ask/detail/rhclcllwqb)

**核心机制拆解**：

1. **种族平等作为国家基本原则**：血月王国以"种族平等"为第一原则，"社会化养育"为基本国策——所有种族的后代由国家统一抚养。
   - 机制：**制度设计消除种族差异的政治尝试——但代价是什么？**

2. **虚境(彼界类似物)中的平等**：在虚境中，不同种族通过同样的机制（真谛之门）进入。虚境是法则和知识构建的世界，比现实世界大数万倍。
   - 机制：**灵魂世界作为种族平等的试验场——在那里，身体差异不重要**

**可迁移机制**：制度性种族平等的尝试与代价+灵魂/精神世界中身体差异消失
**适配度**：中高
**适配方案**：本项目彼界中，不同种族的灵体形态可能趋同（因为只有灵魂进入）——这意味着彼界可以成为"唯一真正种族平等的空间"。但灵的源流亲和度差异仍然存在，创造另一种不平等。

## 第一部分小结：种族设计核心原则提炼

从10个来源中提炼出的6条核心原则：

| # | 原则 | 来源依据 | 本项目潜在应用 |
|---|------|---------|---------------|
| 1 | **异质思维 > 异质外表** | 风暴光典(形态影响思维)、猎人蚁王(非人伦理) | 非人种的价值观/感知方式必须真正不同 |
| 2 | **制度性歧视 > 个体暴力** | 巫师(非人种族税)、术师手册(制度平等) | 种族冲突通过法律/经济/文化体现 |
| 3 | **文化包完整性** | 托尔金(语言→文化)、赤心巡天(仪式) | 每个种族从核心特质推导完整文化 |
| 4 | **跨种族关系的代价** | 虚构推理(失去右眼)、来自深渊(异化) | 跨种族理解/交流不是免费的 |
| 5 | **灰色道德的种族冲突** | 巫师(松鼠党)、猎人蚁王(穷人玫瑰) | 反抗者不等于好人，压迫者不等于纯恶 |
| 6 | **灵魂世界的种族平等悖论** | 术师手册(虚境)、风暴光典(韵律) | 彼界中身体消失但灵的差异仍在 |

# 第三部分：彼界种族元素调研

## 核心问题：不同种族进入彼界后体验是否不同？

### 现有设定基础（来自世界通鉴·彼界V1.1）

- 进入方式统一：灵契者感知灵痕→顺着灵痕"滑入"→灵魂以灵体形态存在
- 灵体外观与本人相似但半透明
- 进入位置随机，与他人相遇概率极低
- 停留时限按序位分级

### 调研发现：种族差异在灵魂世界中的4种可能模式

**模式A：感知差异模式（来源：风暴光典韵律系统）**

某些种族能感知到彼界中人类感知不到的层面——类似歌者能听到"韵律"。例如：
- 科尔曼兽灵型种族可能在彼界中能"听到"灵脉的节奏（萨满传统中的万灵信仰本质上就是与灵的另一种交流方式）
- 某些非人种可能能"看到"彼界的颜色更丰富（因为他们的灵与彼界的亲和度更高）
- 这创造了"同一个地方，不同种族看到完全不同的景象"的叙事可能

**模式B：灵体形态差异模式（来源：来自深渊的异化机制）**

不同种族进入彼界后，灵体形态可能不同：
- 人类灵体=半透明人形（已确认）
- 某些非人种的灵体可能更接近其"源流亲和"的形态——如兽灵型种族的灵体可能带有兽类特征
- 这意味着在彼界中，"你是什么种族"是无法隐藏的——灵体直接暴露了你与灵的关系

**模式C：交互方式差异模式（来源：术师手册虚境+赤心巡天海族万瞳）**

不同种族可能以不同方式与彼界实体互动：
- 人类灵契者通过"分析/推理"理解彼界（阿尔温的方式）
- 科尔曼萨满通过"直觉/共鸣"理解彼界（诺敏的方式——已在V1 Line C中体现）
- 某些非人种可能通过"集体感知"理解彼界（类似海族的万瞳——多个个体的感知汇聚成集体图景）
- 这直接呼应了V1 Line C中阿尔温与诺敏的"分析法vs直觉法"文化碰撞

**模式D：历史记忆差异模式（来源：进击的巨人地下室+托尔金）**

不同种族在彼界中可能触发不同的"记忆回声"：
- 彼界中的时间残像/古代遗迹投影，对不同种族可能呈现不同内容
- 人类看到的是帝国时代的人类历史碎片
- 非人种看到的可能是大崩溃之前的非人历史——那段被人类历史书写抹去的记忆
- 这为彼界的超长线伏笔提供了强大工具：不同种族在彼界中"看到的历史"互相矛盾，而所有碎片拼合后才是完整真相

### Line C（彼界超长线）中种族元素的可能嵌入点

| V1阶段 | 已有内容 | 种族元素嵌入可能 |
|--------|---------|----------------|
| Ch10 表层初入 | 诺敏救人 | 诺敏的灵体形态可能与阿尔温不同（模式B） |
| Ch29-34 表层探索 | 文化碰撞(分析法vs直觉法) | 已天然包含种族感知差异（模式C） |
| Ch51-58 显影区 | 中层投影震撼 | 诺敏可能看到与阿尔温不同的投影内容（模式D） |
| Ch66-68 满潮名场面 | 潮汐共振+情感爆发 | 满潮中不同种族的灵体反应可能不同（模式A） |

### 彼界实体化生物与"种族"的关系

已确认的5种实体化生物（哥布林/石蛙/雾蛾/棘鼠/影鹿）都是低智力存在。但调研提出一个可能性：

- **高智力实体化生物是否存在？** 如果表层的寄生虫群可以实体化为哥布林（儿童水平智力），那么中层/深层的更高级存在是否也可以实体化？
- **实体化生物 vs "非人种"的关系**：某些"非人种"可能是古代高智力实体化生物的后裔——它们在大崩溃时涌入现世，在数千年中发展出了自己的文化
- 这个假说（如果被采纳）可以解释"为什么非人种占总人口66%"——因为大崩溃带来的不仅是灵，还有大量彼界存在的实体化

# 种族设计建议（给本项目的5条具体建议）

**以下仅为调研结论，所有建议需用户确认后方可进入创作流程。**

### 建议1：种族差异的核心不是外表，而是"与灵的关系不同"

本项目的世界基石是灵契系统。与其设计"长耳朵的精灵"或"绿皮的兽人"这类传统外观差异，不如让种族差异建立在**"不同种族与灵的互动方式根本不同"**这一核心机制上。

- 人类：通过"契约"与灵建立关系（理性、交易式）
- 兽灵型种族（科尔曼相关）：通过"共生"与灵建立关系（本能、融合式）
- 可能的第三类：通过"共鸣/集体意识"与灵建立关系（群体、网络式）

这避免了"换皮人类"问题，因为**思维方式/感知方式/与世界互动的方式**才是种族的本质差异。

**来源依据**：风暴光典(形态影响思维/韵律系统)、赤心巡天(海族万瞳)

---

### 建议2：种族政治应该因国而异，避免单一的"歧视/平权"叙事

六国对非人种的态度应该各不相同，且每种态度都有其内在逻辑：

| 国家 | 对非人种的态度 | 内在逻辑 | 叙事功能 |
|------|-------------|---------|---------|
| 卡萨林 | 边缘化但不迫害 | 三执政各有利用非人种的方式 | 德米特里的信息网可能包含非人种线人 |
| 瓦尔德联邦 | 制度性平等 | 民主制+工业化需要劳动力 | 表面平等下的实际不平等（玻璃天花板） |
| 塞伦共和国 | 实用主义接纳 | 只看利润不看种族 | 非人种商人/雇佣兵的活跃空间 |
| 艾瑞克帝国 | 按战斗力分层 | 军事帝国逻辑 | 非人种战士的军事价值vs政治地位 |
| 米拉尔圣国 | 宗教性歧视 | 教义将非人种视为"不完全的灵魂" | 最尖锐的种族冲突+宗教改革伏笔 |
| 科尔曼汗国 | 自然融合 | 万灵信仰不区分种族 | 跨种族共存的范例+萨满传统的包容性 |

**来源依据**：巫师(制度性歧视)、术师手册(制度平等)、赤心巡天(文化差异)

---

### 建议3：在V1中用"微型事件+日常细节"自然引入种族元素，而非集中设定倾倒

V1的核心任务是画商之死(Line A)和政治暗流(Line B)。种族元素不应喧宾夺主，而应以以下方式自然融入：

- **微型事件**：阿尔温在调查过程中路过非人种聚居区，遇到一个小谜题（蟲師模式）
- **日常细节**：市场上非人种商人的独特商品/砍价方式；酒馆里人类与非人种的微妙互动
- **配角闪光**：一个非人种配角在3秒内展示独特魅力（配角闪光铁律）
- **政治背景**：安东尼奥的商人网络中包含非人种商人，他的死导致这些人失去保护

**来源依据**：蟲師(行旅式氛围)、异兽迷城(日常违和感)、冰菓(日常之谜)

---

### 建议4：彼界（Line C）是种族机制最自然的展示窗口

V1 Line C（彼界独立超长线）中，阿尔温与诺敏的搭档已经包含"分析法vs直觉法"的文化碰撞。这可以扩展为种族感知差异的展示：

- 诺敏在彼界中能感知到阿尔温感知不到的层面（模式A）
- 两人在同一地点可能看到不同的景象（模式D）
- 满潮名场面中，诺敏的灵体反应可能揭示萨满传统与彼界的深层联系

这些设计不需要改变Line C的独立性——它们只是丰富了已有的"文化碰撞"维度。

**来源依据**：风暴光典(韵律)、术师手册(虚境种族平等)、来自深渊(异化)

---

### 建议5：至少设计一个有名字的非人种配角出现在V1中

当前V1的所有有名角色都是人类。建议至少引入1个有名字、有独特魅力的非人种配角。候选位置：

| 候选位置 | 角色类型 | 功能 | 种族 |
|---------|---------|------|------|
| Line A 画商调查中 | 画商的非人种模特/朋友 | 提供线索+展示跨种族关系 | 待定 |
| Line B 安东尼奥的网络 | 非人种商人/线人 | Thanatos引爆后的"代价承受者" | 塞伦商会的非人种成员 |
| 日常场景 | 提耶林学院的非人种学生 | 自然展示学术界的种族多元性 | 待定 |
| Line E 四国展 | 某国的非人种外交随员 | 展示不同国家对非人种的态度差异 | 瓦尔德联邦/艾瑞克帝国代表 |

**来源依据**：全职猎人蚁王(通过个体角色展示种族叙事)、虚构推理(仲裁者角色)
