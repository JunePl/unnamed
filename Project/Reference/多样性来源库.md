# 多样性来源库 V2.0

> **来源**：事件与冲突设计者(E) + 大纲撰写者(A) + 小说编辑(C) + 文风润色者(F)
> **维护者**：事件与冲突设计者（主）、全角色可补充
> **核心变更**：从"16类智斗"重构为"7层叙事参考"，按规模分层，每层服务不同的创作问题

---

## ★ 使用规则 ★

### 按规模选源（强制）
- 设计 **整书/跨卷架构** → 从 Tier 1 选择
- 设计 **整卷/多幕叙事** → 从 Tier 2 选择
- 设计 **中型事件**（4-15章） → 从 Tier 3 选择
- 设计 **小型事件**（1-3章） → 从 Tier 4 选择
- 设计 **日常/氛围章节** → 从 Tier 5 选择
- 设计 **角色关系/情感场景** → 从 Tier 6 选择
- 需要 **特殊结构手法** → 从 Tier 7 选择

### 跨层混搭（推荐）
每次中型以上的设计，**必须从至少2个不同Tier各选1个来源**。例如设计一个中型事件：从Tier 3选逻辑骨架 + 从Tier 5选氛围质感 + 从Tier 6选情感内核。

### 抽取后强制调研协议
抽到任何来源后，**禁止直接套用**。必须执行：
1. **机制提取**：这部作品在该Tier的核心问题上，用了什么**可迁移的结构机制**？（不是剧情，是手法）
2. **适配评估**：该机制是否与本项目的类型基因档案兼容？
3. **变形应用**：该机制需要怎样变形才能融入本项目？
4. 若不确定，使用 WebSearch 调研该作品的叙事分析文章

输出格式：
```
【来源调研卡】
作品：___
Tier：___
可迁移机制：___（一句话，描述手法而非剧情）
适配度：高/中/低
变形方案：___（如何融入本项目）
```

## Tier 1: 世界架构参考（整书/跨卷级）

> **核心问题**：如何构建一个经得起5卷探索、设定严谨、伏笔深远的世界？

### 1A. 深度世界构建

| 作品 | 可迁移机制 | 规模参考 |
|------|-----------|----------|
| Malazan Book of the Fallen (Steven Erikson) | 多文明交叉史学：用考古碎片式叙事构建跨万年历史，读者从碎片拼出全貌 | 10卷，跨300,000年 |
| The Stormlight Archive (Brandon Sanderson) | 硬魔法体系+Cosmere宇宙：每个力量都有精确规则、代价、社会影响，跨系列暗线连接 | 10卷计划，多星球 |
| The Lord of the Rings (J.R.R. Tolkien) | 语言学驱动世界观：从语言→文化→历史→地理的反向构建，附录即世界 | 神话级纵深 |
| Dune (Frank Herbert) | 生态-政治-宗教三角锁：每个系统变化都牵动其他两个，形成自洽封闭环 | 6卷主线+扩展 |
| 十二国记 (小野不由美) | 天命官僚制世界观：用严谨的政治制度设计驱动所有冲突，而非个人力量 | 12国完整政体 |
| 猎人×猎人 (富�的义博) | 制约与誓约：越强的能力需要越具体的限制，创造出无限的战术可能性 | 持续扩展 |

### 1B. 长线伏笔网络

| 作品 | 可迁移机制 | 伏笔特征 |
|------|-----------|----------|
| One Piece (尾田荣一郎) | 超长周期伏笔回收：数百话前的细节在关键时刻重新赋义，单一细节服务多次回收 | 500+话跨度伏笔 |
| 进击的巨人 (谏山创) | 信息反转型伏笔：早期所有信息都是"正确但不完整"的，完整信息颠覆所有解读 | 地下室=元叙事反转 |
| 钢之炼金术师 (荒川弘) | 等价交换贯穿式：核心规则既是世界法则、也是主题、也是最终解谜关键 | 规则=伏笔=主题 |
| Wheel of Time (Robert Jordan) | 预言解构型：预言在故事中被反复引用，但真实含义在最终卷才完全清晰 | 跨14卷预言体系 |

### 1C. 文明演化参考

| 作品 | 可迁移机制 |
|------|-----------|
| Foundation (Isaac Asimov) | 文明衰亡的数学模型：用"心理史学"概念让历史趋势可预测但个体不可预测 |
| 三体 (刘慈欣) | 黑暗森林法则：一条公理推导出整个文明交互规则，简洁而震撼 |
| 银河英雄传说 (田中芳树) | 双主角文明对比：用两个对立阵营的平行视角展示制度优劣，没有绝对正义 |
| 十二国记 (小野不由美) | 天命政治实验：每个国家都是同一制度的不同实验，失败国家的教训映射现实 |

## Tier 2: 卷级叙事框架（整卷/多幕级）

> **核心问题**：如何让一卷（50-100章）的大棋局层次分明、步步升级、终局爆发？

### 2A. 政治博弈/多派系

| 作品 | 可迁移机制 | 棋局特征 |
|------|-----------|----------|
| A Song of Ice and Fire (George R.R. Martin) | N方博弈无主角：每个派系都有完整的利益计算和内部矛盾，读者被迫多视角思考 | 7+派系同时博弈 |
| 银河英雄传说 (田中芳树) | 战略-战术双层叙事：宏观国家博弈与微观战役指挥交替推进 | 帝国vs同盟vs第三势力 |
| Code Geass | 棋盘翻转型：主角的每次胜利都创造新的敌人，棋盘本身在变化 | 每5集翻转一次格局 |
| 琅琊榜 | 病弱军师型布局：主角用信息差和人心操控替代武力，每步棋都有三层用意 | 一人搅动朝堂 |
| 风起陇西 (马伯庸) | 谍战多面体：同一事件在不同情报层有不同面貌，读者和角色同时被骗 | 三国谍战 |

### 2B. 多线汇聚架构

| 作品 | 可迁移机制 | 汇聚模式 |
|------|-----------|----------|
| Cloud Atlas (David Mitchell) | 六层嵌套+主题共振：六条线时代不同、类型不同，但共享同一主题内核 | 6线，跨千年 |
| Pulp Fiction (Quentin Tarantino) | 非线性交叉：打乱时间线让观众从不同角度重新理解同一事件 | 3线非线性 |
| 20世纪少年 (浦�的直树) | 童年-成年双时间线：过去的游戏变成现在的灾难，记忆碎片就是线索 | 2时间线交织 |
| Monster (浦泽直树) | 追猎式多线：主角追踪目标的过程中，目标的过去通过第三方逐步揭露 | 追踪线+历史线 |

### 2C. 长弧线升级

| 作品 | 可迁移机制 |
|------|-----------|
| 全职猎人·蚂蚁篇 | 从"讨伐怪物"到"理解人性"的类型渐变：读者以为在看战斗热血，实际走向人文悲剧 |
| Breaking Bad | 温水煮青蛙式堕落：每集只越过一条小底线，累积成不可逆的毁灭 |
| Death Note | 猫鼠博弈螺旋升级：每次交锋都让对手进化，读者永远猜不到谁先暴露 |

## Tier 3: 中型事件参考（4-15章级）

> **核心问题**：如何让一个10章左右的事件/调查弧逻辑严密、节奏精良、情感满足？

### 3A. 经典推理结构

| 作品 | 可迁移机制 | 篇幅 | 逻辑特征 |
|------|-----------|------|----------|
| And Then There Were None (Agatha Christie) | 封闭空间递减法：嫌疑人随死亡逐个消除，但最终答案颠覆消除逻辑本身 | ~300页 | 公平线索+终极反转 |
| The Devotion of Suspect X (东野圭吾) | 倒叙式悬疑：读者一开始就知道凶手，悬念转为"他能瞒住吗"和"值得吗" | ~300页 | 动机>手法 |
| 虚构推理 (城平京) | 逻辑拍卖型：主角用3个"虚假但逻辑自洽"的解释击败1个真相，推理成为武器而非求真 | 数卷 | 逻辑本身是工具 |
| Pluto (浦泽直树) | 情感包装推理：表面是调查连环杀手，实质是探讨"机器人能否拥有灵魂" | 8卷 | 推理服务于主题 |

### 3B. 阴谋/行动弧

| 作品 | 可迁移机制 | 弧线特征 |
|------|-----------|----------|
| Ocean's Eleven | 团队组装+计划执行+意外+反转：计划永远不完美，临场应变才是高潮 | 组装→执行→崩盘→反转 |
| The Lies of Locke Lamora (Scott Lynch) | 双层骗局：读者以为在看A骗局，实际B骗局才是主线 | 骗局嵌套 |
| 赌博默示录 (福本伸行) | 规则渐显型：游戏规则在对局中逐步揭示，每次新规则都改变局势 | 规则即悬念 |

### 3C. 危机/灾难弧

| 作品 | 可迁移机制 |
|------|-----------|
| 正解的卡多 (东映) | 外交式科幻危机：不靠战斗解决危机，靠谈判、牺牲和不完美妥协 |
| 寄生兽 (岩明均) | 共生困境弧：敌人不是纯恶，解决方案必须牺牲一部分"自己" |
| Annihilation (Jeff VanderMeer) | 探索即变异：调查过程本身改变调查者，没有人能带着原来的自己回来 |

## Tier 4: 小型事件参考（1-3章级）

> **核心问题**：如何在极短篇幅内构建完整的逻辑闭环，让读者获得"啊！"的满足感？

### 4A. 精巧短篇推理

| 作品 | 可迁移机制 | 篇幅 |
|------|-----------|------|
| 冰菓/古典部系列 (米�的�的信) | 日常之谜：谜题材料全部来自校园日常（为什么教室门锁了？），规模极小但逻辑精密 | 单篇1-2章 |
| Sherlock Holmes短篇 (Arthur Conan Doyle) | 观察-推理-验证三段式：从一个微小线索推导出完整故事，重在展示思维过程 | 单篇20-40页 |
| 古畑任三郎 | 倒叙破案法：观众已知凶手，乐趣在看侦探如何用逻辑逼出真相 | 单集45分钟 |
| 万能鉴定士Q (松冈圭祐) | 知识应用型：用真实的专业知识（美术鉴定、材料学）解决谜题，无超能力 | 单案1-3章 |
| 金田一少年短篇 | 密室+不在场证明的经典变体：用物理诡计创造"不可能"，再用逻辑打破 | 单案3-5话 |

### 4B. 精巧短篇非推理

| 作品 | 可迁移机制 | 特征 |
|------|-----------|------|
| Exhalation/你一生的故事 (Ted Chiang) | 概念即结构：整个故事就是一个概念的展开过程，概念理解完毕=故事结束 | 极致简洁 |
| 世界尽头与冷酷仙境 (村上春树) | 双线交替短章：两条极短的线交替推进，每章仅推进一个微小步骤 | 短章交替 |
| 蟲師 单话 (漆原友纪) | 现象-调查-代价三段式：奇异现象→探索原因→发现代价（总有人付出了什么） | 单话完结 |
| O. Henry短篇 | 结尾反转型：全篇铺垫一个方向，最后一句话翻转一切，但翻转后更温暖 | 极短，反转即主题 |

## Tier 5: 日常与氛围参考

> **核心问题**：如何让"无事发生"的章节成为读者最爱的部分？如何让世界有体温、有质感？

### 5A. 日常即叙事（slice of life大师）

| 作品 | 可迁移机制 | 氛围特征 |
|------|-----------|----------|
| 夏目友人帐 (绿川幸) | 单元治愈+暗线递进：每集独立温馨小故事，但主角的孤独和成长是暗线 | 温暖+淡淡忧伤 |
| 物语系列 (西尾维新) | 对话即剧情：两人站着聊天30页，但对话本身有推理、有情感、有哲学 | 对话密度极高 |
| 3月のライオン (羽海野千花) | 日常中的内心风暴：外表平静的日常场景中，角色内心正在经历巨大波动 | 平静表面+深层痛苦 |
| よつばと! (あずまきよひこ) | 纯有机随机：没有任何暗线/伏笔/冲突，纯粹是"今天发生了有趣的事"的快乐 | 零张力纯欢乐 |
| 蟲師 (漆原友纪) | 行旅式氛围：主角游走于不同村落，每个地方有独特的风土人情和奇异现象 | 乡野+神秘+寂寥 |
| Aria (天野梅花) | 描写即叙事：用大量环境描写和日常对话替代情节推进，世界本身就是故事 | 极致治愈 |

### 5B. 日常中埋暗线

| 作品 | 可迁移机制 | 暗线技巧 |
|------|-----------|----------|
| 冰菓 (米泽穗信) | 日常谜题表皮+青春暗线：表面在解"谁借了书"，实际在解"我为什么活着" | 谜题=青春隐喻 |
| 青春猪头少年 (�的鸭志达) | 超自然嵌入日常：奇异现象是角色心理问题的外化，解决超自然=治愈心灵 | 异常=心理 |
| Another (绫辻行人) | 日常渗透恐惧：越是正常的日常描写，越让人不安——因为读者知道死亡随时来 | 平静=不安 |
| 刺杀骑士团长 (村上春树) | 日常中的裂缝：大量日常描写（做饭、画画、散步）中偶尔闪过不可解释的细节 | 现实99%+异常1% |

### 5C. 世界质感参考

| 作品 | 可迁移机制 |
|------|-----------|
| 狼与香辛料 (支倉凍砂) | 经济日常化：用面包价格、汇率波动、税收政策让奇幻世界像真实世界一样运转 |
| 药屋少女的呢喃 (日向夏) | 专业知识日常化：用真实的药理学/毒理学知识编织日常小事件，读者在娱乐中学到东西 |
| 少女终末旅行 (つくみず) | 废墟诗学：在文明毁灭后的世界中，两个少女的日常吃饭/睡觉/闲聊变成了哲学 |
| 来自深渊 (つくしあきひと) | 冒险日常化：探险过程中的吃饭、休息、整理装备本身就是叙事，展示世界的残酷与美 |

## Tier 6: 情感与关系参考

> **核心问题**：如何让角色之间的关系有体温、有变化、有厚度？

### 6A. 羁绊/信任构建

| 作品 | 可迁移机制 | 关系类型 |
|------|-----------|----------|
| 钢之炼金术师 (荒川弘) | 兄弟共赴：两人目标一致但方式不同，通过分歧→争吵→理解循环深化关系 | 兄弟/搭档 |
| Spy×Family (远藤达哉) | 假装家族真情化：以利益结合的"假关系"在日常中逐渐变成真情 | 伪家庭→真家庭 |
| 我的青春恋爱物语果然有问题 (渡航) | 孤独者社交：用大量内心独白展示社交恐惧者如何一步步建立信任 | 友情从零开始 |
| 命运石之门 (志倉千代丸) | 牺牲循环式羁绊：反复看到同伴死亡→为了拯救而牺牲→羁绊在痛苦中加深 | 同伴/恋人 |

### 6B. 背叛/信任崩塌

| 作品 | 可迁移机制 | 崩塌模式 |
|------|-----------|----------|
| 进击的巨人 (谏山创) | 立场反转型：不是角色变了，是读者对角色的理解变了——同一个人，换个视角就是敌人 | 视角改变=背叛 |
| A Song of Ice and Fire | 渐进式背叛：背叛不是突然发生的，而是一系列小妥协的累积 | 小妥协→大背叛 |
| Banana Fish (吉田秋生) | 被迫背叛型：角色被环境逼到必须背叛最重要的人，读者和角色一起心碎 | 环境逼迫 |

### 6C. 师徒/传承

| 作品 | 可迁移机制 |
|------|-----------|
| 火影忍者 师徒链 | 理念传承型：师傅传给徒弟的不是技能而是信念，信念在每代人身上变形 |
| 棋魂 (堀田由美) | 跨时代对话：师傅已死/离开，徒弟通过棋局/作品与师傅"对话" |
| 暗杀教室 (松井优征) | 限期师生：在倒计时压力下，教育从"教知识"变成"教如何活" |

### 6D. 群像关系网

| 作品 | 可迁移机制 |
|------|-----------|
| Durarara!! (成田良悟) | 蝴蝶效应群像：每个角色的小行动都影响其他人，没有主角但所有人相连 |
| Baccano! (成田良悟) | 无主角叙事：多个角色在同一事件中各自行动，读者从碎片拼出全貌 |
| 文豪野犬 (朝雾卡夫卡) | 组织对撞型群像：三个组织的成员各怀心事，阵营内的关系和阵营间一样复杂 |

## Tier 7: 结构手法与类型融合

> **核心问题**：有哪些特殊的叙事技巧和类型融合方式可以创造独特阅读体验？

### 7A. 叙事实验

| 手法 | 代表作品 | 可迁移机制 |
|------|----------|-----------|
| 不可靠叙述者 | Gone Girl, The Murder of Roger Ackroyd | 叙述者本身在撒谎/遗忘/扭曲，读者必须"阅读弦外之音" |
| 元叙事/打破第四面墙 | Doki Doki Literature Club, House of Leaves | 叙事媒介本身成为叙事对象 |
| 非线性时间 | Arrival (Story of Your Life), Pulp Fiction | 打乱时间线让同一事件拥有多重含义 |
| 多视角同一事件 | Rashomon, Baccano! | 同一事件在不同角色视角中完全不同，真相在交叉中浮现 |
| 循环叙事 | Re:Zero, Steins;Gate, Groundhog Day | 重复经历同一时段，每次循环添加新信息/新策略 |
| 嵌套叙事 | Cloud Atlas, Pale Fire, 1001 Nights | 故事中的故事中的故事，层层映射 |

### 7B. 类型颠覆

| 颠覆方向 | 代表作品 | 可迁移机制 |
|----------|----------|-----------|
