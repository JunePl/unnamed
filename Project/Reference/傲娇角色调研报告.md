## 一、近五年最受欢迎傲娇角色调研

### 1. 中野二乃（《五等分的花嫁》）
- **日本傲娇人气投票5连冠**（2020-2024连续蝉联）
- **核心机制**: 初期是五姐妹中最敌对的——直接下药赶走家教。全程维持高压敌意，转变来得突然且大胆（直接告白，而非渐渐软化）
- **可迁移机制**: ★反差爆炸★ 冷面维持越久越坚定，转变时冲击越大。不是"慢慢变好"，是"一直很坏→突然炸裂"
- **读者反馈关键词**: "因为她一直是最凶的，所以告白那一刻才最震撼"

### 2. 逢坂大河（《龙与虎》）
- **经典标杆·"掌中老虎"**
- **核心机制**: 暴力型傲娇+小身材反差。表面找茬→实际把所有温柔给了别人→最终认清自己的心意
- **可迁移机制**: 对目标人物的攻击性行为本身就是一种"无法处理的感情溢出"

### 3. 茜（《我和班上最讨厌的女生结婚了》2025）
- **2025年新生代傲娇代表**
- **核心机制**: "结婚设定"下仍然嘴硬。不是渐渐接受，是在极端日常中被迫面对情感
- **可迁移机制**: 外部环境迫使两人必须互动→角色无法逃避但拒绝承认

### 4. Stalker Shrine原型（TV Tropes收录·跨作品通用）
- **定义**: 角色在私密空间中维持一个关于执念对象的"祭坛/档案室"——照片、物品、日记、监控记录
- **关键区分**:
  - 纯stalker = 恐怖/病态（Fatal Attraction类型）
  - Shrine + 冷面 = 反差萌/读者震撼（"原来她一直...！"）
  - ★设计关键★: 如果角色在公开场合对目标人物是**冷漠/敌对的**，那么私密档案室的揭示就是**纯正的反差核爆**

### 5. Yandere-lite亚型（2024-2025网文/动画趋势）
- **定义**: 拥有yandere的执念强度，但没有yandere的危险行为（不伤害目标/不伤害情敌）
- **核心特征**: 收集信息/物品、监控动态、私密空间中的执念行为
- **2025年趋势**: 该亚型人气急升——读者喜欢"执念深但不伤人"的角色，因为既有刺激感又有安全感

## 二、维辛妮·蒙特原型定位

### ★ 核心原型: 傲娇(表面) × 暗恋狂(内核) ★

```
                    公开场合                 私密空间
                ┌──────────────┐        ┌──────────────┐
  V1(Ch1-70)    │ 不假辞色·冰面 │        │ 阿尔温档案室  │
                │ 嘲讽·找茬    │        │ 偷窃物品     │
                │ 零温度·无例外 │        │ ★读者不知道★  │
                └──────────────┘        └──────────────┘
  V1(Ch70+)     │ 冰面第一道裂缝│        │ 同上         │
                │ 塔尔拉失踪    │        │              │
                └──────────────┘        └──────────────┘
  V2-V3         │ 冰面偶有裂纹  │        │ ★某次被撞破★  │
                │ 但迅速补上    │        │  反差核爆     │
                └──────────────┘        └──────────────┘
  V4+(回国)     │ 接应阿尔温    │        │ 阿尔温终于知道│
                │ "别误会，只是 │        │              │
                │  恰好有个空房" │        │              │
                └──────────────┘        └──────────────┘
```

### 设计六铁律

1. **冰面恒常律**: V1中Ch70之前，维辛妮对阿尔温的态度**零度恒温**。无渐进软化，无"偶尔真情流露"，无"眼神中闪过温柔"。冰就是冰。
2. **暗线可查律**: 每次维辛妮出现时，都有一个**仅在二刷时才能注意到**的细节——她总"恰好"在阿尔温附近、她知道不该知道的信息、她走后某件小物品不见了。一刷读者只看到冷面。
3. **档案室延迟揭示律**: 阿尔温档案室的存在在V1中**绝不揭示**。最早在V2末或V3中通过意外事件暴露。揭示瞬间=全书级别的反差核爆。
4. **裂缝唯一律**: V1中冰面的唯一裂缝=Ch76-77塔尔拉失踪。且裂缝后立即补上（她会为自己的求助感到恼怒，之后加倍冷淡）。
5. **功能性出场律**: 维辛妮每次出场必须有表面合理理由（送修物品/经过工坊/参加同一活动），绝不是"专门来见阿尔温"。
6. **二乃爆炸律**: 参照中野二乃——冷面维持越久、越坚定，最终揭示时冲击越大。维辛妮的告白/揭示应该是全书最意外的情感炸弹之一。

## 三、暗线伏笔设计参考（仅供节奏规划·S-10子弧使用）

### 二刷可见的暗线细节类型

| 类型 | 示例 | 一刷效果 | 二刷效果 |
|------|------|---------|---------|
| 信息异常 | "你最近走路心不在焉的"——她怎么知道？ | 认为是偶然观察 | 她一直在观察 |
| 物品失踪 | 阿尔温的旧工具/笔/手帕不见了 | 完全不注意 | 是她拿走的 |
| 恰好在场 | 每次出现都有合理理由但频率偏高 | 认为是邻里日常 | 她制造了理由 |
| 对阿涅丝态度 | 对阿涅丝格外好——远超正常邻里关系 | 认为她只是喜欢女孩子 | 接近阿涅丝=接近阿尔温的生活 |
| 信息准确度 | "你不是明天要去王都吗？"——没人告诉她 | 忽略 | 她有情报渠道 |
