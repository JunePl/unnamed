# Ch1 — 灵痕不说谎
# ═══════════════════════════════════════

**【一行钩子】**：一杯茶的温度暴露了委托人的第一个谎言——而他今天会听到更多。

## 1. 本章目的
- 800字内建立核心身份：灵痕分析师，非战斗型主角
- 展示中上阶层经济定位（橡木街独栋小楼+定制检测器）
- 阿涅丝出场，建立兄妹日常
- 阶级氛围第一笔：窗外的"安全税"
- ★ 名场面：凭茶温推断委托人撒谎

## 2. 核心冲突 / 悬念
表面：小委托的谎言被当场拆穿——阿尔温的分析能力首秀
暗面：这个世界的秩序在日常中渗透着压迫

## 3. 主要登场角色
- 阿尔温：目标=处理日常委托 / 情绪=从容、微微无聊 / 阻碍=委托人不老实
- 阿涅丝：目标=给哥哥送茶 / 情绪=活泼、日常 / 状态=刚毕业，在家暂住
- 委托人·韦伯（一次性角色）：中年商人，找阿尔温鉴定灵器真伪，但在隐瞒来路

## 4. 场景节拍

### 场景1：办公室·分析中（约800字）★首章黄金800字★
```
地点：橡木街12号·二楼工作室
目标：建立"灵痕分析师"概念+展示能力+经济定位

· 开篇即动作：阿尔温正在用灵痕检测器分析一件小灵器
· 环境自然渗透世界观：
  → 检测器：黄铜机壳，内嵌灵石感应芯，指针在刻度盘上颤动
  → 工作台上散落着上一个委托的报告——他的字迹工整到像印刷体
  → 书架上混杂着《灵痕鉴定学》与旧案卷宗
· 韦伯推门进来，带着一枚"据说是古物"的灵器吊坠
· 阿尔温没抬头："请坐。茶在桌上。"

★ 名场面·茶温推理 ★
阿尔温（分析灵器的间隙，随口）：
"韦伯先生，您迟到了四十分钟。"
韦伯："路上堵——"
阿尔温："您出门前换过一次衣服。左袖口有两个不同方向的褶痕。
  换衣服意味着犹豫。犹豫意味着你不确定该不该来。
  再加上——"他终于抬起眼睛，
  "你的茶。我十分钟前倒的，现在是38度。
  你一直在看它，但没有喝。
  不是不渴——是怕喝了就得开口说实话。"
韦伯的脸白了。

阿尔温（平静）："这枚吊坠不是古物。是六个月前铸魂系灵契者新造的。
  灵痕衰减曲线一目了然。
  您不是来鉴定真伪的——您早就知道它是假的。
  您想知道的是：造假的人是谁。"

→ 读者第一印象：这个人不靠拳头，靠脑子
→ 伏笔#0（极轻）：灵痕衰减曲线分析——第二幕伯爵案中同一技能的关键运用
```

9. 黄铜检测器在阳光下的反光——暖色调收尾

### 场景2：阿涅丝的抽屉（约500字）★暗线碎片一★
```
地点：阿涅丝的房间（阿尔温不在场）
目标：阿涅丝暗线第一个碎片

· 治安官哈罗德不情不愿："埃德蒙，你已经不是执法局的人了。"
  阿尔温："正因为不是，你才请得起我。"

★ 三层线索首次展示 ★
物理线索：
· 马蒂亚斯仰面倒在画作下方，无外伤
· 表情不是恐惧——是"惊讶"
· "他认识凶手。"
灵痕线索：
· 阿尔温的检测器在尸体旁捕捉到极微弱的灵力残留
· 但残留的衰减模式不对——不像是灵契攻击后的自然衰减
· "像是有人很小心地清理过，但漏了一点。"
人际线索：
· 保险箱被打开，文件被翻过——但值钱的东西没丢
· 凶手不是为了钱。是为了信息。

★ 老彼得（替身）登场 ★
· 一个老人从内室走出来
· 海伦娜："这是彼得，画廊的经理。我丈夫最信任的人。"
· 老彼得（替身）看向阿尔温——
  "先生您是？"
· 阿尔温的反应：
  他的手微微停顿了一下。
  两周前，这个人在这间画廊的后厅，
  亲手给他倒了一杯茶。
  当时他说："马蒂亚斯是个好人，只是太容易相信人了。"
  现在他看着阿尔温——像是第一次见。
· 阿尔温没有追问。只是记住了。

→ 读者感受：有点奇怪？也许老彼得忘了？伤心过度？
→ 伏笔P01正式埋设：不认识阿尔温
→ 伏笔P02同步埋设：阿尔温注意到老彼得的右手食指——
  两周前满是颜料痕迹（多年经手画作的永久染色）
  现在干干净净。
  → 阿尔温没有意识到这意味着什么。读者更不会。
```

### 场景3：治安官的压力（约400字）
```
地点：画廊门外
目标：展示体制压力+阿尔温争取调查空间

· 哈罗德："阿尔温，我知道你能力强。但这案子上面想快结。"
  阿尔温："哪个'上面'？"
  哈罗德沉默。
  阿尔温："给我三天。三天后如果什么也找不到，你按自然死亡结案。"
  哈罗德："……你知道为什么你会离开执法局吗？"
  阿尔温（平静）："因为我看到了不该看的东西。"
  哈罗德看了他很久："三天。别给我添乱。"

→ 阿尔温的过去第一次被触及——但不展开
→ 读者好奇：他看到了什么？
```

### 场景4：D模式第0次感觉（约300字）
```
地点：回家路上
目标：内心独白·直觉触发

· 阿尔温走在街上，脑中反复回放灵痕数据
· 内心：
  "那个灵痕残留……衰减曲线的形状。
   我在哪里见过？
   不是最近。是很久以前。
   是……"

  但身体记住了某种反应——
  后颈微微发凉。

· 他没有深想。现在还太早。
· → 读者不会理解这个"三年前"意味着什么
· → Ch8才会回收

断章：
阿尔温推开家门。
阿涅丝在客厅画画——颜料溅了一地。
"哥哥！你比我还早——"
他在门口站了两秒。
然后笑了一下。
"今天的案子比较有意思。"
```

## 5. 关键台词 / 道具 / 伏笔
台词：
- "先生您是？"（替身·全卷最关键伏笔之一）
- "因为我看到了不该看的东西。"

道具：
- 灵痕检测器的异常读数 → 案件起点
- 老彼得"干净的右手食指" → P02

伏笔：
- [P01] 老彼得不认识阿尔温 → Ch25五段式揭示
- [P02] 右手食指无颜料斑 → Ch25
- [P-轻] 保险箱文件被翻 → Ch10暗格线索
- [P-轻] 灵痕衰减模式的"熟悉感" → Ch8 D模式浮现
- [P-轻] 海伦娜隐瞒的细节 → Ch6展开

## 6. 人物弧线
阿尔温：从日常进入案件模式。观察力首次面对"说不清哪里不对"的困惑。
老彼得（替身）：伪装稳定，但"不认识阿尔温"是致命破绽——他不知道有委托。

## 7. 场景细节（10个）
1. 画廊入口的铃铛没有响——门被人从内部打开过
2. 马蒂亚斯倒在一幅海景画下方，海浪仿佛要淹没他
3. 灵痕检测器的指针异常颤抖，像一只受惊的蜂鸟
4. 海伦娜的手套始终没摘——她在控制自己
5. 老彼得（替身）站着的姿势微微僵硬——不像是悲伤，像是在维持
6. 画廊地板上有一道被拖拽过的痕迹——但不是拖尸体，是移动画框
7. 治安官的制服扣子扣到最上面一颗——他今天穿得很正式（有人施压？）
8. 空气中混合着松节油、蜡烛和一丝甜味（什么的甜味？）
9. 阿尔温蹲在尸体旁时，灵痕检测器在他腰带上轻轻震动
10. 回家路上的路灯在他身后拉出长影，像一条追着他跑的线索

## 8. 节奏与写作提示
- 画廊现场是本章重头——至少占50%篇幅
- 老彼得的"先生您是？"要自然，不能用惊叹号或强调标记
- P01和P02同时埋设但不引起注意——克制、克制、再克制
- D模式前兆用身体反应（后颈发凉）而非思维分析
- 断章回到温暖日常——让Ch4的阴暗形成对比

## 9. 结尾
阿涅丝在画画，颜料溅了一地。日常温暖收束。
钩子方向："今天的案子比较有意思"——读者想知道有多有意思。
断章指数：4（中钩，案件正式开启）

## 10. 压抑/释放值
(5, 2) — 尸体+异常+治安官压力。释放仅来自回家一瞬。

# Ch4 — 死者的一周
# ═══════════════════════════════════════

**【一行钩子】**：一个人在死前七天做的事，比他活着时做的一切都诚实。

## 1. 本章目的
- 还原马蒂亚斯死前一周的异常行为
- 误导路径A启动：学徒艾伦有动机
- 三层线索系统深入展示
- 商人联盟极轻触（马蒂亚斯的社交圈）

## 2. 核心冲突 / 悬念
表面：马蒂亚斯为什么在死前突然转移资产、秘密会面？
暗面：艾伦对马蒂亚斯的怨恨是否足以杀人？

## 3. 主要登场角色
- 阿尔温：目标=还原死者最后一周 / 情绪=专注、系统化 / 方法=灵痕时间线重建
- 艾伦：目标=隐藏自己的秘密 / 情绪=恐惧、防御 / 首次正面登场

## 4. 场景节拍

### 场景1：灵痕时间线重建（约800字）
```
地点：韦恩画廊·二次勘查
目标：展示阿尔温的核心技能——灵痕考古

· 阿尔温独自回到画廊（治安官给了钥匙）
· 他开始"灵痕时间线重建"——
  → 灵痕分析师的独门技术：通过残留灵力的衰减层次
    反推过去数天内该空间的灵力活动序列
· 发现：
  → 马蒂亚斯在死前3天开始频繁进出地下室（脚步灵痕密度异常）
  → 有至少2个不同的灵力签名在案发前48小时内出现过
  → 其中一个签名极其微弱——"像是故意压低了灵力输出"

### 场景2：走访艾伦（约800字）
```
地点：艺术区廉租房
目标：误导A启动+审讯技巧展示

· 艾伦住在画廊附近的廉租房——与马蒂亚斯的富裕形成对比

  阿尔温注意到艾伦袖口有新鲜颜料——

  艾伦："自……自己画的。"
  → 谎言。那种颜料是画廊专用的昂贵矿物颜料，不是学徒买得起的。
· 误导A的种子：
  → 艾伦恨马蒂亚斯（被克扣三年工资）
  → 艾伦那晚的行踪可疑
  → 但阿尔温内心标注：他不是灵契者，造不出那种灵痕

→ 线索#1：艾伦有秘密，但不是杀人

## Ch50 — 下一封信
**【一行钩子】**：你看到了。很好。现在你明白为什么我不在体制内做事了。

(压5/释6) ★第二幕收束★

**目的**：莱奥里克信件升级+阿尔温的决心重建+情感进度条+10%
**核心场景**：
- 阿尔温在工作台上发现新的信——太阳标记
  → 这一封比之前的更长

  "阿尔温。
   你做了一件了不起的事——
   不是因为你查到了真相，
   而是因为你在知道真相没有用之后——
   没有假装自己不知道。
   
   安东尼奥也是这种人。
   他知道自己的死不能扳倒德米特里。
   但他还是选择了死——
   因为他相信总有人会把碎片拼起来。
   
   你就是那个人。
   
   但你不能一个人拼。
   
   你现在明白为什么我不在体制内做事了。
   体制是他们的工具。
   想在体制内打败他们——
   就像用他们造的锁打开他们的门。
   
   如果你准备好了——
   下一封信里有一个地址。
   来见我。
   
   不是现在。等时机到了，你会知道的。"

- 阿尔温读完信。
  情感进度条：+10%
  从"追查真相"到"想改变这个世界"的转折点。
  
  他把信折好，和安东尼奥的D笔记放在一起。
  两页纸。两个不同的人。
  一个选择了死。一个选择了等待。
  
  而他——阿尔温——
  选择了继续。
  
  不是因为他相信能赢。
  是因为安东尼奥相信了他。
  一个他从未见过的人——
  用命赌了他。
  
  这份信任比任何证据都重。

断章：
阿尔温把所有档案收进保险柜。
然后站起来，走到窗边。

提耶林城的早晨在窗外展开。
马车的声音。小贩的叫卖。孩子的笑声。
城市不知道——也不在意——
它的伯爵用命换来的真相，已经被权力吞没了。

但阿尔温知道。
他会记着。

灵停在窗台上，光芒平稳。
它也在看这座城市。

阿尔温转身。
桌上还有三个未拆封的委托。

他拿起第一个，拆开，开始看。

生活继续。
但他不再是三个月前那个只看灵痕的人了。
他现在看到了灵痕背后的东西——
权力。体制。压迫。
以及——
在所有这些之下——
某种更古老、更安静、更耐心的存在。

D。
