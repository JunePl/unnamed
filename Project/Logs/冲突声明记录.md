**状态**: **已解决** (2026-02-15)

**决策**: 采用**方案C：彼界实体化生物**
**世界观确保者方案设计**:

**执行**: 世界通鉴第五篇超凡科技之后新增「彼界实体化生物」小节。

## 冲突 #6：正文章节结构与大纲/章纲严重不匹配

**严重等级**: !!!致命!!!
**状态**: **已解决** (2026-02-15)

**决策**: **以正文节奏为基准调整章纲**（方案B）
**依据**:
- 正文的"尸体开场"策略在商业上更有冲击力（800字内强冲突，满足番茄/起点首章要求）
- 正文节奏：案件现场(严肃) → 酒馆放松(过渡) → 回家温馨(情感) → 再起波澜(钩子)
- 这种节奏在一章内完成了"悬疑+日常+角色建立"三重任务，比原章纲分三章更高效

**执行方案**:
- Ch1（新）= 画廊命案现场（阿尔温+哈罗德+尸体） + 酒馆过渡（莫哈夫+哥布林闲聊） + 回家兄妹场景（阿涅丝蛋糕/进阶揭示）
- Ch2（新）= 次日早晨 + 画廊调查开始（家属+老彼得）+ 阿涅丝暗线碎片一
- Ch3（新）= 调查深入 + 线索展开 + D模式初现的微妙迹象

**小说编辑验证**: 以正文节奏改编后，首章即包含"尸体+调查+人物魅力+日常温暖+悬念钩子"五要素，首章留存率预计提升。

## 冲突 #9：正文基调与设定基调不匹配

**严重等级**: !中危!
**状态**: **已解决** (2026-02-15)

**决策**: 保留**日常温馨+悬疑混合风格**，但区分场景
**执行原则**:
- 案件现场/调查场景：严肃悬疑为主基调，幽默不超过5%
- 日常场景（回家、酒馆、朋友互动）：温馨幽默为主，允许密度较高的轻松元素
- 兄妹互动：是全书的"情感安全港"，允许最高密度的温暖和幽默
- 随剧情推进，日常场景的温馨感逐渐被暗线侵蚀（第一卷前半段温暖 → 后半段开始渗入不安）

**章纲调整**: 压抑值/释放值标注需要区分"日常线"和"案件线"的不同基调期望。
