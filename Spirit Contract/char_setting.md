   ├─ 来源：噬暗·观察型灵契 + 执法局专业训练

   ├─ 专长：识破精心设计的嫁祸、误导、伪装
   ├─ 原理：寻找"不自然"的痕迹
   └─ 示例：灵痕屏蔽器会留下"边缘干扰纹"

3. 逻辑推理
   ├─ 风格：连接表面上无关的线索
   ├─ 特点：善于发现"缺失"的证据
   └─ 弱点：面对情感问题时会失灵

【三层线索系统】

┌─────────────────────────────────────────────────────────┐
│                                                         │
│  【表面线索】—— 所有人都能看到                          │
│       │                                                 │

│                                                         │
│  【隐藏线索】—— 需要解释"为什么重要"                   │
│       │                                                 │
│       ↓ （需要逻辑推理）                               │
│                                                         │
│  【深层线索】—— 连接多条线索得出结论                    │
│                                                         │
└─────────────────────────────────────────────────────────┘

表面线索：

雷纳德："你怎么知道是沉默兄弟会？"

"军用级屏蔽器，市面上买不到。

 沉默兄弟会可能性最大。"

【推理展示示例——伯爵案件】

└─ 死者胸口有灼烧痕迹 → 炬焰系攻击

阿尔温的质疑：

### [语言基因档案·V0.3更新]

```
1. 分析师的精确
   · 习惯用"如果……那么……"的条件句
   · 陈述推理时会说明"证据链"
   · 区分"事实"和"推断"

示例（分析中）：
"这里有三条线索：
 第一，灼烧痕迹从内向外——这是事实。
 第二，受害者三天前做了手术——这也是事实。
 第三，有人在手术中植入了引燃媒介——这是我的推断。
 如果推断成立，那么凶手至少接触过进步之光的技术。"

2. 压抑的情感表达
   · 用短句表达情感
   · 经常中断自己的话
   · 用沉默代替爆发

示例（面对阿涅丝真相）：
"那天晚上，我跟踪她。
 我看到了——"
（停顿）
"……我看到了。"
（不再多说）

3. 自我反省时的独白
   · 使用"如果当时……"的假设句
   · 责备自己时很直接
   · 但不会陷入自怨自艾

示例（反思）：
"如果当时我多问她一句……
 不，没有'如果'。
 是我选择了不问。
 是我选择了不看。
 这是我的错。"
```
