# 第1章「死者的画」
# ═══════════════════════════════════════════════════════

**【章名】**：死者的画
**【一行钩子】**：他在尸体的瞳孔里，看到了奇特的星空。

## 1. 本章目的
- 首章黄金800字内制造强冲突，抓住读者
- 建立阿尔温的核心形象：**灵痕分析师**，观察力惊人、冷静但有温度
- 引出案件一：画商之死
- 埋下灵契世界的第一个暗示
- **V1.2新增**：展示阿尔温的专业背景（前执法局）

## 2. 核心冲突/悬念
```
表面冲突：画商被杀，看似自然死亡但有蹊跷
深层悬念：尸体瞳孔中残留的异象——这不是普通的谋杀
专业冲突：官方执法局想草草结案 vs 阿尔温的专业直觉
```

## 3. 主要登场角色
```
阿尔温·埃德蒙：
├─ 身份：独立灵痕分析师（前卡萨林执法局）
├─ 目标：调查画商之死的真相
├─ 阻碍：官方想草草结案
├─ 情绪：冷静中带着好奇，职业本能被触发
├─ 状态：刚完成上一个小案件，声誉渐起
└─ 榜单位置：铜阶#1024（暂未提及，留待后续）

治安官·哈罗德：
├─ 目标：尽快结案，不想惹麻烦
├─ 阻碍：阿尔温的坚持
├─ 情绪：不耐烦但不敢完全得罪阿尔温
└─ 暗示：他知道阿尔温曾在执法局工作

画廊管家·老彼得：
├─ 目标：为主人讨回公道
├─ 阻碍：不知道该信任谁
└─ 情绪：悲伤、困惑、一丝恐惧
```
